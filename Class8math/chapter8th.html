Exercise 8.1

1. Add the following.

(i) ab-bc, bc-ca, ca-ab
(ab-bc) + (bc-ca) + (ca-ab) = ab-ab-bc+bc-ca+ca = 0.
(ii) a-b+ab, b-c+bc, c-a+ac
(a-b+ab) + (b-c+bc) + (c-a+ac) = a-a-b+b-c+c+ab+bc+ac = ab+bc+ac.
(iii) 2p²q²-3pq+4, 5+7pq-3p²q²
(2p²q²-3pq+4) + (5+7pq-3p²q²) = (2-3)p²q² + (-3+7)pq + (4+5) = -p²q²+4pq+9.
(iv) l²+m², m²+n², n²+l², 2lm+2mn+2nl
(l²+m²) + (m²+n²) + (n²+l²) + (2lm+2mn+2nl) = 2l²+2m²+2n²+2lm+2mn+2nl = 2(l²+m²+n²+lm+mn+nl).

Hinglish Explanation:

Add karne ke liye 'like terms' (jaise ab aur -ab) ko ek saath group karo. Same variable aur same power waale terms hi 'like terms' hote hain. Unke coefficients (numbers) ko add ya subtract karo.

2. Subtract...

(a) Subtract 4a-7ab+3b+12 from 12a-9ab+5b-3
(12a-9ab+5b-3) - (4a-7ab+3b+12)
= 12a-9ab+5b-3 - 4a+7ab-3b-12 = 8a-2ab+2b-15.
(b) Subtract 3xy+5yz-7zx from 5xy-2yz-2zx+10xyz
(5xy-2yz-2zx+10xyz) - (3xy+5yz-7zx)
= 2xy-7yz+5zx+10xyz.
(c) Subtract 4p²q-3pq+5pq²-8p+7q-10 from 18-3p-11q+5pq-2pq²+5p²q
(18-3p-11q+5pq-2pq²+5p²q) - (4p²q-3pq+5pq²-8p+7q-10)
= p²q - 7pq² + 8pq + 5p - 18q + 28.

Hinglish Explanation:

Subtract karte waqt, jisko subtract karna hai (second expression), uske har term ka sign (+ ka -, - ka +) badal do aur fir 'like terms' ke saath add kar do.

(i) 4, 7p → 4 × 7p = 28p
(ii) -4p, 7p → -4p × 7p = -28p²

(iv) 4p³, -3p → 4p³ × -3p = -12p⁴
(v) 4p, 0 → 4p × 0 = 0

Exercise 8.3

1. Carry out the multiplication of the expressions in each of the following pairs.

(i) 4p, q+r → 4p × (q+r) = 4pq + 4pr
(ii) ab, a-b → ab × (a-b) = a²b - ab²
(iii) a+b, 7a²b² → (a+b) × 7a²b² = 7a³b² + 7a²b³
(iv) a²-9, 4a → (a²-9) × 4a = 4a³ - 36a
(v) pq+qr+rp, 0 → (pq+qr+rp) × 0 = 0

Hinglish Explanation:

Ise Distributive Law kehte hain. Bahar waale monomial ko bracket ke andar har term se ek-ek karke multiply karo. Jaise 4p × (q+r) me 4p ko pehle q se, fir r se multiply kiya.

2. Complete the table.

(i) a × (b+c+d) = ab + ac + ad
(ii) (x+y-5) × 5xy = 5x²y + 5xy² - 25xy
(iii) p × (6p²-7p+5) = 6p³ - 7p² + 5p
(iv) 4p²q² × (p²-q²) = 4p⁴q² - 4p²q⁴
(v) (a+b+c) × abc = a²bc + ab²c + abc²

Hinglish Explanation:

First expression ko second expression ke har term se multiply karna hai (distributive property). Sign ka dhyan rakhein aur variables ki power ko add karein.

3. Find the product.

(i) (a²) × (2a²²) × (4a²⁶) → (1×2×4) × a⁽²⁺²²⁺²⁶⁾ = 8a⁵⁰
(ii) (2/3 xy) × (-9/10 x²y²) → (2/3 × -9/10) × (x·x²) × (y·y²) = -3/5 x³y³
(iii) (-10/3 pq³) × (6/5 p³q) → (-10/3 × 6/5) × (p·p³) × (q³·q) = -4p⁴q⁴
(iv) x × x² × x³ × x⁴ → x⁽¹⁺²⁺³⁺⁴⁾ = x¹⁰

Hinglish Explanation:

Monomials ke product ka rule: Saare coefficients (numbers/fractions) ko multiply karo. Saare same variables ki powers ko add kar do.

4. (a) Simplify 3x(4x-5)+3 and find its values for (i) x=3 (ii) x=1/2.

Simplification: 3x(4x-5)+3 = 12x² - 15x + 3
(i) For x=3: 12(3)² - 15(3) + 3 = 12(9) - 45 + 3 = 108 - 45 + 3 = 66.
(ii) For x=1/2: 12(1/2)² - 15(1/2) + 3 = 12(1/4) - 15/2 + 3 = 3 - 15/2 + 3 = 6 - 15/2 = (12-15)/2 = -3/2.

(b) Simplify a(a²+a+1)+5 and find its value for (i) a=0, (ii) a=1, (iii) a=-1.

Simplification: a(a²+a+1)+5 = a³ + a² + a + 5
(i) For a=0: (0)³ + (0)² + 0 + 5 = 5.
(ii) For a=1: (1)³ + (1)² + 1 + 5 = 1 + 1 + 1 + 5 = 8.
(iii) For a=-1: (-1)³ + (-1)² + (-1) + 5 = -1 + 1 - 1 + 5 = 4.

Hinglish Explanation:

Pehle expression ko simplify karo (brackets open karke). Fir jo simplified expression aaye, usme x ya a ki di hui value daal kar final answer nikalo.

5. (a) Add: p(p-q), q(q-r) and r(r-p)

(p²-pq) + (q²-qr) + (r²-rp) = p²+q²+r²-pq-qr-rp

(b) Add: 2x(z-x-y) and 2y(z-y-x)

(2xz-2x²-2xy) + (2yz-2y²-2xy) = 2xz-2x²-4xy+2yz-2y²

(c) Subtract: 3l(l-4m+5n) from 4l(10n-3m+2l)

(40ln-12lm+8l²) - (3l²-12lm+15ln) = 40ln-12lm+8l²-3l²+12lm-15ln = 5l²+25ln

(d) Subtract: 3a(a+b+c)-2b(a-b+c) from 4c(-a+b+c)

First simplify both expressions:
From: 4c(-a+b+c) = -4ac+4bc+4c²
Subtract: (3a²+3ab+3ac) - (2ab-2b²+2bc) = 3a²+ab+3ac+2b²-2bc
Now subtract: (-4ac+4bc+4c²) - (3a²+ab+3ac+2b²-2bc) = -3a²-2b²+4c²-ab+6bc-7ac

Hinglish Explanation:

Pehle brackets ko multiply karke open karo. Fir 'Add' me like terms ko combine karo. 'Subtract' me, 'from' ke baad wale expression me se pehle wala expression minus karo (sign badal kar).

Exercise 8.4

1. Multiply the binomials.

(i) (2x+5) and (4x-3) → 2x(4x-3) + 5(4x-3) = 8x²-6x+20x-15 = 8x²+14x-15
(ii) (y-8) and (3y-4) → y(3y-4) - 8(3y-4) = 3y²-4y-24y+32 = 3y²-28y+32
(iii) (2.5l-0.5m) and (2.5l+0.5m) → (2.5l)² - (0.5m)² = 6.25l² - 0.25m²
(iv) (a+3b) and (x+5) → a(x+5) + 3b(x+5) = ax+5a+3bx+15b
(v) (2pq+3q²) and (3pq-2q²) → 2pq(3pq-2q²)+3q²(3pq-2q²) = 6p²q²-4pq³+9pq³-6q⁴ = 6p²q²+5pq³-6q⁴
(vi) (3/4 a² + 3b²) and 4(a² - 2/3 b²) → (3/4 a² + 3b²) × (4a² - 8/3 b²) = 3a⁴-2a²b²+12a²b²-8b⁴ = 3a⁴+10a²b²-8b⁴

Hinglish Explanation:

Do binomials ko multiply karne ke liye, pehle binomial ke har term ko doosre poore binomial se multiply karo. Jaise (a+b)(c+d) = a(c+d) + b(c+d). Fir simplify karke like terms ko combine kar lo.

2. Find the product.

(i) (5-2x)(3+x) = 5(3+x) - 2x(3+x) = 15+5x-6x-2x² = 15-x-2x²
(ii) (x+7y)(7x-y) = x(7x-y) + 7y(7x-y) = 7x²-xy+49xy-7y² = 7x²+48xy-7y²
(iii) (a²+b)(a+b²) = a²(a+b²) + b(a+b²) = a³+a²b²+ab+b³
(iv) (p²-q²)(2p+q) = p²(2p+q) - q²(2p+q) = 2p³+p²q-2pq²-q³

Hinglish Explanation:

Yeh bhi binomial multiplication hai. Pehle bracket ke first term ko doosre poore bracket se multiply karo, fir pehle bracket ke second term ko doosre poore bracket se multiply karo aur simplify karo.

3. Simplify.

(i) (x²-5)(x+5)+25
= x³+5x²-5x-25+25 = x³+5x²-5x
(ii) (a²+5)(b³+3)+5
= a²b³+3a²+5b³+15+5 = a²b³+3a²+5b³+20
(iii) (t+s²)(t²-s)
= t(t²-s) + s²(t²-s) = t³-ts+s²t²-s³
(iv) (a+b)(c-d)+(a-b)(c+d)+2(ac+bd)
= (ac-ad+bc-bd) + (ac+ad-bc-bd) + 2ac+2bd = 4ac
(v) (x+y)(2x+y)+(x+2y)(x-y)
= (2x²+xy+2xy+y²) + (x²-xy+2xy-2y²) = (2x²+3xy+y²) + (x²+xy-2y²) = 3x²+4xy-y²
(vi) (x+y)(x²-xy+y²)
= x(x²-xy+y²) + y(x²-xy+y²) = x³-x²y+xy²+x²y-xy²+y³ = x³+y³
(vii) (1.5x-4y)(1.5x+4y+3)-4.5x+12y
= 1.5x(1.5x+4y+3) - 4y(1.5x+4y+3) -4.5x+12y
= (2.25x²+6xy+4.5x) - (6xy+16y²+12y) -4.5x+12y
= 2.25x²-16y²
(viii) (a+b+c)(a+b-c)
= a(a+b-c) + b(a+b-c) + c(a+b-c)
= a²+ab-ac + ab+b²-bc + ac+bc-c² = a²+b²+2ab-c²

Hinglish Explanation:

Simplify karne ka matlab hai pehle brackets ko multiply karke open karo, uske baad sabhi 'like terms' ko ek saath group karke add ya subtract kar do taaki expression chhota ho jaye.
