Exercise 12.1

1. Find the common factors of the given terms.

(i) 12x, 36 → Common Factor: 12
(ii) 2y, 22xy → Common Factors: 2, y. Highest Common Factor: 2y
(iii) 14pq, 28p²q² → Common Factors: 14, p, q. HCF: 14pq
(iv) 2x, 3x², 4 → Common Factor: 1
(v) 6abc, 24ab², 12a²b → Common Factors: 6, a, b. HCF: 6ab
(vi) 16x³, -4x², 32x → Common Factors: 4, x. HCF: 4x
(vii) 10pq, 20qr, 30rp → Common Factor: 10
(viii) 3x²y³, 10x³y², 6x²y²z → Common Factors: x², y². HCF: x²y²

Hinglish Explanation:

Common factor nikalne ke liye, har term ke prime factors likho. Fir dekho ki kaunse numbers aur variables sabhi terms me मौजूद hain. Unhi ko multiply karke HCF milta hai.

2. Factorise the following expressions.

(i) 7x - 42 = 7(x - 6)
(ii) 6p - 12q = 6(p - 2q)
(iii) 7a² + 14a = 7a(a + 2)
(iv) -16z + 20z³ = 4z(-4 + 5z²)
(v) 20l²m + 30alm = 10lm(2l + 3a)
(vi) 5x²y - 15xy² = 5xy(x - 3y)
(vii) 10a² - 15b² + 20c² = 5(2a² - 3b² + 4c²)
(viii) -4a² + 4ab - 4ca = 4a(-a + b - c)
(ix) x²yz + xy²z + xyz² = xyz(x + y + z)
(x) ax²y + bxy² + cxyz = xy(ax + by + cz)

Hinglish Explanation:

Factorise karne ka matlab hai common factors ko bahar nikalna. Har expression me se sabse bada common factor (HCF) bahar nikalo aur bache hue terms ko bracket ke andar likh do.

3. Factorise.

(i) x² + xy + 8x + 8y = x(x+y) + 8(x+y) = (x+y)(x+8)
(ii) 15xy - 6x + 5y - 2 = 3x(5y-2) + 1(5y-2) = (5y-2)(3x+1)
(iii) ax + bx - ay - by = x(a+b) - y(a+b) = (a+b)(x-y)
(iv) 15pq + 15 + 9q + 25p = 15pq + 25p + 9q + 15 = 5p(3q+5) + 3(3q+5) = (3q+5)(5p+3)
(v) z - 7 + 7xy - xyz = z - xyz - 7 + 7xy = z(1-xy) - 7(1-xy) = (1-xy)(z-7)

Hinglish Explanation:

Ise 'factorisation by regrouping' kehte hain. Jab sabhi terms me kuch common na ho, to 2-2 terms ke group banao. Har group se common nikalo. Iske baad, aapko ek common bracket milega jise aap fir se common nikal sakte ho.(Download kare StudyScript App aur paaye dher saare courses)

Exercise 12.2

1. Factorise the following expressions.

(i) a² + 8a + 16 = (a)² + 2(a)(4) + (4)² = (a+4)²
(ii) p² - 10p + 25 = (p)² - 2(p)(5) + (5)² = (p-5)²
(iii) 25m² + 30m + 9 = (5m)² + 2(5m)(3) + (3)² = (5m+3)²
(iv) 49y² + 84yz + 36z² = (7y)² + 2(7y)(6z) + (6z)² = (7y+6z)²
(v) 4x² - 8x + 4 = 4(x² - 2x + 1) = 4(x-1)²
(vi) 121b² - 88bc + 16c² = (11b)² - 2(11b)(4c) + (4c)² = (11b-4c)²
(vii) (l+m)² - 4lm = l²+2lm+m² - 4lm = l²-2lm+m² = (l-m)²
(viii) a⁴ + 2a²b² + b⁴ = (a²)² + 2(a²)(b²) + (b²)² = (a²+b²)²

Hinglish Explanation:

Ye saare expressions identities par based hain: (a+b)² = a²+2ab+b² aur (a-b)² = a²-2ab+b². Expression ko pehchano ki wo kounsi identity se match karta hai aur uske hisab se factors likho.

2. Factorise.

(i) 4p² - 9q² = (2p)² - (3q)² = (2p-3q)(2p+3q)
(ii) 63a² - 112b² = 7(9a² - 16b²) = 7((3a)² - (4b)²) = 7(3a-4b)(3a+4b)
(iii) 49x² - 36 = (7x)² - (6)² = (7x-6)(7x+6)
(iv) 16x⁵ - 144x³ = 16x³(x² - 9) = 16x³(x-3)(x+3)
(v) (l+m)² - (l-m)² = [(l+m)-(l-m)][(l+m)+(l-m)] = (2m)(2l) = 4lm
(vi) 9x²y² - 16 = (3xy)² - 4² = (3xy-4)(3xy+4)
(vii) (x² - 2xy + y²) - z² = (x-y)² - z² = (x-y-z)(x-y+z)
(viii) 25a² - 4b² + 28bc - 49c² = 25a² - (4b² - 28bc + 49c²) = (5a)² - (2b-7c)² = (5a-(2b-7c))(5a+(2b-7c)) = (5a-2b+7c)(5a+2b-7c)

Hinglish Explanation:

Ye questions a²-b² = (a-b)(a+b) identity par based hain. Expression ko do perfect squares ke difference ke form me likho aur fir identity apply karke factors banao. Kabhi kabhi pehle kuch common nikalna padta hai (jaise Q2(ii)).

3. Factorise the expressions.

(i) ax² + bx = x(ax+b)
(ii) 7p² + 21q² = 7(p²+3q²)
(iii) 2x³ + 2xy² + 2xz² = 2x(x²+y²+z²)
(iv) am² + bm² + bn² + an² = m²(a+b) + n²(b+a) = (a+b)(m²+n²)
(v) (lm+l)+m+1 = l(m+1) + 1(m+1) = (m+1)(l+1)
(vi) y(y+z)+9(y+z) = (y+z)(y+9)
(vii) 5y² - 20y - 8z + 2yz = 5y(y-4) + 2z(y-4) = (y-4)(5y+2z)

Hinglish Explanation:

4. Factorise.

(i) a⁴ - b⁴ = (a²)² - (b²)² = (a²-b²)(a²+b²) = (a-b)(a+b)(a²+b²)
(ii) p⁴ - 81 = (p²)² - 9² = (p²-9)(p²+9) = (p-3)(p+3)(p²+9)
(iii) x⁴ - (y+z)⁴ = (x²)² - ((y+z)²)² = (x²-(y+z)²)(x²+(y+z)²) = [x-(y+z)][x+(y+z)](x²+(y+z)²) = (x-y-z)(x+y+z)(x²+y²+z²+2yz)
(iv) x⁴ - (x-z)⁴ = (x²)² - ((x-z)²)² = (x²-(x-z)²)(x²+(x-z)²) = [x-(x-z)][x+(x-z)](x²+x²-2xz+z²) = (z)(2x-z)(2x²-2xz+z²)
(v) a⁴ - 2a²b² + b⁴ = (a²)² - 2a²b² + (b²)² = (a²-b²)² = [(a-b)(a+b)]² = (a-b)²(a+b)²

Hinglish Explanation:

In questions me a²-b² identity ko do baar use karna pad sakta hai. Power 4 ko (power 2)² ke form me likho aur identity apply karo. Q(v) me pehle (a-b)² waali identity lagegi.

5. Factorise the following expressions.

(i) p² + 6p + 8 = p²+4p+2p+8 = p(p+4)+2(p+4) = (p+4)(p+2)
(ii) q² - 10q + 21 = q²-7q-3q+21 = q(q-7)-3(q-7) = (q-7)(q-3)
(iii) p² + 6p - 16 = p²+8p-2p-16 = p(p+8)-2(p+8) = (p+8)(p-2)

Hinglish Explanation:

Ise 'splitting the middle term' method kehte hain. Middle term (jaise 6p) ko aise do parts me todo jinka sum middle term ke barabar ho aur product first aur last term ke product ke barabar ho. Jaise Q(i) me, 8 ke factors 4 aur 2 hain, aur 4+2=6 hota hai.

Exercise 12.3

1. Carry out the following divisions.

(i) 28x⁴ ÷ 56x = (28/56) * (x⁴/x) = (1/2)x³
(ii) -36y³ ÷ 9y² = (-36/9) * (y³/y²) = -4y
(iii) 66pq²r³ ÷ 11qr² = (66/11) * p * (q²/q) * (r³/r²) = 6pqr
(iv) 34x³y³z³ ÷ 51xy²z³ = (34/51) * (x³/x) * (y³/y²) * (z³/z³) = (2/3)x²y
(v) 12a⁸b⁸ ÷ (-6a⁶b⁴) = (12/-6) * (a⁸/a⁶) * (b⁸/b⁴) = -2a²b⁴

Hinglish Explanation:

Monomial ko monomial se divide karne ke liye, numbers ko numbers se divide karo aur same variables ki powers ko subtract karo.

2. Divide the given polynomial by the given monomial.

(i) (5x² - 6x) ÷ 3x = 5x²/3x - 6x/3x = (5/3)x - 2
(ii) (3y⁸ - 4y⁶ + 5y⁴) ÷ y⁴ = 3y⁸/y⁴ - 4y⁶/y⁴ + 5y⁴/y⁴ = 3y⁴ - 4y² + 5
(iii) 8(x³y²z² + x²y³z² + x²y²z³) ÷ 4x²y²z² = 2(x+y+z)
(iv) (x³ + 2x² + 3x) ÷ 2x = x³/2x + 2x²/2x + 3x/2x = (1/2)x² + x + 3/2
(v) (p³q⁶ - p⁶q³) ÷ p³q³ = p³q⁶/p³q³ - p⁶q³/p³q³ = q³ - p³

Hinglish Explanation:

Polynomial ke har term ko monomial se alag-alag divide karo. Fir har division ko pichle question ki tarah solve karo.

3. Work out the following divisions.

(i) (10x - 25) ÷ 5 = 5(2x-5) ÷ 5 = 2x-5
(ii) (10x - 25) ÷ (2x - 5) = 5(2x-5) ÷ (2x-5) = 5
(iii) 10y(6y + 21) ÷ 5(2y + 7) = 10y * 3(2y+7) ÷ 5(2y+7) = 30y(2y+7) ÷ 5(2y+7) = 6y
(iv) 9x²y²(3z - 24) ÷ 27xy(z - 8) = 9x²y² * 3(z-8) ÷ 27xy(z-8) = 27x²y²(z-8) ÷ 27xy(z-8) = xy
(v) 96abc(3a - 12)(5b - 30) ÷ 144(a - 4)(b - 6) = 96abc * 3(a-4) * 5(b-6) ÷ 144(a-4)(b-6) = (96*15)abc(a-4)(b-6) ÷ 144(a-4)(b-6) = 10abc

Hinglish Explanation:

Numerator (jise divide karna hai) ko factorise karo. Common factors ko bahar nikalo. Fir dekho ki numerator aur denominator me kya common hai aur use cancel kar do.

4. Divide as directed.

(i) 5(2x + 1)(3x + 5) ÷ (2x + 1) = 5(3x+5)
(ii) 26xy(x + 5)(y - 4) ÷ 13x(y - 4) = 2y(x+5)
(iii) 52pqr(p + q)(q + r)(r + p) ÷ 104pq(q + r)(r + p) = (1/2)r(p+q)
(iv) 20(y + 4)(y² + 5y + 3) ÷ 5(y + 4) = 4(y²+5y+3)
(v) x(x + 1)(x + 2)(x + 3) ÷ x(x + 1) = (x+2)(x+3)

Hinglish Explanation:

Yahan expressions pehle se hi factorised form me hain. Sirf numerator aur denominator me common factors (jaise numbers, variables, brackets) ko identify karke cancel karna hai.

5. Factorise the expressions and divide them as directed.

(i) (y² + 7y + 10) ÷ (y + 5) = (y+5)(y+2) ÷ (y+5) = y+2
(ii) (m² - 14m - 32) ÷ (m + 2) = (m-16)(m+2) ÷ (m+2) = m-16
(iii) (5p² - 25p + 20) ÷ (p - 1) = 5(p²-5p+4) ÷ (p-1) = 5(p-1)(p-4) ÷ (p-1) = 5(p-4)
(iv) 4yz(z² + 6z - 16) ÷ 2y(z + 8) = 4yz(z+8)(z-2) ÷ 2y(z+8) = 2z(z-2)
(v) 5pq(p² - q²) ÷ 2p(p + q) = 5pq(p-q)(p+q) ÷ 2p(p+q) = (5/2)q(p-q)

Hinglish Explanation:

Pehle numerator ko factorise karo. Isme common factor, middle term splitting, ya a²-b² jaisi identities use ho sakti hain. Jab numerator factorise ho jaye, to denominator se common factors cancel kar do.
