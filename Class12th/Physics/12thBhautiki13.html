13.1
नाइट्रोजन नाभिक (N714) की बंधन ऊर्जा MeV में ज्ञात कीजिए। mN = 14.00307 u
दिया गया है:
नाइट्रोजन नाभिक का द्रव्यमान, mN = 14.00307 u
नाइट्रोजन के लिए: Z = 7, A = 14, N = A - Z = 7

बंधन ऊर्जा का सूत्र:
ΔEb = [Z mp + N mn - mN] c2
ΔEb = [7 × 1.007825 + 7 × 1.008665 - 14.00307] u × c2
ΔEb = [7.054775 + 7.060655 - 14.00307] u × c2
ΔEb = [14.11543 - 14.00307] u × c2
ΔEb = 0.11236 u × c2

1 u = 931.5 MeV/c2
ΔEb = 0.11236 × 931.5 MeV
ΔEb = 104.66 MeV
अतः नाइट्रोजन नाभिक की बंधन ऊर्जा 104.66 MeV है।

13.5
किसी नाभिकीय अभिक्रिया A + b → C + d का Q-मान निम्नलिखित समीकरण द्वारा परिभाषित होता है:
Q = [mA + mb - mC - md] c2
जहाँ दिए गए द्रव्यमान, नाभिकीय विराम द्रव्यमान हैं। दिए गए आँकड़ों के आधार पर बताइए कि निम्नलिखित अभिक्रियाएँ ऊष्माक्षेपी हैं या ऊष्माशोषी।
(i) H11 + H13 → H12 + H12
(ii) C612 + C612 → Ne1020 + He24
दिए गए परमाणु द्रव्यमान इस प्रकार हैं:
m(H11) = 1.007825 u, m(H12) = 2.014102 u, m(H13) = 3.016049 u,
m(C612) = 12.000000 u, m(Ne1020) = 19.992439 u, m(He24) = 4.002603 u
(i) अभिक्रिया: H11 + H13 → H12 + H12
Q = [m(H11) + m(H13) - 2 × m(H12)] × 931.5 MeV
Q = [1.007825 + 3.016049 - 2 × 2.014102] × 931.5 MeV
Q = [4.023874 - 4.028204] × 931.5 MeV
Q = [-0.00433] × 931.5 MeV
Q = -4.03 MeV
चूँकि Q ऋणात्मक है, अतः अभिक्रिया ऊष्माशोषी है।

(ii) अभिक्रिया: C612 + C612 → Ne1020 + He24
Q = [2 × m(C612) - m(Ne1020) - m(He24)] × 931.5 MeV
Q = [2 × 12.000000 - 19.992439 - 4.002603] × 931.5 MeV
Q = [24.000000 - 23.995042] × 931.5 MeV
Q = [0.004958] × 931.5 MeV
Q = 4.62 MeV
चूँकि Q धनात्मक है, अतः अभिक्रिया ऊष्माक्षेपी है।
