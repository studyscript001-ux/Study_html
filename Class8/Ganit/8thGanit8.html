प्रश्न 1: निम्नलिखित युग्मों को जोड़िए:

(i) ab - bc, bc - ca, ca - ab

(ab - bc) + (bc - ca) + (ca - ab)

समान पदों को समूहित करने पर:

(ab - ab) + (-bc + bc) + (-ca + ca)

0 + 0 + 0 = 0

उत्तर: 0

(ii) a - b + ab, b - c + bc, c - a + ac

(a - b + ab) + (b - c + bc) + (c - a + ac)

समान पदों को समूहित करने पर:

(a - a) + (-b + b) + (-c + c) + ab + bc + ac

0 + 0 + 0 + ab + bc + ac

उत्तर: ab + bc + ac

(iii) 2p²q² - 3pq + 4, 5 + 7pq - 3p²q²

(2p²q² - 3pq + 4) + (5 + 7pq - 3p²q²)

समान पदों को समूहित करने पर:

(2p²q² - 3p²q²) + (-3pq + 7pq) + (4 + 5)

-p²q² + 4pq + 9

उत्तर: -p²q² + 4pq + 9

(iv) l² + m², m² + n², n² + l², 2lm + 2mn + 2nl

(l² + m²) + (m² + n²) + (n² + l²) + (2lm + 2mn + 2nl)

समान पदों को समूहित करने पर:

(l² + l²) + (m² + m²) + (n² + n²) + 2lm + 2mn + 2nl

2l² + 2m² + 2n² + 2lm + 2mn + 2nl

2 से उभयनिष्ठ लेने पर:

उत्तर: 2(l² + m² + n² + lm + mn + nl)

प्रश्न 2: घटाइए:

(a) 12a - 9ab + 5b - 3 में से 4a - 7ab + 3b + 12 को घटाइए

(12a - 9ab + 5b - 3) - (4a - 7ab + 3b + 12)

कोष्ठक हटाने पर (दूसरे कोष्ठक के चिह्न बदलने पर):

12a - 9ab + 5b - 3 - 4a + 7ab - 3b - 12

समान पदों को समूहित करने पर:

(12a - 4a) + (-9ab + 7ab) + (5b - 3b) + (-3 - 12)

8a - 2ab + 2b - 15

उत्तर: 8a - 2ab + 2b - 15

(b) 5xy - 2yz - 2zx + 10xyz में से 3xy + 5yz - 7zx को घटाइए

(5xy - 2yz - 2zx + 10xyz) - (3xy + 5yz - 7zx)

कोष्ठक हटाने पर (दूसरे कोष्ठक के चिह्न बदलने पर):

5xy - 2yz - 2zx + 10xyz - 3xy - 5yz + 7zx

समान पदों को समूहित करने पर:

(5xy - 3xy) + (-2yz - 5yz) + (-2zx + 7zx) + 10xyz

2xy - 7yz + 5zx + 10xyz

उत्तर: 2xy - 7yz + 5zx + 10xyz

(c) 18 - 3p - 11q + 5pq - 2pq² + 5p²q में से 4p²q - 3pq + 5pq² - 8p + 7q - 10 को घटाइए

(18 - 3p - 11q + 5pq - 2pq² + 5p²q) - (4p²q - 3pq + 5pq² - 8p + 7q - 10)

कोष्ठक हटाने पर (दूसरे कोष्ठक के चिह्न बदलने पर):

18 - 3p - 11q + 5pq - 2pq² + 5p²q - 4p²q + 3pq - 5pq² + 8p - 7q + 10

समान पदों को समूहित करने पर:

(18 + 10) + (-3p + 8p) + (-11q - 7q) + (5pq + 3pq) + (-2pq² - 5pq²) + (5p²q - 4p²q)

28 + 5p - 18q + 8pq - 7pq² + p²q

उत्तर: 28 + 5p - 18q + 8pq - 7pq² + p²q

प्रश्न 4: ऐसे घनाकार बक्सों का आयतन ज्ञात कीजिए जिनकी लंबाई, चौड़ाई और ऊँचाई क्रमशः निम्नलिखित है:

(i) 5a, 3a², 7a⁴

घनाकार बक्से का आयतन = लंबाई × चौड़ाई × ऊँचाई

= 5a × 3a² × 7a⁴

= (5 × 3 × 7) × (a × a² × a⁴)

उत्तर: 105a⁷

(ii) 2p, 4q, 8r

= 64pqr

उत्तर: 64pqr

(iii) xy, 2x²y, 2xy²

= (1 × 2 × 2) × (x × x² × x) × (y × y × y²)

उत्तर: 4x⁴y⁴

(iv) a, 2b, 3c

= (1 × 2 × 3) × a × b × c

= 6abc

उत्तर: 6abc

प्रश्न 4:

(a) 3x(4x - 5) + 3 को सरल कीजिए और (i) x = 3, (ii) x = 1/2 के लिए इसका मान ज्ञात कीजिए

3x(4x - 5) + 3

= 12x² - 15x + 3

(i) x = 3 के लिए:

= 12(3)² - 15(3) + 3

= 12 × 9 - 45 + 3

= 108 - 45 + 3 = 66

(ii) x = 1/2 के लिए:

= 12(1/2)² - 15(1/2) + 3

= 12 × 1/4 - 15/2 + 3

= 3 - 7.5 + 3 = -1.5

उत्तर: सरलीकृत रूप: 12x² - 15x + 3, (i) 66, (ii) -1.5

(b) a(a² + a + 1) + 5 को सरल कीजिए और (i) a = 0, (ii) a = 1, (iii) a = -1 के लिए इसका मान ज्ञात कीजिए

a(a² + a + 1) + 5

= a³ + a² + a + 5

(i) a = 0 के लिए:

= 0³ + 0² + 0 + 5 = 5

(ii) a = 1 के लिए:

= 1³ + 1² + 1 + 5 = 1 + 1 + 1 + 5 = 8

(iii) a = -1 के लिए:

= (-1)³ + (-1)² + (-1) + 5 = -1 + 1 - 1 + 5 = 4

उत्तर: सरलीकृत रूप: a³ + a² + a + 5, (i) 5, (ii) 8, (iii) 4

प्रश्न 5:

(a) p(p - q), q(q - r) और r(r - p) को जोड़िए

p(p - q) = p² - pq

q(q - r) = q² - qr

r(r - p) = r² - rp

जोड़ने पर:

(p² - pq) + (q² - qr) + (r² - rp)

= p² + q² + r² - pq - qr - rp

उत्तर: p² + q² + r² - pq - qr - rp

(b) 2x(z - x - y) और 2y(z - y - x) को जोड़िए

2x(z - x - y) = 2xz - 2x² - 2xy

2y(z - y - x) = 2yz - 2y² - 2xy

जोड़ने पर:

(2xz - 2x² - 2xy) + (2yz - 2y² - 2xy)

= 2xz + 2yz - 2x² - 2y² - 4xy

= 2(xz + yz - x² - y² - 2xy)

उत्तर: 2xz + 2yz - 2x² - 2y² - 4xy

(c) 4l(10n - 3m + 2l) में से 3l(l - 4m + 5n) को घटाइए

4l(10n - 3m + 2l) = 40ln - 12lm + 8l²

3l(l - 4m + 5n) = 3l² - 12lm + 15ln

घटाने पर:

(40ln - 12lm + 8l²) - (3l² - 12lm + 15ln)

= 40ln - 12lm + 8l² - 3l² + 12lm - 15ln

= (40ln - 15ln) + (-12lm + 12lm) + (8l² - 3l²)

= 25ln + 0 + 5l²

= 5l² + 25ln

उत्तर: 5l² + 25ln

(d) 4c(-a + b + c) में से 3a(a + b + c) - 2b(a - b + c) को घटाइए

4c(-a + b + c) = -4ac + 4bc + 4c²

3a(a + b + c) = 3a² + 3ab + 3ac

2b(a - b + c) = 2ab - 2b² + 2bc

3a(a + b + c) - 2b(a - b + c) = (3a² + 3ab + 3ac) - (2ab - 2b² + 2bc)

= 3a² + 3ab + 3ac - 2ab + 2b² - 2bc

= 3a² + ab + 3ac + 2b² - 2bc

अब घटाने पर:

(-4ac + 4bc + 4c²) - (3a² + ab + 3ac + 2b² - 2bc)

= -4ac + 4bc + 4c² - 3a² - ab - 3ac - 2b² + 2bc

= -3a² - 2b² + 4c² - ab + (-4ac - 3ac) + (4bc + 2bc)

= -3a² - 2b² + 4c² - ab - 7ac + 6bc

उत्तर: -3a² - 2b² + 4c² - ab - 7ac + 6bc

प्रश्न 2: गुणनफल ज्ञात कीजिए:

(i) (5 - 2x)(3 + x)

(5 - 2x)(3 + x)

प्रश्न 3: सरल कीजिए:

(i) (x² - 5)(x + 5) + 25

(x² - 5)(x + 5) + 25

(ii) (a² + 5)(b³ + 3) + 5

(a + b)(c - d) + (a - b)(c + d) + 2(ac + bd)

प्रत्येक भाग को अलग-अलग हल करने पर:

(ac - ad + bc - bd) + (ac + ad - bc - bd) + (2ac + 2bd)

= 4ac + 0 + 0 + 0

(v) (x + y)(2x + y) + (x + 2y)(x - y)

(x + y)(2x + y) + (x + 2y)(x - y)

प्रत्येक भाग को अलग-अलग हल करने पर:

(x + y)(2x + y) = 2x² + xy + 2xy + y² = 2x² + 3xy + y²

(x + 2y)(x - y) = x² - xy + 2xy - 2y² = x² + xy - 2y²

दोनों को जोड़ने पर:

(2x² + 3xy + y²) + (x² + xy - 2y²)

= 3x² + 4xy - y²

उत्तर: 3x² + 4xy - y²

(vi) (x + y)(x² - xy + y²)

(x + y)(x² - xy + y²)

वितरण नियम का प्रयोग करने पर:

= x(x² - xy + y²) + y(x² - xy + y²)

= x³ - x²y + xy² + x²y - xy² + y³

= x³ + (-x²y + x²y) + (xy² - xy²) + y³

= x³ + y³

उत्तर: x³ + y³

(vii) (1.5x - 4y)(1.5x + 4y + 3) - 4.5x + 12y

(1.5x - 4y)(1.5x + 4y + 3) - 4.5x + 12y

वितरण नियम का प्रयोग करने पर:

= 1.5x(1.5x + 4y + 3) - 4y(1.5x + 4y + 3) - 4.5x + 12y

= 2.25x² + 6xy + 4.5x - 6xy - 16y² - 12y - 4.5x + 12y

= 2.25x² + (6xy - 6xy) + (4.5x - 4.5x) - 16y² + (-12y + 12y)

= 2.25x² - 16y²

उत्तर: 2.25x² - 16y²

(viii) (a + b + c)(a + b - c)

(a + b + c)(a + b - c)
