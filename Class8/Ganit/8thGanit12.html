अध्याय 12: गुणनखंडन

कक्षा 8 : गणित

प्रश्नावली 12.1

प्रश्न 1: दिए हुए पदों में सार्व गुणनखंड ज्ञात कीजिए:

हल:

(i) 12x, 36

12x = 2 × 2 × 3 × x

36 = 2 × 2 × 3 × 3

सार्व गुणनखंड = 2 × 2 × 3 = 12

उत्तर: 12

(ii) 2y, 22xy

2y = 2 × y

22xy = 2 × 11 × x × y

सार्व गुणनखंड = 2 × y = 2y

उत्तर: 2y

(iii) 14pq, 28p²q²

14pq = 2 × 7 × p × q

28p²q² = 2 × 2 × 7 × p × p × q × q

सार्व गुणनखंड = 2 × 7 × p × q = 14pq

उत्तर: 14pq

(iv) 2x, 3x², 4

2x = 2 × x

3x² = 3 × x × x

4 = 2 × 2

सार्व गुणनखंड = 1 (केवल 1 ही सार्व गुणनखंड है)

उत्तर: 1

(v) 6abc, 24ab², 12a²b

6abc = 2 × 3 × a × b × c

24ab² = 2 × 2 × 2 × 3 × a × b × b

12a²b = 2 × 2 × 3 × a × a × b

सार्व गुणनखंड = 2 × 3 × a × b = 6ab

उत्तर: 6ab

(vi) 16x³, -4x², 32x

16x³ = 2 × 2 × 2 × 2 × x × x × x

-4x² = -1 × 2 × 2 × x × x

32x = 2 × 2 × 2 × 2 × 2 × x

सार्व गुणनखंड = 2 × 2 × x = 4x

उत्तर: 4x

(vii) 10pq, 20qr, 30rp

10pq = 2 × 5 × p × q

20qr = 2 × 2 × 5 × q × r

30rp = 2 × 3 × 5 × r × p

सार्व गुणनखंड = 2 × 5 = 10

उत्तर: 10

(viii) 3x²y³, 10x³y², 6x²y²z

3x²y³ = 3 × x × x × y × y × y

10x³y² = 2 × 5 × x × x × x × y × y

6x²y²z = 2 × 3 × x × x × y × y × z

सार्व गुणनखंड = x × x × y × y = x²y²

उत्तर: x²y²

प्रश्न 2: निम्नलिखित व्यंजकों के गुणनखंड कीजिए:

हल:

(i) 7x - 42

7x = 7 × x

42 = 7 × 6

सार्व गुणनखंड = 7

7x - 42 = 7(x - 6)

उत्तर: 7(x - 6)

(ii) 6p - 12q

6p = 2 × 3 × p

12q = 2 × 2 × 3 × q

सार्व गुणनखंड = 2 × 3 = 6

6p - 12q = 6(p - 2q)

उत्तर: 6(p - 2q)

(iii) 7a² + 14a

7a² = 7 × a × a

14a = 2 × 7 × a

सार्व गुणनखंड = 7 × a = 7a

7a² + 14a = 7a(a + 2)

उत्तर: 7a(a + 2)

(iv) -16z + 20z³

-16z = -1 × 2 × 2 × 2 × 2 × z

20z³ = 2 × 2 × 5 × z × z × z

सार्व गुणनखंड = 2 × 2 × z = 4z

-16z + 20z³ = 4z(-4 + 5z²)

उत्तर: 4z(5z² - 4)

(v) 20l²m + 30alm

20l²m = 2 × 2 × 5 × l × l × m

30alm = 2 × 3 × 5 × a × l × m

सार्व गुणनखंड = 2 × 5 × l × m = 10lm

20l²m + 30alm = 10lm(2l + 3a)

उत्तर: 10lm(2l + 3a)

(vi) 5x²y - 15xy²

5x²y = 5 × x × x × y

15xy² = 3 × 5 × x × y × y

सार्व गुणनखंड = 5 × x × y = 5xy

5x²y - 15xy² = 5xy(x - 3y)

उत्तर: 5xy(x - 3y)

(vii) 10a² - 15b² + 20c²

10a² = 2 × 5 × a × a

15b² = 3 × 5 × b × b

20c² = 2 × 2 × 5 × c × c

सार्व गुणनखंड = 5

10a² - 15b² + 20c² = 5(2a² - 3b² + 4c²)

उत्तर: 5(2a² - 3b² + 4c²)

(viii) -4a² + 4ab - 4ca

-4a² = -1 × 2 × 2 × a × a

4ab = 2 × 2 × a × b

4ca = 2 × 2 × c × a

सार्व गुणनखंड = 2 × 2 × a = 4a

-4a² + 4ab - 4ca = 4a(-a + b - c)

उत्तर: 4a(-a + b - c) या -4a(a - b + c)

(ix) x²yz + xy²z + xyz²

x²yz = x × x × y × z

xy²z = x × y × y × z

xyz² = x × y × z × z

सार्व गुणनखंड = x × y × z = xyz

x²yz + xy²z + xyz² = xyz(x + y + z)

उत्तर: xyz(x + y + z)

(x) ax²y + bxy² + cxyz

ax²y = a × x × x × y

bxy² = b × x × y × y

cxyz = c × x × y × z

सार्व गुणनखंड = x × y = xy

ax²y + bxy² + cxyz = xy(ax + by + cz)

उत्तर: xy(ax + by + cz)

प्रश्न 3: गुणनखंड कीजिए:

हल:

(i) x² + xy + 8x + 8y

समूह बनाने पर: (x² + xy) + (8x + 8y)

x(x + y) + 8(x + y)

(x + y)(x + 8)

उत्तर: (x + y)(x + 8)

(ii) 15xy - 6x + 5y - 2

समूह बनाने पर: (15xy - 6x) + (5y - 2)

3x(5y - 2) + 1(5y - 2)

(5y - 2)(3x + 1)

उत्तर: (5y - 2)(3x + 1)

(iii) ax + bx - ay - by

समूह बनाने पर: (ax + bx) - (ay + by)

x(a + b) - y(a + b)

(a + b)(x - y)

उत्तर: (a + b)(x - y)

(iv) 15pq + 15 + 9q + 25p

पुनर्व्यवस्थित करने पर: 15pq + 9q + 25p + 15

समूह बनाने पर: (15pq + 9q) + (25p + 15)

3q(5p + 3) + 5(5p + 3)

(5p + 3)(3q + 5)

उत्तर: (5p + 3)(3q + 5)

(v) z - 7 + 7xy - xyz

पुनर्व्यवस्थित करने पर: -xyz + 7xy + z - 7

समूह बनाने पर: (-xyz + 7xy) + (z - 7)

-xy(z - 7) + 1(z - 7)

(z - 7)(1 - xy)

उत्तर: (z - 7)(1 - xy)

प्रश्नावली 12.2

प्रश्न 1: सर्वसमिकाओं का प्रयोग करके निम्नलिखित वर्गों के गुणनखंड कीजिए:

हल:

(i) a² + 8a + 16

यह (a + b)² = a² + 2ab + b² के रूप में है

यहाँ a = a, b = 4

a² + 8a + 16 = (a + 4)²

उत्तर: (a + 4)²

(ii) p² - 10p + 25

यह (a - b)² = a² - 2ab + b² के रूप में है

यहाँ a = p, b = 5

p² - 10p + 25 = (p - 5)²

उत्तर: (p - 5)²

(iii) 25m² + 30m + 9

यह (a + b)² = a² + 2ab + b² के रूप में है

यहाँ a = 5m, b = 3

25m² + 30m + 9 = (5m + 3)²

उत्तर: (5m + 3)²

(iv) 49y² + 84yz + 36z²

यह (a + b)² = a² + 2ab + b² के रूप में है

यहाँ a = 7y, b = 6z

49y² + 84yz + 36z² = (7y + 6z)²

उत्तर: (7y + 6z)²

(v) 4x² - 8x + 4

4 उभयनिष्ठ लेने पर: 4(x² - 2x + 1)

x² - 2x + 1 = (x - 1)²

4x² - 8x + 4 = 4(x - 1)²

उत्तर: 4(x - 1)²

(vi) 121b² - 88bc + 16c²

यह (a - b)² = a² - 2ab + b² के रूप में है

यहाँ a = 11b, b = 4c

121b² - 88bc + 16c² = (11b - 4c)²

उत्तर: (11b - 4c)²

(vii) (l + m)² - 4lm

(l + m)² - 4lm = l² + 2lm + m² - 4lm

= l² - 2lm + m²

= (l - m)²

उत्तर: (l - m)²

(viii) a⁴ + 2a²b² + b⁴

यह (a² + b²)² = a⁴ + 2a²b² + b⁴ के रूप में है

a⁴ + 2a²b² + b⁴ = (a² + b²)²

उत्तर: (a² + b²)²

प्रश्न 2: गुणनखंड कीजिए:

हल:

(i) 4p² - 9q²

यह a² - b² = (a + b)(a - b) के रूप में है

यहाँ a = 2p, b = 3q

4p² - 9q² = (2p + 3q)(2p - 3q)

उत्तर: (2p + 3q)(2p - 3q)

(ii) 63a² - 112b²

7 उभयनिष्ठ लेने पर: 7(9a² - 16b²)

9a² - 16b² = (3a)² - (4b)²

(3a + 4b)(3a - 4b)

63a² - 112b² = 7(3a + 4b)(3a - 4b)

उत्तर: 7(3a + 4b)(3a - 4b)

(iii) 49x² - 36

यह a² - b² = (a + b)(a - b) के रूप में है

यहाँ a = 7x, b = 6

49x² - 36 = (7x + 6)(7x - 6)

उत्तर: (7x + 6)(7x - 6)

(iv) 16x⁵ - 144x³

16x³ उभयनिष्ठ लेने पर: 16x³(x² - 9)

x² - 9 = (x + 3)(x - 3)

16x⁵ - 144x³ = 16x³(x + 3)(x - 3)

उत्तर: 16x³(x + 3)(x - 3)

(v) (l + m)² - (l - m)²

यह a² - b² = (a + b)(a - b) के रूप में है

यहाँ a = (l + m), b = (l - m)

(l + m)² - (l - m)² = [(l + m) + (l - m)][(l + m) - (l - m)]

= [l + m + l - m][l + m - l + m]

= [2l][2m] = 4lm

उत्तर: 4lm

(vi) 9x²y² - 16

यह a² - b² = (a + b)(a - b) के रूप में है

यहाँ a = 3xy, b = 4

9x²y² - 16 = (3xy + 4)(3xy - 4)

उत्तर: (3xy + 4)(3xy - 4)

(vii) (x² - 2xy + y²) - z²

x² - 2xy + y² = (x - y)²

(x - y)² - z² = [(x - y) + z][(x - y) - z]

= (x - y + z)(x - y - z)

उत्तर: (x - y + z)(x - y - z)

(viii) 25a² - 4b² + 28bc - 49c²

पुनर्व्यवस्थित करने पर: 25a² - (4b² - 28bc + 49c²)

4b² - 28bc + 49c² = (2b)² - 2(2b)(7c) + (7c)² = (2b - 7c)²

25a² - (2b - 7c)² = (5a)² - (2b - 7c)²

= [5a + (2b - 7c)][5a - (2b - 7c)]

= (5a + 2b - 7c)(5a - 2b + 7c)

उत्तर: (5a + 2b - 7c)(5a - 2b + 7c)

प्रश्न 3: निम्नलिखित व्यंजकों के गुणनखंड कीजिए:

हल:

(i) ax² + bx

x उभयनिष्ठ लेने पर: x(ax + b)

उत्तर: x(ax + b)

(ii) 7p² + 21q²

7 उभयनिष्ठ लेने पर: 7(p² + 3q²)

उत्तर: 7(p² + 3q²)

(iii) 2x³ + 2xy² + 2xz²

2x उभयनिष्ठ लेने पर: 2x(x² + y² + z²)

उत्तर: 2x(x² + y² + z²)

(iv) am² + bm² + bn² + an²

समूह बनाने पर: (am² + bm²) + (bn² + an²)

m²(a + b) + n²(a + b)

(a + b)(m² + n²)

उत्तर: (a + b)(m² + n²)

(v) (lm + l) + m + 1

समूह बनाने पर: l(m + 1) + 1(m + 1)

(m + 1)(l + 1)

उत्तर: (m + 1)(l + 1)

(vi) y(y + z) + 9(y + z)

(y + z) उभयनिष्ठ लेने पर: (y + z)(y + 9)

उत्तर: (y + z)(y + 9)

(vii) 5y² - 20y - 8z + 2yz

पुनर्व्यवस्थित करने पर: 5y² - 20y + 2yz - 8z

समूह बनाने पर: (5y² - 20y) + (2yz - 8z)

5y(y - 4) + 2z(y - 4)

(y - 4)(5y + 2z)

उत्तर: (y - 4)(5y + 2z)

(viii) 10ab + 4a + 5b + 2

समूह बनाने पर: (10ab + 4a) + (5b + 2)

2a(5b + 2) + 1(5b + 2)

(5b + 2)(2a + 1)

उत्तर: (5b + 2)(2a + 1)

(ix) 6xy - 4y + 6 - 9x

पुनर्व्यवस्थित करने पर: 6xy - 9x - 4y + 6

समूह बनाने पर: (6xy - 9x) - (4y - 6)

3x(2y - 3) - 2(2y - 3)

(2y - 3)(3x - 2)

उत्तर: (2y - 3)(3x - 2)

प्रश्न 4: गुणनखंड कीजिए:

हल:

(i) a⁴ - b⁴

(a²)² - (b²)² = (a² + b²)(a² - b²)

= (a² + b²)(a + b)(a - b)

उत्तर: (a² + b²)(a + b)(a - b)

(ii) p⁴ - 81

(p²)² - (9)² = (p² + 9)(p² - 9)

= (p² + 9)(p + 3)(p - 3)

उत्तर: (p² + 9)(p + 3)(p - 3)

(iii) x⁴ - (y + z)⁴

(x²)² - [(y + z)²]²

= [x² + (y + z)²][x² - (y + z)²]

= [x² + (y + z)²][x + (y + z)][x - (y + z)]

= [x² + (y + z)²][x + y + z][x - y - z]

उत्तर: [x² + (y + z)²][x + y + z][x - y - z]

(iv) x⁴ - (x - z)⁴

(x²)² - [(x - z)²]²

= [x² + (x - z)²][x² - (x - z)²]

= [x² + (x - z)²][x + (x - z)][x - (x - z)]

= [x² + (x - z)²][2x - z][z]

उत्तर: z(2x - z)[x² + (x - z)²]

(v) a⁴ - 2a²b² + b⁴

यह (a²)² - 2(a²)(b²) + (b²)² के रूप में है

= (a² - b²)²

= [(a + b)(a - b)]²

= (a + b)²(a - b)²

उत्तर: (a + b)²(a - b)²

प्रश्न 5: निम्नलिखित व्यंजकों के गुणनखंड कीजिए:

हल:

(i) p² + 6p + 8

हमें दो ऐसी संख्याएँ चाहिए जिनका योग 6 और गुणनफल 8 हो

4 और 2: 4 + 2 = 6, 4 × 2 = 8

p² + 6p + 8 = p² + 4p + 2p + 8

= p(p + 4) + 2(p + 4)

= (p + 4)(p + 2)

उत्तर: (p + 4)(p + 2)

(ii) q² - 10q + 21

हमें दो ऐसी संख्याएँ चाहिए जिनका योग -10 और गुणनफल 21 हो

-7 और -3: -7 + (-3) = -10, (-7) × (-3) = 21

q² - 10q + 21 = q² - 7q - 3q + 21

= q(q - 7) - 3(q - 7)

= (q - 7)(q - 3)

उत्तर: (q - 7)(q - 3)

(iii) p² + 6p - 16

हमें दो ऐसी संख्याएँ चाहिए जिनका योग 6 और गुणनफल -16 हो

8 और -2: 8 + (-2) = 6, 8 × (-2) = -16

p² + 6p - 16 = p² + 8p - 2p - 16

= p(p + 8) - 2(p + 8)

= (p + 8)(p - 2)

उत्तर: (p + 8)(p - 2)

प्रश्नावली 12.3

प्रश्न 1: निम्नलिखित विभाजन कीजिए:

हल:

(i) 28x⁴ ÷ 56x

= (28/56) × (x⁴/x)

= (1/2) × x³

= x³/2

उत्तर: x³/2

(ii) -36y³ ÷ 9y²

= (-36/9) × (y³/y²)

= -4 × y

= -4y

उत्तर: -4y

(iii) 66pq²r³ ÷ 11qr²

= (66/11) × (p/p⁰) × (q²/q) × (r³/r²)

= 6 × p × q × r

= 6pqr

उत्तर: 6pqr

(iv) 34x³y³z³ ÷ 51xy²z³

= (34/51) × (x³/x) × (y³/y²) × (z³/z³)

= (2/3) × x² × y × 1

= (2x²y)/3

उत्तर: (2x²y)/3

(v) 12a⁸b⁸ ÷ (-6a⁶b⁴)

= (12/-6) × (a⁸/a⁶) × (b⁸/b⁴)

= -2 × a² × b⁴

= -2a²b⁴

उत्तर: -2a²b⁴

प्रश्न 2: दिए हुए बहुपद को दिए हुए एकपदी से भाग दीजिए:

हल:

(i) (5x² - 6x) ÷ 3x

= (5x²/3x) - (6x/3x)

= (5x/3) - 2

उत्तर: (5x/3) - 2

(ii) (3y⁸ - 4y⁶ + 5y⁴) ÷ y⁴

= (3y⁸/y⁴) - (4y⁶/y⁴) + (5y⁴/y⁴)

= 3y⁴ - 4y² + 5

उत्तर: 3y⁴ - 4y² + 5

(iii) 8(x³y²z² + x²y³z² + x²y²z³) ÷ 4x²y²z²

= [8x³y²z²/(4x²y²z²)] + [8x²y³z²/(4x²y²z²)] + [8x²y²z³/(4x²y²z²)]

= 2x + 2y + 2z

= 2(x + y + z)

उत्तर: 2(x + y + z)

(iv) (x³ + 2x² + 3x) ÷ 2x

= (x³/2x) + (2x²/2x) + (3x/2x)

= (x²/2) + x + (3/2)

उत्तर: (x²/2) + x + (3/2)

(v) (p³q⁶ - p⁶q³) ÷ p³q³

= (p³q⁶/p³q³) - (p⁶q³/p³q³)

= q³ - p³

उत्तर: q³ - p³

प्रश्न 3: निम्नलिखित विभाजन कीजिए:

हल:

(i) (10x - 25) ÷ 5

= (10x/5) - (25/5)

= 2x - 5

उत्तर: 2x - 5

(ii) (10x - 25) ÷ (2x - 5)

5 उभयनिष्ठ लेने पर: 5(2x - 5)

[5(2x - 5)] ÷ (2x - 5) = 5

उत्तर: 5

(iii) 10y(6y + 21) ÷ 5(2y + 7)

10y(6y + 21) = 10y × 3(2y + 7) = 30y(2y + 7)

[30y(2y + 7)] ÷ [5(2y + 7)] = 6y

उत्तर: 6y

(iv) 9x²y²(3z - 24) ÷ 27xy(z - 8)

9x²y²(3z - 24) = 9x²y² × 3(z - 8) = 27x²y²(z - 8)

[27x²y²(z - 8)] ÷ [27xy(z - 8)] = xy

उत्तर: xy

(v) 96abc(3a - 12)(5b - 30) ÷ 144(a - 4)(b - 6)

96abc(3a - 12)(5b - 30) = 96abc × 3(a - 4) × 5(b - 6)

= 1440 abc (a - 4)(b - 6)

[1440 abc (a - 4)(b - 6)] ÷ [144(a - 4)(b - 6)] = 10abc

उत्तर: 10abc

प्रश्न 4: निर्देशानुसार भाग दीजिए:

हल:

(i) 5(2x + 1)(3x + 5) ÷ (2x + 1)

= 5(3x + 5)

उत्तर: 5(3x + 5)

(ii) 26xy(x + 5)(y - 4) ÷ 13x(y - 4)

= 2y(x + 5)

उत्तर: 2y(x + 5)

(iii) 52pqr(p + q)(q + r)(r + p) ÷ 104pq(q + r)(r + p)

= [52/104] × r × (p + q)

= (1/2) × r × (p + q)

= r(p + q)/2

उत्तर: r(p + q)/2

(iv) 20(y + 4)(y² + 5y + 3) ÷ 5(y + 4)

= 4(y² + 5y + 3)

उत्तर: 4(y² + 5y + 3)

(v) x(x + 1)(x + 2)(x + 3) ÷ x(x + 1)

= (x + 2)(x + 3)

उत्तर: (x + 2)(x + 3)

प्रश्न 5: व्यंजक के गुणनखंड कीजिए और निर्देशानुसार भाग दीजिए:

हल:

(i) (y² + 7y + 10) ÷ (y + 5)

y² + 7y + 10 = y² + 5y + 2y + 10

= y(y + 5) + 2(y + 5)

= (y + 5)(y + 2)

[(y + 5)(y + 2)] ÷ (y + 5) = y + 2

उत्तर: y + 2

(ii) (m² - 14m - 32) ÷ (m + 2)

m² - 14m - 32 = m² - 16m + 2m - 32

= m(m - 16) + 2(m - 16)

= (m - 16)(m + 2)

[(m - 16)(m + 2)] ÷ (m + 2) = m - 16

उत्तर: m - 16

(iii) (5p² - 25p + 20) ÷ (p - 1)

5p² - 25p + 20 = 5(p² - 5p + 4)

p² - 5p + 4 = p² - 4p - p + 4

= p(p - 4) - 1(p - 4)

= (p - 4)(p - 1)

5(p - 4)(p - 1) ÷ (p - 1) = 5(p - 4)

उत्तर: 5(p - 4)

(iv) 4yz(z² + 6z - 16) ÷ 2y(z + 8)

z² + 6z - 16 = z² + 8z - 2z - 16

= z(z + 8) - 2(z + 8)

= (z + 8)(z - 2)

4yz(z + 8)(z - 2) ÷ 2y(z + 8) = 2z(z - 2)

उत्तर: 2z(z - 2)

(v) 5pq(p² - q²) ÷ 2p(p + q)

p² - q² = (p + q)(p - q)

5pq(p + q)(p - q) ÷ 2p(p + q) = (5q/2)(p - q)

उत्तर: (5q/2)(p - q)

(vi) 12xy(9x² - 16y²) ÷ 4xy(3x + 4y)

9x² - 16y² = (3x)² - (4y)² = (3x + 4y)(3x - 4y)

12xy(3x + 4y)(3x - 4y) ÷ 4xy(3x + 4y) = 3(3x - 4y)

उत्तर: 3(3x - 4y)

(vii) 39y³(50y² - 98) ÷ 26y²(5y + 7)

50y² - 98 = 2(25y² - 49) = 2[(5y)² - (7)²] = 2(5y + 7)(5y - 7)

39y³ × 2(5y + 7)(5y - 7) = 78y³(5y + 7)(5y - 7)

78y³(5y + 7)(5y - 7) ÷ 26y²(5y + 7) = 3y(5y - 7)

उत्तर: 3y(5y - 7)
