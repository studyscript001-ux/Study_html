1. What will be the unit digit of the squares of the following numbers?

(i) 81 (ii) 272 (iii) 799 (iv) 3853 (v) 1234 (vi) 26387 (vii) 52698 (viii) 99880 (ix) 12796 (x) 55555

Solution:

We know that the unit digit of a square is the unit digit of the square of its unit digit.

(i) 81: Unit digit 1, 1² = 1 → Unit digit of square = 1

(ii) 272: Unit digit 2, 2² = 4 → Unit digit of square = 4

(iii) 799: Unit digit 9, 9² = 81 → Unit digit of square = 1

(iv) 3853: Unit digit 3, 3² = 9 → Unit digit of square = 9

(v) 1234: Unit digit 4, 4² = 16 → Unit digit of square = 6

(vi) 26387: Unit digit 7, 7² = 49 → Unit digit of square = 9

(vii) 52698: Unit digit 8, 8² = 64 → Unit digit of square = 4

(viii) 99880: Unit digit 0, 0² = 0 → Unit digit of square = 0

(ix) 12796: Unit digit 6, 6² = 36 → Unit digit of square = 6

(x) 55555: Unit digit 5, 5² = 25 → Unit digit of square = 5

Answer: (i) 1, (ii) 4, (iii) 1, (iv) 9, (v) 6, (vi) 9, (vii) 4, (viii) 0, (ix) 6, (x) 5
Hinglish Explanation: Kisi bhi number ka square ka unit digit sirf uska unit digit dekh kar pata kar sakte hain. Jaise 81 ka unit digit 1 hai, aur 1 square karo toh 1 aata hai. Isi tarah har number ke liye sirf unit digit ka square karo aur uska unit digit hi answer hoga.

2. The following numbers are obviously not perfect squares. Give reason.

(i) 1057 (ii) 23453 (iii) 7928 (iv) 222222 (v) 64000 (vi) 89722 (vii) 222000 (viii) 505050

Solution:

We know that a perfect square always ends with 0, 1, 4, 5, 6, or 9 at unit's place.

(i) 1057: Unit digit is 7, which is not in the list of possible unit digits for perfect squares.

(ii) 23453: Unit digit is 3, which is not in the list of possible unit digits for perfect squares.

(iii) 7928: Unit digit is 8, which is not in the list of possible unit digits for perfect squares.

(iv) 222222: Unit digit is 2, which is not in the list of possible unit digits for perfect squares.

(v) 64000: Unit digit is 0, but it doesn't have an even number of zeros. 64000 = 64 × 1000 = 8² × 10 × 10 × 10, which is not a perfect square.

(vi) 89722: Unit digit is 2, which is not in the list of possible unit digits for perfect squares.

(vii) 222000: Unit digit is 0, but it doesn't have an even number of zeros.

(viii) 505050: Unit digit is 0, but it doesn't have an even number of zeros.

Answer: All these numbers cannot be perfect squares because they either end with digits that perfect squares cannot end with, or they don't have even number of zeros at the end.
Hinglish Explanation: Perfect square numbers ka unit digit hamesha 0, 1, 4, 5, 6, ya 9 hi hota hai. Agar kisi number ka unit digit inme se nahi hai, toh wo perfect square nahi ho sakta. Saath hi, agar number zero pe khatam ho raha hai, toh zeros ki count even honi chahiye perfect square ke liye.

Exercise 5.2

1. Find the square of the following numbers:

(i) 32 (ii) 35 (iii) 86 (iv) 93 (v) 71 (vi) 46

Solution:

(i) 32² = (30 + 2)²
= 30(30+2) + 2(30+2)
= 30×30 + 30×2 + 2×30 + 2×2
= 30²+30×2+2×30+2²
= 900+60+60+4
= 1024

(ii) 35² = (30 + 5)²
= 30(30+5) + 5(30+5)
= 30² + 30×5 + 5×30 + 5²
= 900+150+150+25
= 1225

(iii) 86² = (80 + 6)²
= 80(80+6) + 6(80+6)
= 80² + 80×6 + 6×80 + 6²
= 6400+480+480+36
= 7396

(iv) 93² = (90 + 3)²
= 90(90+3) + 3(90+3)
= 90² + 90×3 + 3×90 + 3²
= 8100+270+270+9
= 8649

(v) 71² = (70 + 1)²
= 70(70+1) + 1(70+1)
= 70² + 70×1 + 1×70 + 1²
= 4900+70+70+1
= 5041

(vi) 46² = (40 + 6)²
= 40(40+6) + 6(40+6)
= 40² + 40×6 + 6×40 + 6²
= 1600+240+240+36
= 2116

Answer: (i) 1024, (ii) 1225, (iii) 7396, (iv) 8649, (v) 5041, (vi) 2116
Hinglish Explanation: Har number ko (a + b)² ke form me break karke square nikal sakte hain, jahan a nearest 10 ka multiple hai aur b remaining part. Jaise 32 = 30 + 2, toh (30+2)² = 30² + 2×30×2 + 2² = 900 + 120 + 4 = 1024. Same method se sabhi numbers ka square nikal sakte hain.

2. Write a Pythagorean triplet whose one member is:

(i) 6 (ii) 14 (iii) 16 (iv) 18

Solution:

Formula for Pythagorean triplet: 2m, m² - 1, m² + 1

(i) For 6:
Let 2m = 6 → m = 3
m² - 1 = 9 - 1 = 8
m² + 1 = 9 + 1 = 10
Triplet: 6, 8, 10

(ii) For 14:
Let 2m = 14 → m = 7
m² - 1 = 49 - 1 = 48
m² + 1 = 49 + 1 = 50
Triplet: 14, 48, 50

(iii) For 16:
Let 2m = 16 → m = 8
m² - 1 = 64 - 1 = 63
m² + 1 = 64 + 1 = 65
Triplet: 16, 63, 65

(iv) For 18:
Let 2m = 18 → m = 9
m² - 1 = 81 - 1 = 80
m² + 1 = 81 + 1 = 82
Triplet: 18, 80, 82

Answer: (i) 6, 8, 10; (ii) 14, 48, 50; (iii) 16, 63, 65; (iv) 18, 80, 82
Hinglish Explanation: Pythagorean triplet ke liye formula use kar sakte hain: 2m, m²-1, m²+1. Agar given number even hai, toh usko 2m maankar m find karenge. Phir m²-1 aur m²+1 calculate karke triplet complete karenge. Jaise 6 ke liye 2m=6 se m=3, phir m²-1=8 aur m²+1=10, triplet (6,8,10) hoga.

2. Find the number of digits in the square root of each of the following numbers (without any calculation):

(i) 64 (ii) 144 (iii) 4489 (iv) 27225 (v) 390625

Solution:

We know that if a number has n digits, then its square root has:

  • n/2 digits if n is even
  • (n+1)/2 digits if n is odd

(i) 64: 2 digits → Digits in square root = 2/2 = 1

(ii) 144: 3 digits → Digits in square root = (3+1)/2 = 2

(iii) 4489: 4 digits → Digits in square root = 4/2 = 2

(iv) 27225: 5 digits → Digits in square root = (5+1)/2 = 3

(v) 390625: 6 digits → Digits in square root = 6/2 = 3

Answer: (i) 1, (ii) 2, (iii) 2, (iv) 3, (v) 3
Hinglish Explanation: Square root me digits ki number pata karne ka formula simple hai: Agar number me n digits hain, toh square root me n/2 digits honge (agar n even hai) ya (n+1)/2 digits honge (agar n odd hai). Jaise 64 me 2 digits hain (even), toh square root me 2/2=1 digit hoga. Isi tarah 144 me 3 digits hain (odd), toh square root me (3+1)/2=2 digits honge.

7. In a right triangle ABC, ∠B = 90°

(a) If AB = 6 cm, BC = 8 cm, find AC

(b) If AC = 13 cm, BC = 5 cm, find AB

Solution:

By Pythagoras theorem: Hypotenuse² = Perpendicular² + Base²

(a) AB = 6 cm, BC = 8 cm, AC = ?
AC² = AB² + BC² = 6² + 8² = 36 + 64 = 100
AC = √100 = 10 cm

(b) AC = 13 cm, BC = 5 cm, AB = ?
AC² = AB² + BC²
13² = AB² + 5²
169 = AB² + 25
AB² = 169 - 25 = 144
AB = √144 = 12 cm

Answer: (a) 10 cm, (b) 12 cm
Hinglish Explanation: Right triangle me Pythagoras theorem use karte hain: (Hypotenuse)² = (Side1)² + (Side2)². Part (a) me AC hypotenuse hai, toh AC² = AB² + BC² = 6² + 8² = 36+64=100, AC=√100=10 cm. Part (b) me AC=13 cm diya hai, toh AB² = AC² - BC² = 169-25=144, AB=√144=12 cm.
