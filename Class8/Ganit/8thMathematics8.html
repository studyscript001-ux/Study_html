Question 1: Add the following pairs:

(i) ab - bc, bc - ca, ca - ab

(ab - bc) + (bc - ca) + (ca - ab)

Grouping like terms:

(ab - ab) + (-bc + bc) + (-ca + ca)

0 + 0 + 0 = 0

Answer: 0

Hinglish Explanation: Sabhi terms cancel ho jaati hain jab hum unhe add karte hain. Jaise ab - ab = 0, -bc + bc = 0, aur -ca + ca = 0. Isliye final answer 0 hai.

(ii) a - b + ab, b - c + bc, c - a + ac

(a - b + ab) + (b - c + bc) + (c - a + ac)

Grouping like terms:

(a - a) + (-b + b) + (-c + c) + ab + bc + ac

0 + 0 + 0 + ab + bc + ac

Answer: ab + bc + ac

Hinglish Explanation: a, b, c ke terms cancel ho jaate hain, lekin ab, bc, ac ke terms bach jaate hain. Isliye final answer ab + bc + ac hai.

(iii) 2p²q² - 3pq + 4, 5 + 7pq - 3p²q²

(2p²q² - 3pq + 4) + (5 + 7pq - 3p²q²)

Grouping like terms:

(2p²q² - 3p²q²) + (-3pq + 7pq) + (4 + 5)

-p²q² + 4pq + 9

Answer: -p²q² + 4pq + 9

Hinglish Explanation: p²q² ke terms combine hoke -p²q² ban jaate hain, pq ke terms combine hoke 4pq ban jaate hain, aur constant terms 4+5=9 ho jaate hain.

(iv) l² + m², m² + n², n² + l², 2lm + 2mn + 2nl

(l² + m²) + (m² + n²) + (n² + l²) + (2lm + 2mn + 2nl)

(l² + l²) + (m² + m²) + (n² + n²) + 2lm + 2mn + 2nl

2l² + 2m² + 2n² + 2lm + 2mn + 2nl

Taking 2 common:

Answer: 2(l² + m² + n² + lm + mn + nl)

Hinglish Explanation: Sabhi l², m², n² ke terms double ho jaate hain, aur hum 2 common le sakte hain. Isliye final answer 2(l² + m² + n² + lm + mn + nl) hai.
