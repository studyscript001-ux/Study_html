Question 1: In Fig. 6.13, lines AB and CD intersect at O. If ∠AOC + ∠BOE = 70° and ∠BOD = 40°, find ∠BOE and reflex ∠COE.

Solution:

Given: ∠AOC + ∠BOE = 70° and ∠BOD = 40°

From the figure, ∠AOC = ∠BOD (vertically opposite angles)

So, ∠AOC = 40°

Now, ∠AOC + ∠BOE = 70°

40° + ∠BOE = 70°

∠BOE = 70° - 40° = 30°

Now, ∠AOC + ∠COE + ∠BOE = 180° (angles on a straight line)

40° + ∠COE + 30° = 180°

∠COE = 180° - 70° = 110°

Reflex ∠COE = 360° - 110° = 250°

Answer: ∠BOE = 30°, Reflex ∠COE = 250°

Hinglish Explanation: Diya gaya hai ∠AOC + ∠BOE = 70° aur ∠BOD = 40°. Vertically opposite angles ke karan ∠AOC = ∠BOD = 40°. Isse ∠BOE = 30° milta hai. Straight line par angles 180° hote hain, isliye ∠COE = 110° aur reflex angle 360° - 110° = 250°.

Question 2: In Fig. 6.14, lines XY and MN intersect at O. If ∠POY = 90° and a : b = 2 : 3, find c.

Solution:

Given: ∠POY = 90° and a : b = 2 : 3

Let a = 2x and b = 3x

From the figure, ∠POY + ∠POX = 180° (linear pair)

90° + a + b = 180°

90° + 2x + 3x = 180°

5x = 90°

x = 18°

So, a = 2 × 18° = 36°, b = 3 × 18° = 54°

Now, ∠XON = ∠YOM = c (vertically opposite angles)

Also, ∠XON + ∠NOM = 180° (linear pair)

c + b = 180°

c + 54° = 180°

c = 180° - 54° = 126°

Answer: c = 126°

Hinglish Explanation: Diya gaya hai ∠POY = 90° aur a : b = 2 : 3. Linear pair ke karan 90° + a + b = 180°. a = 2x, b = 3x rakhne par x = 18° milta hai. Vertically opposite angles ke karan c = b nahi hai, balki linear pair ke karan c + b = 180°, isliye c = 126°.

Question 3: In Fig. 6.15, ∠PQR = ∠PRQ, then prove that ∠PQS = ∠PRT.

Solution:

Given: ∠PQR = ∠PRQ

To prove: ∠PQS = ∠PRT

∠PQS + ∠PQR = 180° (linear pair) ...(1)

∠PRT + ∠PRQ = 180° (linear pair) ...(2)

From (1) and (2):

∠PQS + ∠PQR = ∠PRT + ∠PRQ

But ∠PQR = ∠PRQ (given)

So, ∠PQS = ∠PRT

Hence proved.

Hinglish Explanation: Diya gaya hai ∠PQR = ∠PRQ. Linear pair ke karan ∠PQS + ∠PQR = 180° aur ∠PRT + ∠PRQ = 180°. Dono equations ko barabar rakhne par, aur ∠PQR = ∠PRQ substitute karne par, ∠PQS = ∠PRT milta hai.

Question 4: In Fig. 6.16, if x + y = w + z, then prove that AOB is a line.

Solution:

Given: x + y = w + z

To prove: AOB is a straight line

We know that the sum of all angles around a point is 360°

So, x + y + w + z = 360°

But x + y = w + z (given)

So, x + y + x + y = 360°

2(x + y) = 360°

x + y = 180°

This means that ∠AOC and ∠BOC form a linear pair

Therefore, AOB is a straight line

Hence proved.

Hinglish Explanation: Diya gaya hai x + y = w + z. Point O par sabhi angles ka sum 360° hota hai, isliye x + y + w + z = 360°. x + y = w + z substitute karne par 2(x + y) = 360°, yaani x + y = 180°. Iska matlab ∠AOC aur ∠BOC linear pair banate hain, isliye AOB ek straight line hai.

Question 5: In Fig. 6.17, POQ is a line. Ray OR is perpendicular to line PQ. OS is another ray lying between rays OP and OR. Prove that ∠ROS =
12
(∠QOS - ∠POS).

Solution:

Given: POQ is a straight line, OR ⊥ PQ

To prove: ∠ROS =

12
(∠QOS - ∠POS)

Since OR ⊥ PQ, ∠POR = 90° and ∠ROQ = 90°

∠QOS = ∠QOR + ∠ROS ...(1)

∠POS = ∠POR - ∠ROS ...(2)

Subtracting (2) from (1):

∠QOS - ∠POS = (∠QOR + ∠ROS) - (∠POR - ∠ROS)

= ∠QOR + ∠ROS - ∠POR + ∠ROS

= 90° + ∠ROS - 90° + ∠ROS

= 2∠ROS

Therefore, ∠ROS =

12
(∠QOS - ∠POS)

Hence proved.

Hinglish Explanation: Diya gaya hai POQ straight line hai aur OR ⊥ PQ. Iska matlab ∠POR = 90° aur ∠ROQ = 90°. ∠QOS = ∠QOR + ∠ROS aur ∠POS = ∠POR - ∠ROS. Dono equations ko subtract karne par ∠QOS - ∠POS = 2∠ROS milta hai, isliye ∠ROS = 1/2 (∠QOS - ∠POS).

Question 6: It is given that ∠XYZ = 64° and XY is produced to point P. Draw a figure from the given information. If ray YQ bisects ∠ZYP, find ∠XYQ and reflex ∠QYP.

Solution:

Given: ∠XYZ = 64°, XY produced to P, YQ bisects ∠ZYP

∠XYZ + ∠ZYP = 180° (linear pair)

64° + ∠ZYP = 180°

∠ZYP = 180° - 64° = 116°

Since YQ bisects ∠ZYP, ∠ZYQ = ∠QYP =

12
× 116° = 58°

Now, ∠XYQ = ∠XYZ + ∠ZYQ = 64° + 58° = 122°

Reflex ∠QYP = 360° - ∠QYP = 360° - 58° = 302°

Answer: ∠XYQ = 122°, Reflex ∠QYP = 302°

Hinglish Explanation: Diya gaya hai ∠XYZ = 64° aur XY ko P tak extend kiya gaya hai. Linear pair ke karan ∠XYZ + ∠ZYP = 180°, isliye ∠ZYP = 116°. YQ ∠ZYP ko bisect karta hai, isliye ∠ZYQ = ∠QYP = 58°. ∠XYQ = ∠XYZ + ∠ZYQ = 122°. Reflex ∠QYP = 360° - 58° = 302°.

Exercise 6.2

Question 1: In Fig. 6.23, if AB ∥ CD, CD ∥ EF and y : z = 3 : 7, find x.

Solution:

Given: AB||CD, CD ∥ EF, y : z = 3 : 7

Since CD ∥ EF, x + y = 180° (interior angles on same side of transversal)

Since AB ∥ CD, x = z (corresponding angles)

Let y = 3k and z = 7k

Then x = z = 7k

Now, x + y = 180°

7k + 3k = 180°

10k = 180°

k = 18°

So, x = 7k = 7 × 18° = 126°

Answer: x = 126°

Hinglish Explanation: Diya gaya hai AB ∥ CD, CD ∥ EF aur y : z = 3 : 7. CD ∥ EF hone ke karan x + y = 180° (same side interior angles). AB ∥ CD hone ke karan x = z (corresponding angles). y = 3k, z = 7k maan kar x = 7k milta hai. Equation 7k + 3k = 180° solve karne par k = 18°, isliye x = 126°.

Question 2: In Fig. 6.24, if AB ∥ CD, EF ⊥ CD and ∠GED = 126°, find ∠AGE, ∠GEF and ∠FGE.

Solution:

Given: AB ∥ CD, EF ⊥ CD, ∠GED = 126°

Since AB ∥ CD, ∠AGE = ∠GED = 126° (alternate interior angles)

EF ⊥ CD, so ∠FED = 90°

∠GED = ∠GEF + ∠FED

126° = ∠GEF + 90°

∠GEF = 126° - 90° = 36°

In triangle GEF, ∠GEF + ∠EGF + ∠FGE = 180°

But ∠EGF = 90° (since EF ⊥ CD and AB ∥ CD, so EF ⊥ AB)

36° + 90° + ∠FGE = 180°

∠FGE = 180° - 126° = 54°

Answer: ∠AGE = 126°, ∠GEF = 36°, ∠FGE = 54°

Hinglish Explanation: Diya gaya hai AB ∥ CD, EF ⊥ CD aur ∠GED = 126°. Alternate interior angles ke karan ∠AGE = ∠GED = 126°. EF ⊥ CD hone ke karan ∠FED = 90°, isliye ∠GEF = 126° - 90° = 36°. Triangle GEF mein angle sum property use karne par ∠FGE = 54° milta hai.

Question 3: In Fig. 6.25, if PQ ∥ ST, ∠PQR = 110° and ∠RST = 130°, find ∠QRS.

Solution:

Given: PQ ∥ ST, ∠PQR = 110°, ∠RST = 130°

Draw a line RU parallel to ST through point R

Since RU ∥ ST and PQ ∥ ST, so RU ∥ PQ

∠PQR + ∠QRU = 180° (interior angles on same side of transversal)

110° + ∠QRU = 180°

∠QRU = 70°

∠RST + ∠SRU = 180° (interior angles on same side of transversal)

130° + ∠SRU = 180°

∠SRU = 50°

∠QRS = ∠QRU + ∠SRU = 70° + 50° = 120°

Answer: ∠QRS = 120°

Hinglish Explanation: Diya gaya hai PQ ∥ ST, ∠PQR = 110° aur ∠RST = 130°. Point R se ST ke parallel ek line RU draw karte hain. RU ∥ ST aur PQ ∥ ST hone ke karan RU ∥ PQ. Interior angles on same side of transversal ke karan ∠QRU = 70° aur ∠SRU = 50°. Isliye ∠QRS = 70° + 50° = 120°.

Question 4: In Fig. 6.26, if AB ∥ CD, ∠APQ = 50° and ∠PRD = 127°, find x and y.

Solution:

Given: AB ∥ CD, ∠APQ = 50°, ∠PRD = 127°

Since AB ∥ CD, ∠APQ = ∠PQR = 50° (alternate interior angles)

So, x = 50°

Also, ∠PRD = ∠APR = 127° (alternate interior angles)

Now, ∠APR = ∠APQ + ∠QPR

127° = 50° + y

y = 127° - 50° = 77°

Answer: x = 50°, y = 77°

Hinglish Explanation: Diya gaya hai AB ∥ CD, ∠APQ = 50° aur ∠PRD = 127°. Alternate interior angles ke karan ∠APQ = ∠PQR = 50°, isliye x = 50°. Alternate interior angles ke karan ∠PRD = ∠APR = 127°. ∠APR = ∠APQ + ∠QPR, isliye 127° = 50° + y, y = 77°.

Question 5: In Fig. 6.27, PQ and RS are two mirrors placed parallel to each other. An incident ray AB strikes the mirror PQ at B, the reflected ray moves along the path BC and strikes the mirror RS at C and again reflects back along CD. Prove that AB ∥ CD.

Solution:

To prove: AB ∥ CD

Draw normals at points B and C

According to laws of reflection:

Angle of incidence = Angle of reflection

So, at point B: ∠ABN = ∠NBC

At point C: ∠BCM = ∠MCD

Since PQ ∥ RS, the normals BN and CM are parallel

So, ∠NBC = ∠BCM (alternate interior angles)

Therefore, ∠ABN = ∠MCD

But these are alternate interior angles for lines AB and CD with transversal BC

Hence, AB ∥ CD

Hence proved.

Hinglish Explanation: PQ aur RS do parallel mirrors hain. Reflection ke laws ke according angle of incidence = angle of reflection. Point B par ∠ABN = ∠NBC aur point C par ∠BCM = ∠MCD. PQ ∥ RS hone ke karan normals BN aur CM parallel hain, isliye ∠NBC = ∠BCM (alternate interior angles). Isse ∠ABN = ∠MCD milta hai, jo ki AB aur CD ke liye alternate interior angles hain, isliye AB ∥ CD.
