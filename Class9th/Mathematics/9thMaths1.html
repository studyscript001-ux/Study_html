Question 3: Show how √5 can be represented on the number line.

Solution:

Step 1: Draw a number line and mark point O at 0 and point A at 2.

Step 2: At point A, draw a perpendicular line of length 1 unit and mark the endpoint as B.

Step 3: Join O to B. Using Pythagoras theorem, OB = √(2² + 1²) = √5

Step 4: With O as center and OB as radius, draw an arc that intersects the number line at point C.

Step 5: Point C represents √5 on the number line.

Answer: √5 can be represented on the number line using geometric construction.

Hinglish Explanation: √5 ko number line par represent karne ke liye geometric construction use karte hain. Pehle 0 (O) aur 2 (A) mark karte hain. A se perpendicular line 1 unit ki banate hain (B). O se B ko join karte hain. Pythagoras theorem se OB = √5. Phir O ko center aur OB ko radius maankar arc banate hain jo number line ko C par cut karta hai. C √5 ko represent karta hai.

Question 4: Classroom activity (Constructing the 'square root spiral')

Solution:

Step 1: Start with a point O and draw a line segment OP₁ of unit length.

Step 2: Draw a line segment P₁P₂ perpendicular to OP₁ of unit length.

Step 3: Now draw a line segment P₂P₃ perpendicular to OP₂.

Step 4: Continue this process, each time drawing a line segment of unit length perpendicular to the previous hypotenuse.

Step 5: This creates a spiral where OP₂ = √2, OP₃ = √3, OP₄ = √4, and so on.

Answer: The square root spiral is constructed by repeatedly drawing perpendicular segments of unit length.

Hinglish Explanation: Square root spiral banane ke liye O se start karte hain aur OP₁ = 1 unit banate hain. Phir P₁ se perpendicular P₁P₂ = 1 unit banate hain. Is tarah har baar previous hypotenuse ke perpendicular ek unit length ka segment banate hain. Isse spiral banta hai jahan OP₂ = √2, OP₃ = √3, OP₄ = √4, aise increase hota hai.

Question 3: Recall, π is defined as the ratio of the circumference (say c) of a circle to its diameter (say d). That is, π = c/d. This seems to contradict the fact that π is irrational. How will you resolve this contradiction?

Solution:

There is no contradiction.

The definition π = c/d does not imply that π is rational.

For π to be rational, both c and d would need to be rational numbers.

However, for a circle, if d is rational, then c is irrational, and vice versa.

So, π = c/d is always irrational.

Answer: There is no contradiction because for a circle, if diameter is rational, circumference is irrational, making π irrational.

Hinglish Explanation: π = circumference/diameter definition se ye nahi samajhna chahiye ki π rational hai. Kisi circle ke liye, agar diameter rational hai toh circumference irrational hota hai, aur vice versa. Isliye π = c/d hamesha irrational hi hota hai. Koi contradiction nahi hai.

Question 5: Rationalise the denominators of the following:

Multiply numerator and denominator by √7:

1/√7 × √7/√7 = √7/7

1/(√7 - 2) × (√7 + 2)/(√7 + 2) = (√7 + 2)/(7 - 4) = (√7 + 2)/3

Answer: (√7 + 2)/3

Hinglish Explanation: Denominator ko rationalise karne ke liye hum numerator aur denominator ko appropriate expression se multiply karte hain taki denominator rational ho jaye.
