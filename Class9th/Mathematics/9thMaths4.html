Question 1: The cost of a notebook is twice the cost of a pen. Write a linear equation in two variables to represent this statement. (Take the cost of a notebook to be ₹ x and that of a pen to be ₹ y).

Solution:

Let cost of a notebook = ₹ x

Let cost of a pen = ₹ y

According to the problem:

Cost of notebook = 2 × Cost of pen

x = 2y

Rewriting in standard form:

x - 2y = 0

Answer: x - 2y = 0

Hinglish Explanation: Maan lete hain notebook ki cost ₹x aur pen ki cost ₹y hai. Question ke hisaab se, notebook ki cost pen ki cost se do guna hai. Iska matlab x = 2y. Standard form mein likhne par x - 2y = 0 milta hai.

Question 2: Express the following linear equations in the form ax + by + c = 0 and indicate the values of a, b and c in each case:

(i) 2x + 3y = 9.35

Given equation: 2x + 3y = 9.35

Rewriting in standard form:

2x + 3y - 9.35 = 0

Comparing with ax + by + c = 0:

a = 2, b = 3, c = -9.35

Answer: a = 2, b = 3, c = -9.35

Hinglish Explanation: Equation 2x + 3y = 9.35 ko standard form ax + by + c = 0 mein likhne ke liye sabko left side le jayenge: 2x + 3y - 9.35 = 0. Yahan a = 2, b = 3, c = -9.35.

Given equation: x - y/5 - 10 = 0

This is already in the form ax + by + c = 0

Comparing with ax + by + c = 0:

a = 1, b = -1/5, c = -10

Answer: a = 1, b = -1/5, c = -10

Hinglish Explanation: Equation x - y/5 - 10 = 0 already standard form mein hai. Yahan a = 1, b = -1/5, c = -10.

(iii) -2x + 3y = 6

Given equation: -2x + 3y = 6

Rewriting in standard form:

-2x + 3y - 6 = 0

Comparing with ax + by + c = 0:

Answer: a = -2, b = 3, c = -6

Hinglish Explanation: Equation -2x + 3y = 6 ko standard form mein likhne ke liye sabko left side le jayenge: -2x + 3y - 6 = 0. Yahan a = -2, b = 3, c = -6.

(iv) x = 3y

Given equation: x = 3y

Rewriting in standard form:

x - 3y = 0

Comparing with ax + by + c = 0:

a = 1, b = -3, c = 0

Hinglish Explanation: Equation x = 3y ko standard form mein likhne ke liye sabko left side le jayenge: x - 3y = 0. Yahan a = 1, b = -3, c = 0.

(v) 2x = -5y

Given equation: 2x = -5y

Hinglish Explanation: Equation 2x = -5y ko standard form mein likhne ke liye sabko left side le jayenge: 2x + 5y = 0. Yahan a = 2, b = 5, c = 0.

Hinglish Explanation: Equation y - 2 = 0 ko standard form ax + by + c = 0 mein likhne ke liye hum ise 0.x + 1.y - 2 = 0 likh sakte hain. Yahan a = 0, b = 1, c = -2.

(viii) 5 = 2x

a = 2, b = 0, c = -5

Exercise 4.2

Question 1: Which one of the following options is true, and why? y = 3x + 5 has

(i) a unique solution, (ii) only two solutions, (iii) infinitely many solutions

The equation y = 3x + 5 is a linear equation in two variables.

A linear equation in two variables has infinitely many solutions.

We can verify this by substituting different values of x:

When x = 0, y = 5 → (0, 5) is a solution

When x = 1, y = 8 → (1, 8) is a solution

When x = 2, y = 11 → (2, 11) is a solution

And so on...

Since we can find infinite pairs (x, y) that satisfy the equation, it has infinitely many solutions.

Answer: (iii) infinitely many solutions

Hinglish Explanation: Equation y = 3x + 5 ek linear equation hai do variables mein. Linear equations in two variables ke infinite solutions hote hain. Hum koi bhi x ki value lekar corresponding y ki value nikal sakte hain. Jaise x = 0 ke liye y = 5, x = 1 ke liye y = 8, x = 2 ke liye y = 11, aur aise hi infinite values mil sakti hain. Isliye iske infinite solutions hain.

Question 2: Write four solutions for each of the following equations:

(i) 2x + y = 7

Let's find four solutions for 2x + y = 7:

When x = 0:
2(0) + y = 7
→ y = 7
Solution: (0, 7)

When x = 1:
2(1) + y = 7
→ 2 + y = 7
→ y = 5
Solution: (1, 5)

When x = 2:
2(2) + y = 7
→ 4 + y = 7
→ y = 3
Solution: (2, 3)

When x = 3:
2(3) + y = 7
→ 6 + y = 7
→ y = 1
Solution: (3, 1)

Answer: (0, 7), (1, 5), (2, 3), (3, 1)

Hinglish Explanation: Equation 2x + y = 7 ke liye char solutions nikalne hain. Hum x ki different values lekar corresponding y ki values nikal sakte hain. Jaise x = 0 ke liye y = 7, x = 1 ke liye y = 5, x = 2 ke liye y = 3, x = 3 ke liye y = 1. Is tarah humein char solutions mil jaate hain: (0, 7), (1, 5), (2, 3), (3, 1).

(ii) πx + y = 9

Let's find four solutions for πx + y = 9:

When x = 0:
π(0) + y = 9
→ y = 9
→ Solution: (0, 9)

When x = 1:
π(1) + y = 9
→ π + y = 9
→ y = 9 - π → Solution: (1, 9 - π)

When x = 2:
π(2) + y = 9
→ 2π + y = 9
→ y = 9 - 2π → Solution: (2, 9 - 2π)

When x = 3:
π(3) + y = 9
→ 3π + y = 9
→ y = 9 - 3π → Solution: (3, 9 - 3π)

Answer: (0, 9), (1, 9 - π), (2, 9 - 2π), (3, 9 - 3π)

Hinglish Explanation: Equation πx + y = 9 ke liye char solutions nikalne hain. Hum x ki different values lekar corresponding y ki values nikal sakte hain. Jaise x = 0 ke liye y = 9, x = 1 ke liye y = 9 - π, x = 2 ke liye y = 9 - 2π, x = 3 ke liye y = 9 - 3π. Is tarah humein char solutions mil jaate hain: (0, 9), (1, 9 - π), (2, 9 - 2π), (3, 9 - 3π).

(iii) x = 4y

Let's find four solutions for x = 4y:

When y = 0:
x = 4(0) = 0
→ Solution: (0, 0)

When y = 1:
x = 4(1) = 4
→ Solution: (4, 1)

When y = 2:
x = 4(2) = 8
→ Solution: (8, 2)

When y = 3:
x = 4(3) = 12
→ Solution: (12, 3)

Answer: (0, 0), (4, 1), (8, 2), (12, 3)

Hinglish Explanation: Equation x = 4y ke liye char solutions nikalne hain. Hum y ki different values lekar corresponding x ki values nikal sakte hain. Jaise y = 0 ke liye x = 0, y = 1 ke liye x = 4, y = 2 ke liye x = 8, y = 3 ke liye x = 12. Is tarah humein char solutions mil jaate hain: (0, 0), (4, 1), (8, 2), (12, 3).

Question 3: Check which of the following are solutions of the equation x - 2y = 4 and which are not:

(i) (0, 2)

Substitute x = 0, y = 2 in x - 2y = 4:

0 - 2(2) = 0 - 4 = -4 ≠ 4

-4 is not equal to 4

Answer: Not a solution

Hinglish Explanation: (0, 2) ko equation x - 2y = 4 mein substitute karenge: 0 - 2(2) = -4, jo 4 ke barabar nahi hai. Isliye yeh solution nahi hai.

(ii) (2, 0)

Substitute x = 2, y = 0 in x - 2y = 4:

2 - 2(0) = 2 - 0 = 2 ≠ 4

2 is not equal to 4

Answer: Not a solution

Hinglish Explanation: (2, 0) ko equation x - 2y = 4 mein substitute karenge: 2 - 2(0) = 2, jo 4 ke barabar nahi hai. Isliye yeh solution nahi hai.

(iii) (4, 0)

Substitute x = 4, y = 0 in x - 2y = 4:

4 - 2(0) = 4 - 0 = 4 = 4

4 is equal to 4

Answer: Is a solution

Hinglish Explanation: (4, 0) ko equation x - 2y = 4 mein substitute karenge: 4 - 2(0) = 4, jo 4 ke barabar hai. Isliye yeh solution hai.

(iv) (√2, 4√2)

Substitute x = √2, y = 4√2 in x - 2y = 4:

√2 - 2(4√2) = √2 - 8√2 = -7√2 ≠ 4

-7√2 is not equal to 4

Answer: Not a solution

Hinglish Explanation: (√2, 4√2) ko equation x - 2y = 4 mein substitute karenge: √2 - 2(4√2) = √2 - 8√2 = -7√2, jo 4 ke barabar nahi hai. Isliye yeh solution nahi hai.

(v) (1, 1)

Substitute x = 1, y = 1 in x - 2y = 4:

1 - 2(1) = 1 - 2 = -1 ≠ 4

-1 is not equal to 4

Answer: Not a solution

Hinglish Explanation: (1, 1) ko equation x - 2y = 4 mein substitute karenge: 1 - 2(1) = -1, jo 4 ke barabar nahi hai. Isliye yeh solution nahi hai.

Question 4: Find the value of k, if x = 2, y = 1 is a solution of the equation 2x + 3y = k.

Solution:

Given: x = 2, y = 1 is a solution of 2x + 3y = k

Substitute x = 2, y = 1 in the equation:

2(2) + 3(1) = k

4 + 3 = k

7 = k

Answer: k = 7

Hinglish Explanation: Kyunki x = 2, y = 1 equation 2x + 3y = k ka solution hai, isliye in values ko equation mein substitute karenge: 2(2) + 3(1) = 4 + 3 = 7. Isliye k ki value 7 hai.
