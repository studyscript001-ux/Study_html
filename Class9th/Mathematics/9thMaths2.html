Question 5: Factorise:
(i) x3 - 2x2 - x + 2
Group the terms: (x3 - 2x2) + (-x + 2)
= x2(x - 2) - 1(x - 2)
= (x - 2)(x2 - 1)
= (x - 2)(x - 1)(x + 1)
Answer: (x - 2)(x - 1)(x + 1)
(ii) x3 - 3x2 - 9x - 5
Try x = -1: (-1)3 - 3(-1)2 - 9(-1) - 5 = -1 - 3 + 9 - 5 = 0
So (x + 1) is a factor.
Divide x3 - 3x2 - 9x - 5 by x + 1
We get x2 - 4x - 5
Factorise x2 - 4x - 5 = (x - 5)(x + 1)
So x3 - 3x2 - 9x - 5 = (x + 1)(x - 5)(x + 1) = (x + 1)2(x - 5)
Answer: (x + 1)2(x - 5)
(iii) x3 + 13x2 + 32x + 20
Try x = -1: (-1)3 + 13(-1)2 + 32(-1) + 20 = -1 + 13 - 32 + 20 = 0
So (x + 1) is a factor.
Divide x3 + 13x2 + 32x + 20 by x + 1
We get x2 + 12x + 20
Factorise x2 + 12x + 20 = (x + 2)(x + 10)
So x3 + 13x2 + 32x + 20 = (x + 1)(x + 2)(x + 10)
Answer: (x + 1)(x + 2)(x + 10)
(iv) 2y3 + y2 - 2y - 1
Group the terms: (2y3 + y2) + (-2y - 1)
= y2(2y + 1) - 1(2y + 1)
= (2y + 1)(y2 - 1)
= (2y + 1)(y - 1)(y + 1)
Answer: (2y + 1)(y - 1)(y + 1)

Question 9: Verify:
(i) x3 + y3 = (x + y)(x2 - xy + y2)
RHS: (x + y)(x2 - xy + y2) = x(x2 - xy + y2) + y(x2 - xy + y2)
= x3 - x2y + xy2 + x2y - xy2 + y3 = x3 + y3
LHS = RHS, hence verified.
Answer: Verified
(ii) x3 - y3 = (x - y)(x2 + xy + y2)
RHS: (x - y)(x2 + xy + y2) = x(x2 + xy + y2) - y(x2 + xy + y2)
= x3 + x2y + xy2 - x2y - xy2 - y3 = x3 - y3
LHS = RHS, hence verified.
Answer: Verified
Question 10: Factorise each of the following:
(i) 27y3 + 125z3
Using identity: a3 + b3 = (a + b)(a2 - ab + b2)
Here, a = 3y, b = 5z
27y3 + 125z3 = (3y)3 + (5z)3 = (3y + 5z)[(3y)2 - (3y)(5z) + (5z)2] = (3y + 5z)(9y2 - 15yz + 25z2)
Answer: (3y + 5z)(9y2 - 15yz + 25z2)
(ii) 64m3 - 343n3
Using identity: a3 - b3 = (a - b)(a2 + ab + b2)
Here, a = 4m, b = 7n
64m3 - 343n3 = (4m)3 - (7n)3 = (4m - 7n)[(4m)2 + (4m)(7n) + (7n)2] = (4m - 7n)(16m2 + 28mn + 49n2)
Answer: (4m - 7n)(16m2 + 28mn + 49n2)
Question 11: Factorise: 27x3 + y3 + z3 - 9xyz
Using identity: a3 + b3 + c3 - 3abc = (a + b + c)(a2 + b2 + c2 - ab - bc - ca)
Here, a = 3x, b = y, c = z
27x3 + y3 + z3 - 9xyz = (3x)3 + y3 + z3 - 3(3x)(y)(z) = (3x + y + z)[(3x)2 + y2 + z2 - (3x)(y) - (y)(z) - (z)(3x)]
= (3x + y + z)(9x2 + y2 + z2 - 3xy - yz - 3zx)
Answer: (3x + y + z)(9x2 + y2 + z2 - 3xy - yz - 3zx)

Question 15: Give possible expressions for the length and breadth of each of the following rectangles, in which their areas are given:
Area: 25a2 - 35a + 12
Factorise 25a2 - 35a + 12
Split the middle term: -35a
Find two numbers whose product is 25 × 12 = 300 and sum is -35
The numbers are -20 and -15
25a2 - 35a + 12 = 25a2 - 20a - 15a + 12 = 5a(5a - 4) - 3(5a - 4) = (5a - 4)(5a - 3)
So, possible length and breadth are (5a - 4) and (5a - 3)
Answer: Length = 5a - 4, Breadth = 5a - 3 (or vice versa)

Answer: Length = 5y + 4, Breadth = 7y - 3 (or vice versa)
