Chapter 7
Binomial Theorem

Class 11th Mathematics

Introduction

The Binomial Theorem provides a way to expand expressions of the form (a + b)n without having to multiply repeatedly. It is especially useful for higher powers where direct multiplication becomes tedious.

The binomial theorem states that for any positive integer n:

(a + b)n = nC0 an + nC1 an-1b + nC2 an-2b2 + ... + nCn bn

Where nCr = n! / (r! × (n - r)!)

Hinglish Explanation: Binomial theorem humein (a + b)n ko expand karne ka ek aasan tarika batata hai, khaas kar jab n ki value badi ho. Isme hum coefficients ke liye combinations ka use karte hain.

Question 1: Expand (1 - 2x)5

Solution:

Using the binomial theorem:

(a + b)n = nC0 an + nC1 an-1b + nC2 an-2b2 + ... + nCn bn

Here, a = 1, b = -2x, n = 5

(1 - 2x)5 = 5C0 (1)5 + 5C1 (1)4 (-2x) + 5C2 (1)3 (-2x)2 + 5C3 (1)2 (-2x)3 + 5C4 (1) (-2x)4 + 5C5 (-2x)5

Calculate the binomial coefficients:

5C0 = 1, 5C1 = 5, 5C2 = 10, 5C3 = 10, 5C4 = 5, 5C5 = 1

Now substitute and simplify:

= 1(1) + 5(1)(-2x) + 10(1)(4x2) + 10(1)(-8x3) + 5(1)(16x4) + 1(-32x5)

= 1 - 10x + 40x2 - 80x3 + 80x4 - 32x5

Answer: 1 - 10x + 40x2 - 80x3 + 80x4 - 32x5

Hinglish Explanation: Binomial theorem use karke (1-2x)5 ko expand karna hai. Binomial coefficients calculate karke (5C0 se 5C5 tak), fir har term ko simplify karte hain.

Question 10: Using Binomial Theorem, indicate which number is larger (1.1)10000 or 1000

Solution:

We can write (1.1)10000 as (1 + 0.1)10000

Using the binomial theorem:

(1 + 0.1)10000 = 10000C0 + 10000C1 (0.1) + 10000C2 (0.1)2 + ... + 10000C10000 (0.1)10000

The first two terms are:

= 1 + 10000 × 0.1 + other positive terms

= 1 + 1000 + other positive terms

= 1001 + other positive terms

Since all terms are positive, (1.1)10000 > 1001 > 1000

Answer: (1.1)10000 is larger than 1000

Hinglish Explanation: (1.1)10000 ko (1+0.1)10000 likh kar binomial expansion karte hain. First two terms 1+1000=1001 hain, aur baaki terms positive hain, isliye ye 1000 se bada hai.

Question 11: Find (a + b)4 - (a - b)4. Hence, evaluate (√3 + √2)4 - (√3 - √2)4

Solution:

First, find (a + b)4 - (a - b)4

Using binomial theorem:

(a + b)4 = 4C0 a4 + 4C1 a3b + 4C2 a2b2 + 4C3 a b3 + 4C4 b4

= a4 + 4a3b + 6a2b2 + 4a b3 + b4

(a - b)4 = 4C0 a4 - 4C1 a3b + 4C2 a2b2 - 4C3 a b3 + 4C4 b4

= a4 - 4a3b + 6a2b2 - 4a b3 + b4

Now subtract:

(a + b)4 - (a - b)4 = (a4 + 4a3b + 6a2b2 + 4a b3 + b4) - (a4 - 4a3b + 6a2b2 - 4a b3 + b4)

= 8a3b + 8a b3

= 8ab(a2 + b2)

Now evaluate (√3 + √2)4 - (√3 - √2)4

Here, a = √3, b = √2

So, the expression = 8 × √3 × √2 × (3 + 2)

= 8 × √6 × 5

= 40√6

Answer: 40√6

Hinglish Explanation: Pehle (a+b)4 - (a-b)4 ko simplify karte hain, jo 8ab(a2+b2) aata hai. Fir is formula mein a=√3 aur b=√2 substitute karke final value nikalte hain.

Question 12: Find (x + 1)6 + (x - 1)6. Hence or otherwise evaluate (√2 + 1)6 + (√2 - 1)6

Solution:

First, find (x + 1)6 + (x - 1)6

Using binomial theorem:

(x + 1)6 = 6C0 x6 + 6C1 x5 + 6C2 x4 + 6C3 x3 + 6C4 x2 + 6C5 x + 6C6

= x6 + 6x5 + 15x4 + 20x3 + 15x2 + 6x + 1

(x - 1)6 = 6C0 x6 - 6C1 x5 + 6C2 x4 - 6C3 x3 + 6C4 x2 - 6C5 x + 6C6

= x6 - 6x5 + 15x4 - 20x3 + 15x2 - 6x + 1

Now add:

(x + 1)6 + (x - 1)6 = (x6 + 6x5 + 15x4 + 20x3 + 15x2 + 6x + 1) + (x6 - 6x5 + 15x4 - 20x3 + 15x2 - 6x + 1)

= 2x6 + 30x4 + 30x2 + 2

= 2(x6 + 15x4 + 15x2 + 1)

Now evaluate (√2 + 1)6 + (√2 - 1)6

Here, x = √2

So, the expression = 2[(√2)6 + 15(√2)4 + 15(√2)2 + 1]

= 2[(23) + 15(22) + 15(2) + 1]

= 2[8 + 15×4 + 30 + 1]

= 2[8 + 60 + 30 + 1]

= 2 × 99 = 198

Answer: 198

Hinglish Explanation: Pehle (x+1)6 + (x-1)6 ko simplify karte hain, jo 2(x6+15x4+15x2+1) aata hai. Fir is formula mein x=√2 substitute karke final value nikalte hain.

Question 13: Show that 9n+1 - 8n - 9 is divisible by 64, whenever n is a positive integer.

Solution:

We can write 9n+1 as (8 + 1)n+1

Using binomial theorem:

(8 + 1)n+1 = n+1C0 8n+1 + n+1C1 8n + n+1C2 8n-1 + ... + n+1Cn 8 + n+1Cn+1

So, 9n+1 = (8 + 1)n+1 = 8n+1 + (n+1)8n + n+1C2 8n-1 + ... + (n+1)8 + 1

Now, 9n+1 - 8n - 9 = [8n+1 + (n+1)8n + n+1C2 8n-1 + ... + (n+1)8 + 1] - 8n - 9

= 8n+1 + (n+1)8n + n+1C2 8n-1 + ... + (n+1)8 + 1 - 8n - 9

= 8n+1 + (n+1)8n + n+1C2 8n-1 + ... + 8n + 8 + 1 - 8n - 9

= 8n+1 + (n+1)8n + n+1C2 8n-1 + ... + 82 × k (where k is some integer)

Since each term has at least 82 = 64 as a factor, the entire expression is divisible by 64.

Answer: Hence proved that 9n+1 - 8n - 9 is divisible by 64 for all positive integers n.

Hinglish Explanation: 9n+1 ko (8+1)n+1 likh kar binomial expansion karte hain. Fir 9n+1 - 8n - 9 ko simplify karte hain. Har term mein 64 ka factor hota hai, isliye pure expression ko 64 divide kar sakte hain.

Question 14: Prove that nΣr=0 3r nCr = 4n

Solution:

We know the binomial theorem:

(1 + x)n = nC0 + nC1 x + nC2 x2 + ... + nCn xn

Substitute x = 3:

(1 + 3)n = nC0 + nC1 (3) + nC2 (3)2 + ... + nCn (3)n

4n = nC0 + 3 nC1 + 32 nC2 + ... + 3n nCn

This can be written as:

4n = nΣr=0 3r nCr

Answer: Hence proved that nΣr=0 3r nCr = 4n

Hinglish Explanation: Binomial theorem mein (1+x)n ki expansion ko use karte hain. x=3 substitute karte hain, toh (1+3)n = 4n milta hai, jo exactly nΣr=0 3r nCr ke equal hai.
