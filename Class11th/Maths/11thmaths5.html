प्रश्न 1: हल कीजिए 24x < 100, जब
(i) x एक प्राकृतिक संख्या है
(ii) x एक पूर्णांक है

हल:

हमारे पास है: 24x < 100

दोनों पक्षों को 24 से विभाजित करने पर: x < 100 24 = 25 6 ≈ 4.1667

(i) जब x एक प्राकृतिक संख्या है:

4.1667 से छोटी प्राकृतिक संख्याएँ हैं: 1, 2, 3, 4

हल समुच्चय: {1, 2, 3, 4}

(ii) जब x एक पूर्णांक है:

4.1667 से छोटे पूर्णांक हैं: ..., -3, -2, -1, 0, 1, 2, 3, 4

हल समुच्चय: {..., -3, -2, -1, 0, 1, 2, 3, 4}

उत्तर: (i) {1, 2, 3, 4}; (ii) {..., -3, -2, -1, 0, 1, 2, 3, 4}

सरल हिंदी व्याख्या: पहले असमिका हल करते हैं: 24x < 100 ⇒ x < 25/6 ≈ 4.1667। (i) प्राकृतिक संख्याओं के लिए: 1, 2, 3, 4। (ii) पूर्णांकों के लिए: ...-3, -2, -1, 0, 1, 2, 3, 4.

प्रश्न 2: हल कीजिए -12x > 30, जब
(i) x एक प्राकृतिक संख्या है
(ii) x एक पूर्णांक है

हल:

हमारे पास है: -12x > 30

दोनों पक्षों को -12 से विभाजित करने पर (चिह्न बदल जाएगा): x < 30 -12 = - 5 2 = -2.5

(i) जब x एक प्राकृतिक संख्या है:

-2.5 से छोटी कोई प्राकृतिक संख्या नहीं है।

हल समुच्चय: ∅ (रिक्त समुच्चय)

(ii) जब x एक पूर्णांक है:

-2.5 से छोटे पूर्णांक हैं: ..., -6, -5, -4, -3

हल समुच्चय: {..., -6, -5, -4, -3}

उत्तर: (i) ∅; (ii) {..., -6, -5, -4, -3}

सरल हिंदी व्याख्या: असमिका हल करते हैं: -12x > 30 ⇒ x < -2.5 (ऋणात्मक से भाग देने पर चिह्न बदलता है)। (i) प्राकृतिक संख्याओं में -2.5 से छोटी कोई संख्या नहीं, इसलिए रिक्त समुच्चय। (ii) पूर्णांकों के लिए: ...-6, -5, -4, -3.

प्रश्न 3: हल कीजिए 5x - 3 < 7, जब
(i) x एक पूर्णांक है
(ii) x एक वास्तविक संख्या है

हल:

हमारे पास है: 5x - 3 < 7

दोनों पक्षों में 3 जोड़ने पर: 5x < 10

दोनों पक्षों को 5 से विभाजित करने पर: x < 2

(i) जब x एक पूर्णांक है:

2 से छोटे पूर्णांक हैं: ..., -3, -2, -1, 0, 1

हल समुच्चय: {..., -3, -2, -1, 0, 1}

(ii) जब x एक वास्तविक संख्या है:

2 से छोटी सभी वास्तविक संख्याएँ

हल समुच्चय: (-∞, 2)

उत्तर: (i) {..., -3, -2, -1, 0, 1}; (ii) (-∞, 2)

सरल हिंदी व्याख्या: असमिका हल करते हैं: 5x - 3 < 7 ⇒ 5x < 10 ⇒ x < 2। (i) पूर्णांकों के लिए: ...-3, -2, -1, 0, 1। (ii) वास्तविक संख्याओं के लिए: (-∞, 2).

प्रश्न 4: हल कीजिए 3x + 8 > 2, जब
(i) x एक पूर्णांक है
(ii) x एक वास्तविक संख्या है

हल:

हमारे पास है: 3x + 8 > 2

दोनों पक्षों में से 8 घटाने पर: 3x > -6

दोनों पक्षों को 3 से विभाजित करने पर: x > -2

(i) जब x एक पूर्णांक है:

-2 से बड़े पूर्णांक हैं: -1, 0, 1, 2, 3, ...

हल समुच्चय: {-1, 0, 1, 2, 3, ...}

(ii) जब x एक वास्तविक संख्या है:

-2 से बड़ी सभी वास्तविक संख्याएँ

हल समुच्चय: (-2, ∞)

उत्तर: (i) {-1, 0, 1, 2, 3, ...}; (ii) (-2, ∞)

सरल हिंदी व्याख्या: असमिका हल करते हैं: 3x + 8 > 2 ⇒ 3x > -6 ⇒ x > -2। (i) पूर्णांकों के लिए: -1, 0, 1, 2, 3, ... (ii) वास्तविक संख्याओं के लिए: (-2, ∞).

प्रश्न 5: वास्तविक x के लिए हल कीजिए: 4x + 3 < 5x + 7

हल:

हमारे पास है: 4x + 3 < 5x + 7

दोनों पक्षों में से 4x घटाने पर: 3 < x + 7

दोनों पक्षों में से 7 घटाने पर: -4 < x ⇒ x > -4

हल समुच्चय: (-4, ∞)

उत्तर: (-4, ∞)

सरल हिंदी व्याख्या: 4x + 3 < 5x + 7 ⇒ 3 < x + 7 ⇒ -4 < x या x > -4। हल समुच्चय: (-4, ∞).

प्रश्न 6: वास्तविक x के लिए हल कीजिए: 3x - 7 > 5x - 1

हल:

3x - 7 > 5x - 1 ⇒ -7 > 2x - 1 ⇒ -6 > 2x ⇒ -3 > x ⇒ x < -3

हल समुच्चय: (-∞, -3)

उत्तर: (-∞, -3)

सरल हिंदी व्याख्या: 3x - 7 > 5x - 1 ⇒ -6 > 2x ⇒ x < -3। हल समुच्चय: (-∞, -3).

प्रश्न 7: वास्तविक x के लिए हल कीजिए: 3(x - 1) ≤ 2(x - 3)

हल:

3x - 3 ≤ 2x - 6 ⇒ x - 3 ≤ -6 ⇒ x ≤ -3

हल समुच्चय: (-∞, -3]

उत्तर: (-∞, -3]

सरल हिंदी व्याख्या: 3(x-1) ≤ 2(x-3) ⇒ 3x-3 ≤ 2x-6 ⇒ x ≤ -3। हल समुच्चय: (-∞, -3].

प्रश्न 8: वास्तविक x के लिए हल कीजिए: 3(2 - x) ≥ 2(1 - x)

हल:

6 - 3x ≥ 2 - 2x ⇒ 6 ≥ 2 + x ⇒ 4 ≥ x ⇒ x ≤ 4

हल समुच्चय: (-∞, 4]

उत्तर: (-∞, 4]

सरल हिंदी व्याख्या: 3(2-x) ≥ 2(1-x) ⇒ 6-3x ≥ 2-2x ⇒ 4 ≥ x ⇒ x ≤ 4। हल: (-∞, 4].

प्रश्न 9: वास्तविक x के लिए हल कीजिए: x + x2 + x3 < 11

हल:

x + x/2 + x/3 < 11 ⇒ (6x+3x+2x)/6 < 11 ⇒ 11x/6 < 11 ⇒ 11x < 66 ⇒ x < 6

हल समुच्चय: (-∞, 6)

उत्तर: (-∞, 6)

सरल हिंदी व्याख्या: x + x/2 + x/3 < 11 ⇒ 11x/6 < 11 ⇒ x < 6। हल: (-∞, 6).

प्रश्न 10: हल कीजिए: x3 > x2 + 1

हल:

x/3 - x/2 > 1 ⇒ (2x-3x)/6 > 1 ⇒ -x/6 > 1 ⇒ -x > 6 ⇒ x < -6

हल समुच्चय: (-∞, -6)

उत्तर: (-∞, -6)

सरल हिंदी व्याख्या: x/3 > x/2 + 1 ⇒ -x/6 > 1 ⇒ x < -6। हल: (-∞, -6).

प्रश्न 11: हल कीजिए: 3(x-2)5 ≤ 5(2-x)3

उत्तर: (-∞, 2]

प्रश्न 12: हल कीजिए: 12(3x5+4) ≥ 13(x-6)

3x/10 + 2 ≥ x/3 - 2 ⇒ 3x/10 - x/3 ≥ -4 ⇒ (9x-10x)/30 ≥ -4 ⇒ -x/30 ≥ -4 ⇒ -x ≥ -120 ⇒ x ≤ 120

हल समुच्चय: (-∞, 120]

उत्तर: (-∞, 120]

सरल हिंदी व्याख्या: असमिका को हल करने पर x ≤ 120 प्राप्त होता है।

प्रश्न 21: रवि ने पहली दो इकाई परीक्षाओं में 70 और 75 अंक प्राप्त किए। तीसरी परीक्षा में उसे कम से कम कितने अंक प्राप्त करने चाहिए ताकि उसका औसत कम से कम 60 अंक हो?

हल:

माना तीसरी परीक्षा में x अंक मिले। औसत = (70+75+x)/3 ≥ 60 ⇒ 145+x ≥ 180 ⇒ x ≥ 35

तीसरी परीक्षा में न्यूनतम अंक = 35

उत्तर: 35

सरल हिंदी व्याख्या: (70+75+x)/3 ≥ 60 ⇒ x ≥ 35. अतः कम से कम 35 अंक चाहिए।

प्रश्न 22: एक पाठ्यक्रम में 'A' ग्रेड पाने के लिए पांच परीक्षाओं (प्रत्येक 100 अंकों की) में कम से कम 90 अंकों का औसत आवश्यक है। यदि सुनिता के पहली चार परीक्षाओं के अंक 87, 92, 94, 95 हैं तो पाँचवीं परीक्षा में न्यूनतम कितने अंक लाने होंगे?

हल:

माना पाँचवीं में x अंक। औसत = (87+92+94+95+x)/5 ≥ 90 ⇒ 368+x ≥ 450 ⇒ x ≥ 82

न्यूनतम अंक = 82

उत्तर: 82

सरल हिंदी व्याख्या: (368+x)/5 ≥ 90 ⇒ x ≥ 82.

प्रश्न 23: क्रमागत विषम धनात्मक पूर्णांकों के सभी युग्म ज्ञात कीजिए जिनमें प्रत्येक 10 से छोटा है और जिनका योग 11 से अधिक है।

हल:

माना क्रमागत विषम पूर्णांक x तथा x+2 हैं। शर्तें: x+2 < 10 ⇒ x < 8, तथा x+(x+2) > 11 ⇒ 2x+2 > 11 ⇒ x > 4.5, तथा x विषम धनात्मक ⇒ x = 5, 7

युग्म: (5,7) और (7,9)

उत्तर: (5,7) और (7,9)

सरल हिंदी व्याख्या: शर्तों के अनुसार x = 5, 7 अतः युग्म (5,7) व (7,9).

प्रश्न 24: क्रमागत सम धनात्मक पूर्णांकों के सभी युग्म ज्ञात कीजिए जिनमें प्रत्येक 5 से बड़ा है तथा योग 23 से कम है।

हल:

माना x तथा x+2 सम पूर्णांक हैं। x > 5 तथा 2x+2 < 23 ⇒ x < 10.5, x सम धनात्मक ⇒ x = 6, 8, 10

उत्तर: (6,8), (8,10), (10,12)

सरल हिंदी व्याख्या: शर्तों से x = 6,8,10 अतः उपरोक्त युग्म।

प्रश्न 25: एक त्रिभुज की सबसे बड़ी भुजा सबसे छोटी भुजा की तीन गुनी है तथा तीसरी भुजा सबसे बड़ी भुजा से 2 सेमी कम है। यदि त्रिभुज का परिमाप कम से कम 61 सेमी है तो सबसे छोटी भुजा की न्यूनतम लंबाई ज्ञात कीजिए।

हल:

माना सबसे छोटी भुजा x सेमी। तब सबसे बड़ी भुजा = 3x, तीसरी = 3x-2। परिमाप = x+3x+(3x-2)=7x-2। प्रतिबंध: 7x-2 ≥ 61 ⇒ 7x ≥ 63 ⇒ x ≥ 9

न्यूनतम लंबाई = 9 सेमी

उत्तर: 9 सेमी

सरल हिंदी व्याख्या: 7x-2 ≥ 61 ⇒ x ≥ 9, अतः न्यूनतम 9 सेमी।

प्रश्न 26: एक व्यक्ति 91 सेमी लंबे बोर्ड से तीन टुकड़े काटना चाहता है। दूसरा टुकड़ा सबसे छोटे टुकड़े से 3 सेमी अधिक लंबा है तथा तीसरा टुकड़ा सबसे छोटे का दोगुना है। यदि तीसरा टुकड़ा दूसरे टुकड़े से कम से कम 5 सेमी अधिक लंबा हो तो सबसे छोटे टुकड़े की संभावित लंबाइयाँ क्या हैं?

हल:

माना सबसे छोटा टुकड़ा x सेमी। दूसरा = x+3, तीसरा = 2x। कुल लंबाई ≤ 91 ⇒ 4x+3 ≤ 91 ⇒ x ≤ 22। तीसरा ≥ दूसरा + 5 ⇒ 2x ≥ x+8 ⇒ x ≥ 8।

इसलिए 8 ≤ x ≤ 22

उत्तर: 8 सेमी से 22 सेमी तक

सरल हिंदी व्याख्या: x ≥ 8 और x ≤ 22, अतः सबसे छोटे टुकड़े की लंबाई 8 से 22 सेमी के बीच।
