त्रिविमीय ज्यामिति का परिचय

📘 प्रश्नावली 11.1
1. एक बिंदु x-अक्ष पर स्थित है। इसके y-निर्देशांक तथा z-निर्देशांक क्या हैं?
हम जानते हैं कि x-अक्ष पर स्थित किसी भी बिंदु के लिए y और z निर्देशांक हमेशा शून्य (0) होते हैं।
अतः बिंदु के निर्देशांक (x, 0, 0) के रूप में होंगे।
इसलिए, y-निर्देशांक = 0 और z-निर्देशांक = 0
स्पष्टीकरण: x-अक्ष पर बिंदु केवल x दिशा में आगे बढ़ता है, ऊपर-नीचे (y) या आगे-पीछे (z) नहीं। इसलिए y और z दोनों 0 होते हैं।
2. एक बिंदु XZ-तल में है। इसके y-निर्देशांक के बारे में आप क्या कह सकते हैं?
XZ-तल वह तल है जो x-अक्ष और z-अक्ष से मिलकर बनता है।
इस तल पर स्थित किसी भी बिंदु के लिए y-अक्ष की दिशा में कोई विस्थापन नहीं होता है।
अतः XZ-तल पर स्थित किसी भी बिंदु का y-निर्देशांक हमेशा शून्य (0) होता है।
स्पष्टीकरण: जिस तल का नाम दिया होता है (जैसे XZ), उसे छोड़कर तीसरा निर्देशांक (यहाँ y) हमेशा 0 होता है।
3. उन अष्टांशों (octants) के नाम बताइए जिनमें निम्नलिखित बिंदु स्थित हैं: (1, 2, 3), (4, -2, 3), (4, -2, -5), (4, 2, -5), (-4, 2, 5), (-4, 2, -5), (-4, -2, 5), (-4, -2, -5).
बिंदुओं के चिह्नों (+, -, +, आदि) के आधार पर अष्टांश निर्धारित किए जाते हैं:
1. (1, 2, 3) → (+, +, +) → प्रथम (I) अष्टांश
2. (4, -2, 3) → (+, -, +) → चतुर्थ (IV) अष्टांश
3. (4, -2, -5) → (+, -, -) → अष्टम (VIII) अष्टांश
4. (4, 2, -5) → (+, +, -) → पंचम (V) अष्टांश
5. (-4, 2, 5) → (-, +, +) → द्वितीय (II) अष्टांश
6. (-4, 2, -5) → (-, +, -) → षष्ठ (VI) अष्टांश
7. (-4, -2, 5) → (-, -, +) → तृतीय (III) अष्टांश
8. (-4, -2, -5) → (-, -, -) → सप्तम (VII) अष्टांश
स्पष्टीकरण: अष्टांश (Octants) त्रिविमीय ज्यामिति में 8 भाग होते हैं। इनके चिह्न तालिका के अनुसार याद रखे जा सकते हैं।
4. रिक्त स्थानों की पूर्ति कीजिए:
(i) x-अक्ष और y-अक्ष दोनों एक साथ मिल कर एक तल बनाते हैं, उस तल को XY-तल कहते हैं।
(ii) XY-तल में एक बिंदु के निर्देशांक (x, y, 0) रूप के होते हैं।
(iii) निर्देशांक तल अंतरिक्ष को आठ (8) अष्टांशों में विभाजित करते हैं।
स्पष्टीकरण: यह त्रिविमीय ज्यामिति की बुनियादी परिभाषाएँ हैं।
📗 प्रश्नावली 11.2
1. निम्नलिखित बिंदु-युग्मों के बीच की दूरी ज्ञात कीजिए:
सूत्र: दूरी d = √((x2 - x1)2 + (y2 - y1)2 + (z2 - z1)2)
(i) (2, 3, 5) और (4, 3, 1)
d = √((4 - 2)2 + (3 - 3)2 + (1 - 5)2)
d = √((2)2 + (0)2 + (-4)2) = √(4 + 0 + 16)
d = √20 = 2√5 इकाई
(ii) (-3, 7, 2) और (2, 4, -1)
d = √((2 - (-3))2 + (4 - 7)2 + (-1 - 2)2)
d = √((5)2 + (-3)2 + (-3)2) = √(25 + 9 + 9)
d = √43 इकाई
(iii) (-1, 3, -4) और (1, -3, 4)
d = √((1 - (-1))2 + (-3 - 3)2 + (4 - (-4))2)
d = √((2)2 + (-6)2 + (8)2) = √(4 + 36 + 64)
d = √104 = 2√26 इकाई
(iv) (2, -1, 3) और (-2, 1, 3)
d = √((-2 - 2)2 + (1 - (-1))2 + (3 - 3)2)
d = √((-4)2 + (2)2 + (0)2) = √(16 + 4 + 0)
d = √20 = 2√5 इकाई
स्पष्टीकरण: त्रिविमीय दूरी सूत्र 2D दूरी सूत्र का ही विस्तार है जिसमें z-निर्देशांक का अंतर भी जुड़ जाता है।
2. दर्शाइए कि बिंदु (-2, 3, 5), (1, 2, 3) और (7, 0, -1) सरेख (collinear) हैं।
माना बिंदु A(-2, 3, 5), B(1, 2, 3) और C(7, 0, -1) हैं।
AB के बीच की दूरी = √((1 - (-2))2 + (2 - 3)2 + (3 - 5)2) = √(32 + (-1)2 + (-2)2) = √(9 + 1 + 4) = √14
BC के बीच की दूरी = √((7 - 1)2 + (0 - 2)2 + (-1 - 3)2) = √(62 + (-2)2 + (-4)2) = √(36 + 4 + 16) = √56 = 2√14
AC के बीच की दूरी = √((7 - (-2))2 + (0 - 3)2 + (-1 - 5)2) = √(92 + (-3)2 + (-6)2) = √(81 + 9 + 36) = √126 = 3√14
यहाँ हम देखते हैं कि AB + BC = √14 + 2√14 = 3√14 = AC
चूँकि दो दूरियों का योग तीसरी दूरी के बराबर है, अतः दिए गए बिंदु A, B और C सरेख (Collinear) हैं।
स्पष्टीकरण: तीन बिंदु एक ही रेखा (collinear) पर तब होते हैं जब किन्हीं दो दूरियों का योग सबसे बड़ी दूरी के बराबर होता है।
3. निम्नलिखित को सत्यापित कीजिए:
(i) (0, 7, -10), (1, 6, -6) और (4, 9, -6) एक समद्विबाहु त्रिभुज के शीर्ष हैं।
माना A(0, 7, -10), B(1, 6, -6) और C(4, 9, -6) हैं।
AB = √((1-0)2 + (6-7)2 + (-6 - (-10))2) = √(1 + 1 + 16) = √18 = 3√2
BC = √((4-1)2 + (9-6)2 + (-6 - (-6))2) = √(9 + 9 + 0) = √18 = 3√2
AC = √((4-0)2 + (9-7)2 + (-6 - (-10))2) = √(16 + 4 + 16) = √36 = 6
चूँकि AB = BC (√18 = √18), अतः यह एक समद्विबाहु (Isosceles) त्रिभुज है।
(ii) (0, 7, 10), (-1, 6, 6) और (-4, 9, 6) एक समकोण त्रिभुज के शीर्ष हैं।
माना P(0, 7, 10), Q(-1, 6, 6) और R(-4, 9, 6) हैं।
PQ2 = (-1-0)2 + (6-7)2 + (6-10)2 = 1 + 1 + 16 = 18
QR2 = (-4 - (-1))2 + (9-6)2 + (6-6)2 = (-3)2 + 32 + 0 = 9 + 9 = 18
PR2 = (-4-0)2 + (9-7)2 + (6-10)2 = 16 + 4 + 16 = 36
चूँकि PQ2 + QR2 = 18 + 18 = 36 = PR2, अतः पाइथागोरस प्रमेय से यह एक समकोण (Right-angled) त्रिभुज है।
(iii) (-1, 2, 1), (1, -2, 5), (4, -7, 8) और (2, -3, 4) एक समांतर चतुर्भुज के शीर्ष हैं।
माना A(-1, 2, 1), B(1, -2, 5), C(4, -7, 8) और D(2, -3, 4) हैं।
AB = √((1 - (-1))2 + (-2-2)2 + (5-1)2) = √(4 + 16 + 16) = √36 = 6
BC = √((4-1)2 + (-7 - (-2))2 + (8-5)2) = √(9 + 25 + 9) = √43
CD = √((2-4)2 + (-3 - (-7))2 + (4-8)2) = √(4 + 16 + 16) = √36 = 6
DA = √((-1-2)2 + (2 - (-3))2 + (1-4)2) = √(9 + 25 + 9) = √43
चूँकि विपरीत भुजाएँ बराबर हैं (AB = CD और BC = DA), अतः यह एक समांतर चतुर्भुज (Parallelogram) है।
स्पष्टीकरण: दूरी सूत्र की मदद से भुजाओं की लंबाइयाँ निकाली गई हैं। समद्विबाहु के लिए 2 भुजाएँ बराबर, समकोण के लिए पाइथागोरस प्रमेय और समांतर चतुर्भुज के लिए आमने-सामने की भुजाएँ बराबर दिखाई गई हैं।
4. ऐसे बिंदुओं के समुच्चय का समीकरण ज्ञात कीजिए जो बिंदु (1, 2, 3) और (3, 2, -1) से समदूरस्थ (equidistant) हैं।
माना ऐसा कोई बिंदु P(x, y, z) है। दिए गए बिंदु A(1, 2, 3) और B(3, 2, -1) हैं।
प्रश्नानुसार, PA = PB
दोनों पक्षों का वर्ग करने पर: PA2 = PB2
(x - 1)2 + (y - 2)2 + (z - 3)2 = (x - 3)2 + (y - 2)2 + (z - (-1))2
समीकरण को खोलने पर:
(x2 - 2x + 1) + (y2 - 4y + 4) + (z2 - 6z + 9) = (x2 - 6x + 9) + (y2 - 4y + 4) + (z2 + 2z + 1)
दोनों तरफ से x2, y2, z2 और (y2 - 4y + 4) को काटने पर:
-2x - 6z + 10 = -6x + 2z + 10
-2x + 6x - 6z - 2z = 10 - 10
4x - 8z = 0
4 से भाग देने पर: x - 2z = 0
स्पष्टीकरण: बिंदु P की A और B से दूरियों को बराबर रखकर (PA=PB) समीकरण बनाया गया। वर्ग करने से अंडररूट हट गया और x, y, z में एक सरल रैखिक (linear) समीकरण प्राप्त हुआ।
5. बिंदुओं P से बने समुच्चय का समीकरण ज्ञात कीजिए जिनकी A (4, 0, 0) और B (-4, 0, 0) से दूरियों का योगफल 10 है।
माना बिंदु P के निर्देशांक (x, y, z) हैं। प्रश्नानुसार, PA + PB = 10
√((x - 4)2 + y2 + z2) + √((x + 4)2 + y2 + z2) = 10
√((x - 4)2 + y2 + z2) = 10 - √((x + 4)2 + y2 + z2)
दोनों पक्षों का वर्ग करने पर:
(x - 4)2 + y2 + z2 = 100 + (x + 4)2 + y2 + z2 - 20√((x + 4)2 + y2 + z2)
x2 - 8x + 16 + y2 + z2 = 100 + x2 + 8x + 16 + y2 + z2 - 20√((x + 4)2 + y2 + z2)
समान पदों को काटने पर: -8x = 100 + 8x - 20√((x + 4)2 + y2 + z2)
-16x - 100 = -20√((x + 4)2 + y2 + z2)
-4 से भाग देने पर: 4x + 25 = 5√((x + 4)2 + y2 + z2)
पुनः वर्ग करने पर: (4x + 25)2 = 25[(x + 4)2 + y2 + z2]
16x2 + 200x + 625 = 25(x2 + 8x + 16 + y2 + z2)
16x2 + 200x + 625 = 25x2 + 200x + 400 + 25y2 + 25z2
सभी पदों को एक तरफ लाने पर: 25x2 - 16x2 + 25y2 + 25z2 - 625 + 400 = 0
9x2 + 25y2 + 25z2 - 225 = 0
स्पष्टीकरण: इस प्रश्न में अंडररूट (square root) को हटाने के लिए दो बार वर्ग (squaring both sides) करना पड़ा। यह एक दीर्घवृत्त (Ellipsoid) का समीकरण है।
📕 विविध प्रश्नावली
1. एक समांतर चतुर्भुज ABCD के तीन शीर्ष A (3, -1, 2), B (1, 2, -4) और C (-1, 1, 2) हैं। चौथे शीर्ष D के निर्देशांक ज्ञात कीजिए।
माना चौथे शीर्ष D के निर्देशांक (x, y, z) हैं।
हम जानते हैं कि समांतर चतुर्भुज के विकर्ण (diagonals) AC और BD एक-दूसरे को समद्विभाजित (bisect) करते हैं।
अतः विकर्ण AC का मध्य-बिंदु = विकर्ण BD का मध्य-बिंदु
AC का मध्य-बिंदु = (3 + (-1)2, -1 + 12, 2 + 22) = (22, 02, 42) = (1, 0, 2)
BD का मध्य-बिंदु = (1 + x2, 2 + y2, -4 + z2)
दोनों की तुलना करने पर:
1 + x2 = 1 ⇒ 1 + x = 2 ⇒ x = 1
2 + y2 = 0 ⇒ 2 + y = 0 ⇒ y = -2
-4 + z2 = 2 ⇒ -4 + z = 4 ⇒ z = 8
अतः चौथे शीर्ष D के निर्देशांक (1, -2, 8) हैं।
स्पष्टीकरण: मध्य-बिंदु सूत्र (Midpoint formula) का प्रयोग करके अज्ञात निर्देशांक (x,y,z) बहुत आसानी से निकाले जा सकते हैं।
2. एक त्रिभुज ABC के शीर्ष A (0, 0, 6), B (0, 4, 0) और C (6, 0, 0) हैं। इसकी माध्यिकाओं (medians) की लंबाइयाँ ज्ञात कीजिए।
माध्यिका वह रेखा है जो शीर्ष को सामने वाली भुजा के मध्य-बिंदु से मिलाती है।
1. माध्यिका AD की लंबाई: बिंदु D, भुजा BC का मध्य-बिंदु है।
D के निर्देशांक = (0 + 62, 4 + 02, 0 + 02) = (3, 2, 0)
AD = √((3 - 0)2 + (2 - 0)2 + (0 - 6)2) = √(9 + 4 + 36) = √49 = 7
2. माध्यिका BE की लंबाई: बिंदु E, भुजा AC का मध्य-बिंदु है।
E के निर्देशांक = (0 + 62, 0 + 02, 6 + 02) = (3, 0, 3)
BE = √((3 - 0)2 + (0 - 4)2 + (3 - 0)2) = √(9 + 16 + 9) = √34
3. माध्यिका CF की लंबाई: बिंदु F, भुजा AB का मध्य-बिंदु है।
F के निर्देशांक = (0 + 02, 0 + 42, 6 + 02) = (0, 2, 3)
CF = √((0 - 6)2 + (2 - 0)2 + (3 - 0)2) = √(36 + 4 + 9) = √49 = 7
स्पष्टीकरण: त्रिभुज की माध्यिकाएँ निकालने के लिए पहले प्रत्येक भुजा का मध्य बिंदु (Midpoint) निकाला गया, फिर शीर्ष से उस मध्य-बिंदु तक की दूरी निकाली गई।
3. यदि त्रिभुज PQR का केंद्रक (centroid) मूल बिंदु (0, 0, 0) है और शीर्ष P (2a, 2, 6), Q (-4, 3b, -10) और R (8, 14, 2c) हैं, तो a, b और c के मान ज्ञात कीजिए।
त्रिभुज के केंद्रक के निर्देशांक का सूत्र: (x1 + x2 + x33, y1 + y2 + y33, z1 + z2 + z33)
प्रश्नानुसार, केंद्रक (0, 0, 0) है।
x-निर्देशांक की तुलना: 2a - 4 + 83 = 0 ⇒ 2a + 4 = 0 ⇒ 2a = -4 ⇒ a = -2
y-निर्देशांक की तुलना: 2 + 3b + 143 = 0 ⇒ 3b + 16 = 0 ⇒ 3b = -16 ⇒ b = -163
z-निर्देशांक की तुलना: 6 - 10 + 2c3 = 0 ⇒ 2c - 4 = 0 ⇒ 2c = 4 ⇒ c = 2
स्पष्टीकरण: केंद्रक (Centroid) निकालने के सूत्र को मूल बिंदु (0,0,0) के बराबर रख कर सीधे a, b और c का मान प्राप्त कर लिया गया।
4. y-अक्ष पर उस बिंदु के निर्देशांक ज्ञात कीजिए जिसकी बिंदु P (3, -2, 5) से दूरी 5√2 है।
चूँकि बिंदु y-अक्ष पर स्थित है, अतः इसके निर्देशांक (0, y, 0) होंगे। माना यह बिंदु A है।
दिया गया है कि AP = 5√2
दोनों पक्षों का वर्ग करने पर: AP2 = (5√2)2 = 25 × 2 = 50
दूरी सूत्र का प्रयोग करने पर: (3 - 0)2 + (-2 - y)2 + (5 - 0)2 = 50
9 + (y + 2)2 + 25 = 50
34 + y2 + 4y + 4 = 50
y2 + 4y + 38 = 50
y2 + 4y - 12 = 0
गुणनखंड करने पर: y2 + 6y - 2y - 12 = 0
y(y + 6) - 2(y + 6) = 0
(y + 6)(y - 2) = 0
अतः y = -6 या y = 2
अतः y-अक्ष पर ऐसे दो बिंदु संभव हैं: (0, 2, 0) और (0, -6, 0)
