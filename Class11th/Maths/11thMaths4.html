Question 1: Express (5i) -35 i in the form a + ib

Solution:

(5i) × -35 i = 5 × -35 × i × i
= -3 × i2
Since i2 = -1
= -3 × (-1) = 3
= 3 + i0

Answer: 3 + i0

Hinglish Explanation:

Pehle constants ko multiply karte hain: 5 × (-3/5) = -3. Phir i × i = i² = -1. Isliye -3 × (-1) = 3. Final answer 3 + 0i yaani 3 + i0.

Question 2: Express i9 + i19 in the form a + ib

Solution:

We know: i2 = -1, i4 = 1
i9 = i4×2+1 = (i4)2 × i = 12 × i = i
i19 = i4×4+3 = (i4)4 × i3 = 14 × i3 = i3 = i2 × i = -i
So, i9 + i19 = i + (-i) = 0
= 0 + i0

Answer: 0 + i0

Hinglish Explanation:

i9 nikalne ke liye: i4 = 1, toh i9 = i8 × i = (i4)2 × i = 1 × i = i. i19 = i16 × i3 = (i4)4 × i3 = 1 × (-i) = -i. Dono ko add karte hain: i + (-i) = 0.

Question 3: Express i-39 in the form a + ib

Solution:

i-39 = 1i39
i39 = i4×9+3 = (i4)9 × i3 = 19 × i3 = i3 = -i
So, i-39 = 1-i
Multiply numerator and denominator by i: 1 × i-i × i = i-i2
Since i2 = -1, i-(-1) = i1 = i
= 0 + i1

Answer: 0 + i1

Hinglish Explanation:

i-39 = 1/i39. i39 = i36 × i3 = (i4)9 × i3 = 1 × (-i) = -i. Toh i-39 = 1/(-i). Isko simplify karne ke liye numerator aur denominator ko i se multiply karte hain: i/(-i²) = i/1 = i.

Question 11: Find the multiplicative inverse of 4 - 3i

Solution:

Let z = 4 - 3i
Multiplicative inverse = 1z = 14 - 3i
Multiply numerator and denominator by conjugate: 14 - 3i × 4 + 3i4 + 3i
= 4 + 3i(4)2 - (3i)2 = 4 + 3i16 - 9i2
Since i2 = -1, = 4 + 3i16 - 9(-1) = 4 + 3i16 + 9 = 4 + 3i25
= 425 + 325 i

Answer: 425 + 325 i

Hinglish Explanation:

Multiplicative inverse nikalne ke liye 1/(4-3i) ko conjugate (4+3i) se multiply karte hain. Denominator mein (4-3i)(4+3i) = 16 - (3i)² = 16 - 9i² = 16 - 9(-1) = 25. Toh answer (4+3i)/25 = 4/25 + (3/25)i.

Question 12: Find the multiplicative inverse of √5 + 3i

Solution:

Let z = √5 + 3i
Multiplicative inverse = 1z = 1√5 + 3i
Multiply numerator and denominator by conjugate: 1√5 + 3i × √5 - 3i√5 - 3i
= √5 - 3i(√5)2 - (3i)2 = √5 - 3i5 - 9i2
Since i2 = -1, = √5 - 3i5 - 9(-1) = √5 - 3i5 + 9 = √5 - 3i14
= √514 - 314 i

Answer: √514 - 314 i

Hinglish Explanation:

Multiplicative inverse nikalne ke liye 1/(√5+3i) ko conjugate (√5-3i) se multiply karte hain. Denominator mein (√5+3i)(√5-3i) = (√5)² - (3i)² = 5 - 9i² = 5 - 9(-1) = 14. Toh answer (√5-3i)/14 = √5/14 - (3/14)i.

Question 14: Express (3 + i√5)(3 - i√5) in the form a + ib

Solution:

(3 + i√5)(3 - i√5) = (3)2 - (i√5)2
= 9 - i2(√5)2
Since i2 = -1, = 9 - (-1)(5)
= 9 + 5 = 14
= 14 + i0

Answer: 14 + i0

Hinglish Explanation:

Yeh (a+b)(a-b) = a² - b² form mein hai. a=3, b=i√5. Toh (3)² - (i√5)² = 9 - i²×5 = 9 - (-1)×5 = 9 + 5 = 14.

Question 1: Evaluate: [i18 + (1i)25]3

Solution:

First, find i18 = i4×4+2 = (i4)4 × i2 = 1 × (-1) = -1
Now, (1i)25 = i-25 = 1i25
i25 = i4×6+1 = (i4)6 × i = 1 × i = i
So, (1i)25 = 1i
Now, i18 + (1i)25 = -1 + 1i
Multiply 1i by ii: ii2 = i-1 = -i
So, -1 + (-i) = -1 - i
Now, [-1 - i]3 = (-1)3 (1 + i)3 = -1 × (1 + 3i + 3i2 + i3)
= -1 × (1 + 3i - 3 - i) = -1 × (-2 + 2i) = 2 - 2i

Answer: 2 - 2i

Hinglish Explanation:

Pehle i18 nikalte hain: i18 = (i4)4 × i2 = 1 × (-1) = -1. Phir (1/i)25 = i-25 = 1/i25. i25 = i, toh 1/i = -i. Toh -1 + (-i) = -1 - i. Ab (-1-i)3 = -1 × (1+i)3 = -1 × (1 + 3i + 3i² + i³) = -1 × (1 + 3i - 3 - i) = -1 × (-2 + 2i) = 2 - 2i.

Question 2: For any two complex numbers z1 and z2, prove that Re(z1z2) = Re(z1)Re(z2) - Im(z1)Im(z2)

Solution:

Let z1 = a + ib, z2 = c + id
Then z1z2 = (a + ib)(c + id) = ac + iad + ibc + i2bd
= ac + i(ad + bc) - bd (since i2 = -1)
= (ac - bd) + i(ad + bc)
So, Re(z1z2) = ac - bd
Now, Re(z1)Re(z2) - Im(z1)Im(z2) = a × c - b × d = ac - bd
Hence, Re(z1z2) = Re(z1)Re(z2) - Im(z1)Im(z2)

Answer: Proved

Hinglish Explanation:

z₁ = a+ib aur z₂ = c+id maan kar multiply karte hain: z₁z₂ = (a+ib)(c+id) = ac + iad + ibc + i²bd = (ac - bd) + i(ad+bc). Iska real part ac - bd hai. Right side: Re(z₁)Re(z₂) - Im(z₁)Im(z₂) = a×c - b×d = ac - bd. Dono equal hain, isliye proved.

Question 13: If (a + ib)(c + id)(e + if)(g + ih) = A + iB, then show that (a2 + b2)(c2 + d2)(e2 + f2)(g2 + h2) = A2 + B2

Solution:

Given: (a + ib)(c + id)(e + if)(g + ih) = A + iB
Taking modulus on both sides:
|(a + ib)(c + id)(e + if)(g + ih)| = |A + iB|
We know |z1z2z3z4| = |z1||z2||z3||z4|
So, |a + ib||c + id||e + if||g + ih| = |A + iB|
Now, |a + ib| = √(a2 + b2), so |a + ib|2 = a2 + b2
Similarly, |A + iB| = √(A2 + B2), so |A + iB|2 = A2 + B2
So, (a2 + b2)(c2 + d2)(e2 + f2)(g2 + h2) = A2 + B2

Answer: Proved

Hinglish Explanation:

Complex numbers ke product ka modulus, moduli ke product ke equal hota hai. Isliye |(a+ib)(c+id)(e+if)(g+ih)| = |a+ib||c+id||e+if||g+ih|. Left side |A+iB| hai, jiska square A²+B² hai. Right side ke moduli ke squares a²+b², c²+d², etc. hain. Inhe multiply karte hain toh A²+B² milta hai.

Question 14: If (1+i1-i)m = 1, then find the least positive integral value of m

Solution:

First, simplify 1+i1-i
Multiply numerator and denominator by (1+i): (1+i)(1+i)(1-i)(1+i) = 1 + 2i + i21 - i2
= 1 + 2i - 11 - (-1) = 2i2 = i
So, (1+i1-i)m = im = 1
We know i4 = 1, so m must be a multiple of 4
The least positive integral value of m is 4

Answer: 4

Hinglish Explanation:

Pehle (1+i)/(1-i) simplify karte hain. Numerator aur denominator ko (1+i) se multiply karte hain: (1+i)²/(1-i²) = (1+2i+i²)/(1-(-1)) = (1+2i-1)/2 = 2i/2 = i. Toh (i)m = 1. i4 = 1, isliye m ka smallest positive integer value 4 hai.
