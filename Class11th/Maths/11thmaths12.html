📘 प्रश्नावली 12.1 (सीमाएँ)

प्रश्न 1 से 22 तक निम्नलिखित सीमाओं के मान प्राप्त कीजिए:

1. limx → 3 (x + 3)
x का मान सीधे 3 रखने पर:
= 3 + 3 = 6
सीमा का सीधा प्रतिस्थापन (Direct substitution)।
2. limx → π (x - 227)
x का मान π रखने पर:
= π - 227
π एक अपरिमेय संख्या है, अतः इसे ऐसे ही छोड़ देते हैं।
3. limr → 1 πr2
r का मान 1 रखने पर:
= π(1)2 = π
4. limx → 4 4x + 3x - 2
x का मान 4 रखने पर:
= 4(4) + 34 - 2
= 16 + 32 = 192
5. limx → -1 x10 + x5 + 1x - 1
x का मान -1 रखने पर:
= (-1)10 + (-1)5 + 1-1 - 1
= 1 - 1 + 1-2 = -12
6. limx → 0 (x + 1)5 - 1x
माना y = x + 1. जब x → 0, तो y → 1. और x = y - 1
सीमा बनती है: limy → 1 y5 - 15y - 1
सूत्र limx → a xn - anx - a = n an-1 से:
= 5(1)5-1 = 5 × 1 = 5
मानक सूत्र का प्रयोग करने के लिए प्रतिस्थापन (Substitution) किया गया।
7. limx → 2 3x2 - x - 10x2 - 4
x = 2 रखने पर 0/0 रूप बनता है। अतः गुणनखंड (Factorize) करें।
अंश: 3x2 - 6x + 5x - 10 = 3x(x - 2) + 5(x - 2) = (3x + 5)(x - 2)
हर: x2 - 22 = (x + 2)(x - 2)
limx → 2 (3x + 5)(x - 2)(x + 2)(x - 2) = limx → 2 3x + 5x + 2
= 3(2) + 52 + 2 = 114
8. limx → 3 x4 - 812x2 - 5x - 3
0/0 रूप है। गुणनखंड करें।
अंश: x4 - 34 = (x2 - 9)(x2 + 9) = (x - 3)(x + 3)(x2 + 9)
हर: 2x2 - 6x + x - 3 = 2x(x - 3) + 1(x - 3) = (2x + 1)(x - 3)
limx → 3 (x - 3)(x + 3)(x2 + 9)(2x + 1)(x - 3) = limx → 3 (x + 3)(x2 + 9)2x + 1
= (3 + 3)(32 + 9)2(3) + 1 = 6 × 187 = 1087
9. limx → 0 ax + bcx + 1
x का मान 0 रखने पर:
= a(0) + bc(0) + 1 = b1 = b
10. limz → 1 z1/3 - 1z1/6 - 1
माना z1/6 = t. अतः z1/3 = (z1/6)2 = t2. जब z → 1, तो t → 1.
limt → 1 t2 - 1t - 1
= limt → 1 (t - 1)(t + 1)t - 1 = limt → 1 (t + 1)
= 1 + 1 = 2
घात (power) को सरल बनाने के लिए z1/6 को t मान लिया।
11. limx → 1 ax2 + bx + ccx2 + bx + a, a+b+c ≠ 0
x का मान 1 रखने पर:
= a(1)2 + b(1) + cc(1)2 + b(1) + a
= a + b + cc + b + a = 1
12. limx → -2 1x + 12x + 2
अंश को हल करने पर: 1x + 12 = 2 + x2x
limx → -2 x + 22x(x + 2)
(x + 2) को काटने पर: limx → -2 12x
= 12(-2) = -14
13. limx → 0 sin axbx
हम जानते हैं कि limy → 0 sin yy = 1
अंश और हर में a से गुणा करने पर:
= limx → 0 ab × sin axax
= ab × 1 = ab
14. limx → 0 sin axsin bx, a, b ≠ 0
अंश और हर दोनों को x से भाग देने पर:
limx → 0 sin axxsin bxx
अंश में a और हर में b से गुणा-भाग करने पर:
= limx → 0 a × sin axaxlimx → 0 b × sin bxbx
= a × 1b × 1 = ab
15. limx → π sin(π - x)π(π - x)
माना y = π - x. जब x → π, तो y → 0.
limy → 0 sin yπy
= 1π limy → 0 sin yy
= 1π × 1 = 1π
16. limx → 0 cos xπ - x
सीधे मान रखने पर 0/0 रूप नहीं बनता है।
= cos(0)π - 0 = 1π
17. limx → 0 cos 2x - 1cos x - 1
सूत्र: 1 - cos θ = 2sin2(θ/2). अतः cos θ - 1 = -2sin2(θ/2)
limx → 0 -2sin2(x)-2sin2(x/2) = limx → 0 sin2 xsin2(x/2)
अंश और हर को x2 से विभाजित करें:
= limx → 0 (sin xx)2sin2(x/2)x2
हर को व्यवस्थित करें: sin2(x/2)x2 = 14 × (sin(x/2)x/2)2
= 12(1/4) × 12 = 4
त्रिकोणमिति सर्वसमिका (Trigonometric Identity) लगाकर सीमा के मानक रूप (Standard Form) में बदला।
18. limx → 0 ax + x cos xb sin x
अंश से x कॉमन लेने पर: limx → 0 x(a + cos x)b sin x
= limx → 0 a + cos xb (sin xx)
चूँकि limx → 0 sin xx = 1 और cos(0) = 1
= a + 1b(1) = a + 1b
19. limx → 0 x sec x
x sec x = x × 1cos x
x = 0 रखने पर: 0 × 1cos(0) = 0 × 11 = 0
20. limx → 0 sin ax + bxax + sin bx a, b, a+b ≠ 0
अंश और हर दोनों को x से भाग देने पर:
limx → 0 sin axx + ba + sin bxx
= a(sin axax) + ba + b(sin bxbx)
सीमा लागू करने पर: = a(1) + ba + b(1) = a + ba + b = 1
21. limx → 0 (cosec x - cot x)
limx → 0 (1sin x - cos xsin x) = limx → 0 1 - cos xsin x
त्रिकोणमिति सूत्र से: 1 - cos x = 2sin2(x/2) और sin x = 2sin(x/2)cos(x/2)
= limx → 0 2sin2(x/2)2sin(x/2)cos(x/2)
= limx → 0 sin(x/2)cos(x/2) = limx → 0 tan(x/2)
= tan(0) = 0
22. limx → π/2 tan 2xx - π/2
माना x - π/2 = h. जब x → π/2, तो h → 0. और x = π/2 + h
limh → 0 tan 2(π/2 + h)h = limh → 0 tan(π + 2h)h
tan(π + θ) = tan θ होता है। अतः = limh → 0 tan 2hh
= limh → 0 2 × tan 2h2h = 2 × 1 = 2

प्रश्न 23 से 32 तक फलनों की सीमाएँ ज्ञात कीजिए:

23. limx → 0 f(x) और limx → 1 f(x) ज्ञात कीजिए, जहाँ f(x) = { 2x+3 (x≤0), 3(x+1) (x>0) }
x → 0 के लिए:
वाम पक्ष सीमा (LHL) = limx → 0- (2x + 3) = 2(0) + 3 = 3
दक्षिण पक्ष सीमा (RHL) = limx → 0+ 3(x + 1) = 3(0 + 1) = 3
चूँकि LHL = RHL, अतः limx → 0 f(x) = 3
x → 1 के लिए: x=1 > 0 है, अतः केवल नीचे वाला फलन प्रयोग होगा।
limx → 1 3(x + 1) = 3(1 + 1) = 6
खंडशः फलन (Piecewise function) के लिए उस बिंदु पर बाएँ (LHL) और दाएँ (RHL) सीमा की जाँच करनी होती है जहाँ फलन बदलता है।
24. limx → 1 f(x) ज्ञात कीजिए, जहाँ f(x) = { x2-1 (x≤1), -x2-1 (x>1) }
वाम पक्ष सीमा (LHL) = limx → 1- (x2 - 1) = (1)2 - 1 = 0
दक्षिण पक्ष सीमा (RHL) = limx → 1+ (-x2 - 1) = -(1)2 - 1 = -2
चूँकि LHL ≠ RHL (0 ≠ -2), अतः limx → 1 f(x) का अस्तित्व नहीं है (Does not exist)।
25. limx → 0 f(x) ज्ञात कीजिए, जहाँ f(x) = { |x|/x (x≠0), 0 (x=0) }
LHL: जब x → 0-, x ऋणात्मक है, तो |x| = -x. अतः f(x) = -x/x = -1. limx → 0- f(x) = -1
RHL: जब x → 0+, x धनात्मक है, तो |x| = x. अतः f(x) = x/x = 1. limx → 0+ f(x) = 1
चूँकि LHL ≠ RHL (-1 ≠ 1), अतः सीमा का अस्तित्व नहीं है।
26. limx → 0 f(x) ज्ञात कीजिए, जहाँ f(x) = { x/|x| (x≠0), 0 (x=0) }
प्रश्न 25 के समान: LHL = limx → 0- x/(-x) = -1
RHL = limx → 0+ x/x = 1
LHL ≠ RHL, अतः सीमा का अस्तित्व नहीं है।
27. limx → 5 f(x) ज्ञात कीजिए, जहाँ f(x) = |x| - 5
x → 5 पर x धनात्मक है, अतः |x| = x.
limx → 5 (x - 5) = 5 - 5 = 0
28. माना f(x) = { a+bx (x<1), 4 (x=1), b-ax (x>1) }. यदि limx → 1 f(x) = f(1), तो a और b के मान ज्ञात कीजिए।
दिया है f(1) = 4, और सीमा का अस्तित्व है तथा वह 4 के बराबर है।
LHL = limx → 1- (a + bx) = a + b(1) = a + b
RHL = limx → 1+ (b - ax) = b - a(1) = b - a
चूँकि LHL = RHL = f(1), तो: a + b = 4 और b - a = 4
दोनों समीकरणों को जोड़ने पर: 2b = 8 → b = 4
b का मान रखने पर: a + 4 = 4 → a = 0
अतः a = 0 और b = 4.
29. मान लीजिए a1, a2, ... an अचर वास्तविक संख्याएँ हैं। f(x) = (x-a1)(x-a2)...(x-an). limx → a1 f(x) क्या है? किसी a ≠ a1...an के लिए limx → a f(x) क्या है?
limx → a1 f(x) = (a1 - a1)(a1 - a2)...(a1 - an) = 0 × (अन्य पद) = 0
किसी a के लिए (जो इनमें से कोई नहीं है): limx → a f(x) = (a - a1)(a - a2)...(a - an)
यह एक बहुपद (Polynomial) है, अतः सीमा निकालने के लिए सीधे x का मान रख दिया गया है।
30. f(x) = { |x|+1 (x<0), 0 (x=0), |x|-1 (x>0) }. a के किन मानों के लिए limx → a f(x) का अस्तित्व है?
a = 0 पर जाँच: LHL = limx → 0- (|x| + 1) = 0 + 1 = 1. RHL = limx → 0+ (|x| - 1) = 0 - 1 = -1.
a = 0 पर LHL ≠ RHL, अतः a = 0 पर सीमा का अस्तित्व नहीं है।
a < 0 के लिए: f(x) = -x + 1 (बहुपद), सीमा का अस्तित्व है।
a > 0 के लिए: f(x) = x - 1 (बहुपद), सीमा का अस्तित्व है।
अतः a के सभी वास्तविक मानों के लिए (a ≠ 0 को छोड़कर) सीमा का अस्तित्व है।
31. यदि फलन f(x) सीमा limx → 1 f(x) - 2x2 - 1 = π को संतुष्ट करता है, तो limx → 1 f(x) ज्ञात कीजिए।
सीमा के गुणधर्म से: limx → 1 (f(x) - 2) = limx → 1 π × (x2 - 1)
limx → 1 (f(x) - 2) = π × (12 - 1) = π × 0 = 0
अतः limx → 1 f(x) - 2 = 0 → limx → 1 f(x) = 2
32. यदि f(x) = { mx2+n (x<0), nx+m (0≤x≤1), nx3+m (x>1) }. m और n के किन पूर्णांक मानों के लिए limx → 0 f(x) और limx → 1 f(x) दोनों का अस्तित्व है?
x = 0 पर: LHL = limx → 0- (mx2 + n) = n. RHL = limx → 0+ (nx + m) = m.
सीमा के अस्तित्व के लिए: LHL = RHL → n = m
x = 1 पर: LHL = limx → 1- (nx + m) = n + m. RHL = limx → 1+ (nx3 + m) = n(1) + m = n + m.
यहाँ हमेशा LHL = RHL (n+m = n+m) रहता है।
अतः दोनों सीमाओं के अस्तित्व के लिए केवल एक शर्त है: m और n बराबर होने चाहिए (m = n)।
📗 प्रश्नावली 12.2 (अवकलज)
1. x = 10 पर x2 - 2 का अवकलज ज्ञात कीजिए।
f(x) = x2 - 2
सूत्र ddx(xn) = nxn-1 से: f'(x) = 2x - 0 = 2x
x = 10 पर: f'(10) = 2(10) = 20
2. x = 100 पर 99x का अवकलज ज्ञात कीजिए।
f(x) = 99x
f'(x) = 99 × 1 = 99
चूँकि अवकलज अचर (constant) है, x = 100 पर भी f'(100) = 99 होगा।
3. x = 1 पर x का अवकलज ज्ञात कीजिए।
f(x) = x
f'(x) = 1 × x1-1 = x0 = 1
x = 1 पर: f'(1) = 1
4. प्रथम सिद्धांत (First Principles) से निम्नलिखित फलनों के अवकलज ज्ञात कीजिए:
(i) x3 - 27
f'(x) = limh → 0 f(x+h) - f(x)h
= limh → 0 ((x+h)3 - 27) - (x3 - 27)h = limh → 0 x3 + 3x2h + 3xh2 + h3 - x3h
= limh → 0 h(3x2 + 3xh + h2)h = limh → 0 (3x2 + 3xh + h2) = 3x2
(ii) (x - 1)(x - 2)
f(x) = x2 - 3x + 2
f'(x) = limh → 0 ((x+h)2 - 3(x+h) + 2) - (x2 - 3x + 2)h
= limh → 0 x2 + 2xh + h2 - 3x - 3h + 2 - x2 + 3x - 2h
= limh → 0 2xh + h2 - 3hh = limh → 0 (2x + h - 3) = 2x - 3
(iii) 1x2
f'(x) = limh → 0 1(x+h)2 - 1x2h = limh → 0 x2 - (x+h)2h x2 (x+h)2
= limh → 0 x2 - (x2 + 2xh + h2)h x2 (x+h)2 = limh → 0 -h(2x + h)h x2 (x+h)2
= limh → 0 -(2x + h)x2 (x+h)2 = -2xx4 = -2x3
(iv) x + 1x - 1
f'(x) = limh → 0 x+h+1x+h-1 - x+1x-1h
LCM लेकर हल करने पर: = limh → 0 (x+h+1)(x-1) - (x+1)(x+h-1)h (x+h-1)(x-1)
= limh → 0 (x2 - x + hx - h + x - 1) - (x2 + hx - x + x + h - 1)h (x+h-1)(x-1)
= limh → 0 -2hh (x+h-1)(x-1) = -2(x-1)2
प्रथम सिद्धांत (First Principles) में h → 0 सीमा लगाकर अवकलज निकाला जाता है।
5. फलन f(x) = x100100 + x9999 + ... + x22 + x + 1 के लिए सिद्ध कीजिए कि f'(1) = 100 f'(0).
अवकलन करने पर: f'(x) = 100x99100 + 99x9899 + ... + 2x2 + 1 + 0
f'(x) = x99 + x98 + ... + x + 1
f'(1) = 199 + 198 + ... + 1 + 1 (यह 1 कुल 100 बार है) = 100
f'(0) = 099 + 098 + ... + 0 + 1 = 1
अतः 100 × f'(0) = 100 × 1 = 100 = f'(1). सिद्ध हुआ।
6. किसी अचर वास्तविक संख्या a के लिए xn + axn-1 + a2xn-2 + ... + an-1x + an का अवकलज ज्ञात कीजिए।
f(x) का x के सापेक्ष अवकलन:
f'(x) = ddx(xn) + a ddx(xn-1) + a2 ddx(xn-2) + ... + an-1 ddx(x) + ddx(an)
f'(x) = nxn-1 + a(n-1)xn-2 + a2(n-2)xn-3 + ... + an-1(1) + 0
= nxn-1 + a(n-1)xn-2 + a2(n-2)xn-3 + ... + an-1
7. किन्हीं अचरों a और b के लिए, निम्नलिखित के अवकलज ज्ञात कीजिए:
(i) (x - a)(x - b)
f(x) = x2 - (a + b)x + ab
f'(x) = 2x - (a + b) + 0 = 2x - a - b
(ii) (ax2 + b)2
f(x) = a2x4 + 2abx2 + b2
f'(x) = 4a2x3 + 4abx + 0 = 4ax(ax2 + b)
(iii) x - ax - b
भागफल नियम (Quotient Rule) से: f'(x) = (x-b)·1 - (x-a)·1(x-b)2
= x - b - x + a(x-b)2 = a - b(x - b)2
8. किसी अचर a के लिए xn - anx - a का अवकलज ज्ञात कीजिए।
भागफल नियम से: f'(x) = (x-a)·ddx(xn-an) - (xn-an)·ddx(x-a)(x-a)2
= (x-a)(nxn-1) - (xn-an)(1)(x-a)2
= nxn - anxn-1 - xn + an(x-a)2
9. निम्नलिखित के अवकलज ज्ञात कीजिए:
(i) 2x - 34
f'(x) = 2(1) - 0 = 2
(ii) (5x3 + 3x - 1)(x - 1)
गुणनफल नियम (Product Rule): f'(x) = (5x3 + 3x - 1)·1 + (x - 1)·(15x2 + 3)
= 5x3 + 3x - 1 + 15x3 + 3x - 15x2 - 3 = 20x3 - 15x2 + 6x - 4
(iii) x-3(5 + 3x)
f(x) = 5x-3 + 3x-2
f'(x) = 5(-3)x-4 + 3(-2)x-3 = -15x-4 - 6x-3
(iv) x5(3 - 6x-9)
f(x) = 3x5 - 6x-4
f'(x) = 15x4 - 6(-4)x-5 = 15x4 + 24x-5
(v) x-4(3 - 4x-5)
f(x) = 3x-4 - 4x-9
f'(x) = -12x-5 - 4(-9)x-10 = -12x-5 + 36x-10
(vi) 2x + 1 - x23x - 1
f'(x) = -2(x + 1)2 - (3x-1)(2x) - x2(3)(3x - 1)2
= -2(x + 1)2 - 6x2 - 2x - 3x2(3x - 1)2 = -2(x + 1)2 - 3x2 - 2x(3x - 1)2
कोष्ठकों को खोलकर बहुपदों में बदलने से (ii) से (v) तक के अवकलन सरल हो जाते हैं।
10. प्रथम सिद्धांत (First Principle) से cos x का अवकलज ज्ञात कीजिए।
f'(x) = limh → 0 cos(x + h) - cos xh
सूत्र cos C - cos D = -2 sinC+D2 sinC-D2 से:
= limh → 0 -2 sin(x + h/2) sin(h/2)h
= limh → 0 -sin(x + h/2) × sin(h/2)h/2
चूँकि limθ → 0 sin θθ = 1, h = 0 रखने पर:
= -sin(x + 0) × 1 = -sin x
11. निम्नलिखित फलनों के अवकलज ज्ञात कीजिए:
(i) sin x cos x
गुणनफल नियम से: sin x(-sin x) + cos x(cos x) = cos2 x - sin2 x = cos 2x
(ii) sec x
sec x = (cos x)-1 → -1(cos x)-2(-sin x) = sin xcos2 x = sec x tan x
(iii) 5 sec x + 4 cos x
f'(x) = 5(sec x tan x) + 4(-sin x) = 5 sec x tan x - 4 sin x
(iv) cosec x
cosec x = (sin x)-1 → -1(sin x)-2(cos x) = -cos xsin2 x = -cosec x cot x
(v) 3 cot x + 5 cosec x
f'(x) = 3(-cosec2 x) + 5(-cosec x cot x) = -3 cosec2 x - 5 cosec x cot x
(vi) 5 sin x - 6 cos x + 7
f'(x) = 5(cos x) - 6(-sin x) + 0 = 5 cos x + 6 sin x
(vii) 2 tan x - 7 sec x
f'(x) = 2(sec2 x) - 7(sec x tan x) = 2 sec2 x - 7 sec x tan x
यहाँ सीधे मानक त्रिकोणमिति अवकलज (Standard Trigonometric Derivatives) के सूत्रों का उपयोग किया गया है।
