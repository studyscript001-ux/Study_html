📘 प्रश्नावली 7.1
1. (1 - 2x)⁵ का प्रसार कीजिए।
द्विपद प्रमेय: (a + b)ⁿ = ⁿC₀ aⁿ + ⁿC₁ aⁿ⁻¹b + ... + ⁿCₙ bⁿ
यहाँ a = 1, b = -2x और n = 5 है।
(1 - 2x)⁵ = ⁵C₀(1)⁵ + ⁵C₁(1)⁴(-2x) + ⁵C₂(1)³(-2x)² + ⁵C₃(1)²(-2x)³ + ⁵C₄(1)(-2x)⁴ + ⁵C₅(-2x)⁵
= 1(1) + 5(-2x) + 10(4x²) + 10(-8x³) + 5(16x⁴) + 1(-32x⁵)
= 1 - 10x + 40x² - 80x³ + 80x⁴ - 32x⁵
स्पष्टीकरण: द्विपद प्रमेय (Binomial Theorem) के सूत्र का सीधा उपयोग किया गया है। ⁵C₀=1, ⁵C₁=5, ⁵C₂=10 का प्रयोग किया गया है।
2. (2x - x2)⁵ का प्रसार कीजिए।
यहाँ a = 2x, b = -x2 और n = 5 है।
= ⁵C₀(2x)⁵ + ⁵C₁(2x)⁴(-x2) + ⁵C₂(2x)³(-x2)² + ⁵C₃(2x)²(-x2)³ + ⁵C₄(2x)(-x2)⁴ + ⁵C₅(-x2)⁵
= 1(32x⁵) - 5(16x⁴)(x2) + 10(8x³)(x²4) - 10(4x²)(x³8) + 5(2x)(x⁴16) - 1(x⁵32)
= 32x⁵ - 40x³ + 20x - 5x + 58x³ - x⁵32
स्पष्टीकरण: x की घातों (powers) को सरल करते समय अंश (numerator) और हर (denominator) को ध्यान से काटा गया है।
3. (2x - 3)⁶ का प्रसार कीजिए।
यहाँ a = 2x, b = -3 और n = 6 है।
= ⁶C₀(2x)⁶ + ⁶C₁(2x)⁵(-3) + ⁶C₂(2x)⁴(-3)² + ⁶C₃(2x)³(-3)³ + ⁶C₄(2x)²(-3)⁴ + ⁶C₅(2x)(-3)⁵ + ⁶C₆(-3)⁶
= 1(64x⁶) - 6(32x⁵)(3) + 15(16x⁴)(9) - 20(8x³)(27) + 15(4x²)(81) - 6(2x)(243) + 1(729)
= 64x⁶ - 576x⁵ + 2160x⁴ - 4320x³ + 4860x² - 2916x + 729
स्पष्टीकरण: गुणांक ⁶C₀=1, ⁶C₁=6, ⁶C₂=15, ⁶C₃=20, ⁶C₄=15, ⁶C₅=6, ⁶C₆=1 का उपयोग किया गया है।
4. (x3 + 1x)⁵ का प्रसार कीजिए।
यहाँ a = x3, b = 1x और n = 5 है।
= ⁵C₀(x3)⁵ + ⁵C₁(x3)⁴(1x) + ⁵C₂(x3)³(1x)² + ⁵C₃(x3)²(1x)³ + ⁵C₄(x3)(1x)⁴ + ⁵C₅(1x)⁵
= 1(x⁵243) + 5(x⁴81)(1x) + 10(x³27)(1x²) + 10(x²9)(1x³) + 5(x3)(1x⁴) + 1(1x⁵)
= x⁵243 + 5x³81 + 10x27 + 109x + 53x³ + 1x⁵
स्पष्टीकरण: पदों को काट कर (जैसे x⁴ / x = x³) सरलतम रूप में लिखा गया है।
5. (x + 1x)⁶ का प्रसार कीजिए।
यहाँ a = x, b = 1x और n = 6 है।
= ⁶C₀(x)⁶ + ⁶C₁(x)⁵(1x) + ⁶C₂(x)⁴(1x)² + ⁶C₃(x)³(1x)³ + ⁶C₄(x)²(1x)⁴ + ⁶C₅(x)(1x)⁵ + ⁶C₆(1x)⁶
= x⁶ + 6x⁵(1x) + 15x⁴(1x²) + 20x³(1x³) + 15x²(1x⁴) + 6x(1x⁵) + 1x⁶
= x⁶ + 6x⁴ + 15x² + 20 + 15x² + 6x⁴ + 1x⁶
स्पष्टीकरण: मध्य पद में x³ और 1/x³ कट जाते हैं, इसलिए यह 20 बचता है।
6. (96)³ का मान ज्ञात कीजिए।
96 को (100 - 4) के रूप में लिख सकते हैं।
(100 - 4)³ = ³C₀(100)³ - ³C₁(100)²(4) + ³C₂(100)(4)² - ³C₃(4)³
= 1(1000000) - 3(10000)(4) + 3(100)(16) - 1(64)
= 1000000 - 120000 + 4800 - 64
= 884736
स्पष्टीकरण: बड़ी संख्या को (100 - 4) में तोड़कर (a - b)³ का सूत्र लगाया गया है जिससे गणना आसान हो जाती है।
7. (102)⁵ का मान ज्ञात कीजिए।
102 को (100 + 2) के रूप में लिख सकते हैं।
(100 + 2)⁵ = ⁵C₀(100)⁵ + ⁵C₁(100)⁴(2) + ⁵C₂(100)³(2)² + ⁵C₃(100)²(2)³ + ⁵C₄(100)(2)⁴ + ⁵C₅(2)⁵
= 10000000000 + 5(100000000)(2) + 10(1000000)(4) + 10(10000)(8) + 5(100)(16) + 32
= 10000000000 + 1000000000 + 40000000 + 800000 + 8000 + 32
= 11040808032
स्पष्टीकरण: (100 + 2) में तोड़ने से 100 की घातों को हल करना बहुत सरल हो जाता है। सभी पदों को जोड़ दिया गया है।
8. (101)⁴ का मान ज्ञात कीजिए।
101 को (100 + 1) के रूप में लिख सकते हैं।
(100 + 1)⁴ = ⁴C₀(100)⁴ + ⁴C₁(100)³(1) + ⁴C₂(100)²(1)² + ⁴C₃(100)(1)³ + ⁴C₄(1)⁴
= 100000000 + 4(1000000) + 6(10000) + 4(100) + 1
= 100000000 + 4000000 + 60000 + 400 + 1
= 104060401
स्पष्टीकरण: 100 की घातों के अनुसार पदों को व्यवस्थित कर योगफल निकाला गया है।
9. (99)⁵ का मान ज्ञात कीजिए।
99 को (100 - 1) के रूप में लिख सकते हैं।
(100 - 1)⁵ = ⁵C₀(100)⁵ - ⁵C₁(100)⁴(1) + ⁵C₂(100)³(1)² - ⁵C₃(100)²(1)³ + ⁵C₄(100)(1)⁴ - ⁵C₅(1)⁵
= 10000000000 - 5(100000000) + 10(1000000) - 10(10000) + 5(100) - 1
= 10000000000 - 500000000 + 10000000 - 100000 + 500 - 1
= (10000000000 + 10000000 + 500) - (500000000 + 100000 + 1)
= 10010000500 - 500100001 = 9509900499
स्पष्टीकरण: धनात्मक और ऋणात्मक पदों को अलग-अलग जोड़कर अंत में घटाया गया है।
10. द्विपद प्रमेय का प्रयोग करते हुए बताइए कौन-सी संख्या बड़ी है (1.1)10000 या 1000।
(1.1)10000 = (1 + 0.1)10000
द्विपद प्रमेय से: (1 + 0.1)10000 = 10000C₀(1)10000 + 10000C₁(1)9999(0.1) + अन्य धनात्मक पद
= 1 + 10000(0.1) + अन्य धनात्मक पद
= 1 + 1000 + अन्य धनात्मक पद = 1001 + अन्य धनात्मक पद
स्पष्ट रूप से, (1001 + कुछ धनात्मक संख्या) हमेशा 1000 से बड़ी होगी।
अतः (1.1)10000 > 1000
स्पष्टीकरण: केवल पहले दो पदों को हल करने पर ही हमें 1001 मिल जाता है, जो 1000 से बड़ा है। इसलिए आगे के पदों को हल करने की आवश्यकता नहीं है।
11. (a+b)⁴ - (a-b)⁴ का विस्तार कीजिए। इसका प्रयोग करके (√3+√2)⁴ - (√3-√2)⁴ का मान ज्ञात कीजिए।
(a+b)⁴ = a⁴ + 4a³b + 6a²b² + 4ab³ + b⁴
(a-b)⁴ = a⁴ - 4a³b + 6a²b² - 4ab³ + b⁴
घटाने पर: (a+b)⁴ - (a-b)⁴ = 2(4a³b + 4ab³) = 8ab(a² + b²)
अब, a = √3 और b = √2 रखने पर:
= 8(√3)(√2) [ (√3)² + (√2)² ]
= 8(√6) [ 3 + 2 ]
= 8√6 × 5 = 40√6
स्पष्टीकरण: दोनों विस्तारों को घटाने पर a⁴, 6a²b² और b⁴ कट जाते हैं। फिर सूत्र में √3 और √2 रख कर सीधा मान प्राप्त हो जाता है।
12. (x+1)⁶ + (x-1)⁶ का मान ज्ञात कीजिए। इसका प्रयोग करके (√2+1)⁶ + (√2-1)⁶ का मान ज्ञात कीजिए।
(x+1)⁶ = x⁶ + ⁶C₁x⁵ + ⁶C₂x⁴ + ⁶C₃x³ + ⁶C₄x² + ⁶C₅x + 1
(x-1)⁶ = x⁶ - ⁶C₁x⁵ + ⁶C₂x⁴ - ⁶C₃x³ + ⁶C₄x² - ⁶C₅x + 1
जोड़ने पर: (x+1)⁶ + (x-1)⁶ = 2(x⁶ + ⁶C₂x⁴ + ⁶C₄x² + 1)
= 2(x⁶ + 15x⁴ + 15x² + 1)
अब, x = √2 रखने पर:
= 2[ (√2)⁶ + 15(√2)⁴ + 15(√2)² + 1 ]
= 2[ 8 + 15(4) + 15(2) + 1 ]
= 2[ 8 + 60 + 30 + 1 ] = 2[ 99 ] = 198
स्पष्टीकरण: योग करने पर सभी विषम घात वाले पद (जैसे x⁵, x³) कट जाते हैं। x=√2 रखने से (√2)⁶ = 8, (√2)⁴ = 4 होता है।
13. दिखाइए कि 9ⁿ⁺¹ - 8n - 9, 64 से विभाज्य है जहाँ n एक धन पूर्णांक है।
9ⁿ⁺¹ को हम (1 + 8)ⁿ⁺¹ लिख सकते हैं।
द्विपद प्रमेय से: (1 + 8)ⁿ⁺¹ = ⁿ⁺¹C₀ + ⁿ⁺¹C₁(8) + ⁿ⁺¹C₂(8)² + ... + ⁿ⁺¹Cₙ₊₁(8)ⁿ⁺¹
= 1 + (n + 1)8 + ⁿ⁺¹C₂(64) + ... + 8ⁿ⁺¹
= 1 + 8n + 8 + 64[ ⁿ⁺¹C₂ + ... + 8ⁿ⁻¹ ]
= 9 + 8n + 64k (जहाँ k = ⁿ⁺¹C₂ + ... + 8ⁿ⁻¹)
अब, व्यंजक 9ⁿ⁺¹ - 8n - 9 में मान रखने पर:
(9 + 8n + 64k) - 8n - 9 = 64k
चूंकि 64k, 64 से पूर्णतः विभाज्य है, अतः सिद्ध हुआ।
स्पष्टीकरण: 9 को 1+8 लिखकर द्विपद प्रसार किया गया। शुरुआती पद 1+8n+8 बनते हैं, जो दिए गए -8n-9 से कट जाते हैं। बचे हुए सभी पदों में 64 कॉमन है।
14. सिद्ध कीजिए कि ∑(r=0 से n) 3ʳ·ⁿCᵣ = 4ⁿ
द्विपद प्रमेय के अनुसार: (1 + x)ⁿ = ⁿC₀ + ⁿC₁x + ⁿC₂x² + ... + ⁿCₙxⁿ
इसे सिग्मा (∑) रूप में इस प्रकार लिख सकते हैं: (1 + x)ⁿ = ∑(r=0 से n) ⁿCᵣ xʳ
अब x = 3 रखने पर:
(1 + 3)ⁿ = ∑(r=0 से n) ⁿCᵣ (3)ʳ
4ⁿ = ∑(r=0 से n) 3ʳ·ⁿCᵣ
अतः सिद्ध हुआ: बायाँ पक्ष (LHS) = दायाँ पक्ष (RHS)
स्पष्टीकरण: यह द्विपद प्रसार के मानक रूप (1+x)ⁿ का एक सीधा अनुप्रयोग है, जहाँ x का मान 3 रखा गया है।
🔀 विविध प्रश्नावली
1. यदि a और b भिन्न-भिन्न पूर्णांक हों, तो सिद्ध कीजिए कि (aⁿ - bⁿ) का एक गुणनखंड (a-b) है, जबकि n एक धन पूर्णांक है।
हम aⁿ को इस प्रकार लिख सकते हैं: aⁿ = ((a - b) + b)ⁿ
द्विपद प्रमेय से विस्तार करने पर:
((a - b) + b)ⁿ = ⁿC₀(a - b)ⁿ + ⁿC₁(a - b)ⁿ⁻¹b + ... + ⁿCₙ₋₁(a - b)bⁿ⁻¹ + ⁿCₙbⁿ
यहाँ ⁿCₙbⁿ = bⁿ होता है।
aⁿ = (a - b) [ ⁿC₀(a - b)ⁿ⁻¹ + ⁿC₁(a - b)ⁿ⁻²b + ... + ⁿCₙ₋₁bⁿ⁻¹ ] + bⁿ
aⁿ - bⁿ = (a - b) × (कोई पूर्णांक k)
इससे सिद्ध होता है कि (a - b), (aⁿ - bⁿ) का एक गुणनखंड है।
स्पष्टीकरण: a को चालाकी से (a-b+b) लिखकर द्विपद प्रसार किया गया। अंतिम पद bⁿ को बाईं ओर ले जाने पर, बाकी सभी पदों में (a-b) कॉमन आ जाता है।
2. (√3 + √2)⁶ - (√3 - √2)⁶ का मान ज्ञात कीजिए।
माना x = √3 और y = √2
(x + y)⁶ - (x - y)⁶ = 2 [ ⁶C₁x⁵y + ⁶C₃x³y³ + ⁶C₅xy⁵ ]
= 2 [ 6x⁵y + 20x³y³ + 6xy⁵ ]
= 4xy [ 3x⁴ + 10x²y² + 3y⁴ ] (2xy कॉमन लेने पर)
x = √3 और y = √2 रखने पर:
x² = 3, x⁴ = 9 और y² = 2, y⁴ = 4
= 4(√3)(√2) [ 3(9) + 10(3)(2) + 3(4) ]
= 4√6 [ 27 + 60 + 12 ]
= 4√6 [ 99 ] = 396√6
स्पष्टीकरण: दोनों विस्तारों को घटाने पर केवल सम संख्या वाले पद (जैसे y, y³, y⁵) बचते हैं। मान रखने पर सरलता से उत्तर आ जाता है।
3. (a² + √(a²-1))⁴ + (a² - √(a²-1))⁴ का मान ज्ञात कीजिए।
माना x = a² और y = √(a²-1) ⇒ y² = a²-1
(x + y)⁴ + (x - y)⁴ = 2 [ ⁴C₀x⁴ + ⁴C₂x²y² + ⁴C₄y⁴ ]
= 2 [ x⁴ + 6x²y² + y⁴ ]
x और y² का मान रखने पर:
= 2 [ (a²)⁴ + 6(a²)²(a²-1) + (a²-1)² ]
= 2 [ a⁸ + 6a⁴(a²-1) + (a⁴ - 2a² + 1) ]
= 2 [ a⁸ + 6a⁶ - 6a⁴ + a⁴ - 2a² + 1 ]
= 2 [ a⁸ + 6a⁶ - 5a⁴ - 2a² + 1 ]
= 2a⁸ + 12a⁶ - 10a⁴ - 4a² + 2
स्पष्टीकरण: प्लस करने पर सम पद (x⁴, x²y², y⁴) जुड़ जाते हैं। y² का मान (a²-1) सीधे रखने से रूट (√) हट जाता है।
4. (0.99)⁵ के प्रसार के पहले तीन पदों का प्रयोग करते हुए इसका निकटतम मान ज्ञात कीजिए।
0.99 को (1 - 0.01) के रूप में लिख सकते हैं।
(1 - 0.01)⁵ = ⁵C₀(1)⁵ - ⁵C₁(1)⁴(0.01) + ⁵C₂(1)³(0.01)² - ...
केवल पहले तीन पद लेने पर:
≈ 1 - 5(0.01) + 10(0.0001)
≈ 1 - 0.05 + 0.001
≈ 0.95 + 0.001 = 0.951
स्पष्टीकरण: छोटे मान (0.01) की उच्च घातें बहुत छोटी हो जाती हैं, इसलिए केवल 3 पदों से ही हमें लगभग सटीक (निकटतम) मान मिल जाता है।
5. (1 + x2 - 2x)⁴, x ≠ 0 का द्विपद प्रमेय द्वारा प्रसार ज्ञात कीजिए।
माना y = (x2 - 2x), तो व्यंजक (1 + y)⁴ बन जाएगा।
(1 + y)⁴ = 1 + 4y + 6y² + 4y³ + y⁴
अब y की घातों का मान निकालते हैं:
y = x2 - 2x
y² = (x2 - 2x)² = x²4 - 2 + 4x²
y³ = y² × y = (x²4 - 2 + 4x²)(x2 - 2x) = x³8 - x2 - x + 4x + 2x - 8x³ = x³8 - 3x2 + 6x - 8x³
y⁴ = (y²)² = (x²4 - 2 + 4x²)² = x⁴16 + 4 + 16x⁴ - x² - 16x² + 2 = x⁴16 - x² + 6 - 16x² + 16x⁴
सभी मान (1 + y)⁴ में रखने पर:
= 1 + 4(x2 - 2x) + 6(x²4 - 2 + 4x²) + 4(x³8 - 3x2 + 6x - 8x³) + (x⁴16 - x² + 6 - 16x² + 16x⁴)
= 1 + 2x - 8x + 3x²2 - 12 + 24x² + x³2 - 6x + 24x - 32x³ + x⁴16 - x² + 6 - 16x² + 16x⁴
समान घातों वाले पदों को एक साथ जोड़ने पर:
= 16x⁴ - 32x³ + 8x² + 16x - 5 - 4x + x²2 + x³2 + x⁴16
स्पष्टीकरण: त्रिपद (3 terms) को द्विपद (1+y) में बदलकर हल किया गया। सभी घातों का अलग-अलग विस्तार करके अंत में समान घात वाले पद जोड़ दिए गए।
6. (3x² - 2ax + 3a²)³ का द्विपद प्रमेय से प्रसार ज्ञात कीजिए।
माना A = 3x² + 3a² और B = -2ax
व्यंजक (A + B)³ = A³ + 3A²B + 3AB² + B³ बन जाता है।
A³ = [3(x² + a²)]³ = 27(x⁶ + 3x⁴a² + 3x²a⁴ + a⁶) = 27x⁶ + 81a²x⁴ + 81a⁴x² + 27a⁶
3A²B = 3[3(x² + a²)]²(-2ax) = -54ax(x⁴ + 2a²x² + a⁴) = -54ax⁵ - 108a³x³ - 54a⁵x
3AB² = 3(3x² + 3a²)(-2ax)² = 9(x² + a²)(4a²x²) = 36a²x⁴ + 36a⁴x²
B³ = (-2ax)³ = -8a³x³
अब सभी मानों को एक साथ रखने पर:
= (27x⁶ + 81a²x⁴ + 81a⁴x² + 27a⁶) + (-54ax⁵ - 108a³x³ - 54a⁵x) + (36a²x⁴ + 36a⁴x²) - 8a³x³
