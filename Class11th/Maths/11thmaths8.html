a = ARp-1, b = ARq-1, c = ARr-1
aq-r = Aq-r R(p-1)(q-r)
br-p = Ar-p R(q-1)(r-p)
cp-q = Ap-q R(r-1)(p-q)
तीनों का गुणा करने पर: A की घातों का योग = (q-r) + (r-p) + (p-q) = 0
R की घातों का योग = (p-1)(q-r) + (q-1)(r-p) + (r-1)(p-q) = 0
= A0 R0 = 1. सिद्ध हुआ।
घातांकों के नियम से घातें जुड़ जाती हैं और चक्रीय क्रम में होने के कारण योग शून्य हो जाता है।
23. यदि किसी गुणोत्तर श्रेणी का प्रथम तथा nवाँ पद क्रमशः a तथा b हैं, एवं P, n पदों का गुणनफल हो, तो सिद्ध कीजिए कि P2 = (ab)n
दिया है: a1 = a, an = b ⇒ b = arn-1
गुणनफल P = a × ar × ar2 × ... × arn-1
P = an r(1+2+...+(n-1)) = an rn(n-1)/2
P2 = (an rn(n-1)/2)2 = a2n rn(n-1)
(ab)n = (a × arn-1)n = (a2 rn-1)n = a2n rn(n-1)
अतः P2 = (ab)n. सिद्ध हुआ।
r की घातों का योग समांतर श्रेणी (A.P.) के योगफल n(n-1)/2 के सूत्र से निकाला गया।
24. दिखाइए कि एक गुणोत्तर श्रेणी के प्रथम n पदों के योगफल तथा (n+1)वें पद से (2n)वें पद तक के पदों के योगफल का अनुपात 1 / rn है।
प्रथम n पदों का योग S1 = a(rn - 1)r - 1
अगले n पदों का योग S2 (जहाँ प्रथम पद an+1 = arn है) = arn(rn - 1)r - 1
अनुपात = S1S2 = [a(rn - 1) / (r - 1)][arn(rn - 1) / (r - 1)]
सभी समान पद कटने पर = 1rn. सिद्ध हुआ।
अगले n पदों के लिए नई गुणोत्तर श्रेणी का पहला पद arn मानकर हल किया।
25. यदि a, b, c तथा d गुणोत्तर श्रेणी में हैं तो दिखाइए कि (a2+b2+c2)(b2+c2+d2) = (ab+bc+cd)2
चूँकि पद G.P. में हैं: b = ar, c = ar2, d = ar3
LHS: [a2 + a2r2 + a2r4] × [a2r2 + a2r4 + a2r6]
= a2(1+r2+r4) × a2r2(1+r2+r4) = a4r2(1+r2+r4)2
RHS: (a(ar) + (ar)(ar2) + (ar2)(ar3))2
= (a2r + a2r3 + a2r5)2 = [a2r(1+r2+r4)]2 = a4r2(1+r2+r4)2
LHS = RHS. सिद्ध हुआ।
सबको a और r के रूप में बदलकर बीजगणितीय रूप से दोनों पक्षों को समान दिखाया गया है।
26. ऐसी दो संख्याएँ ज्ञात कीजिए जिनको 3 तथा 81 के बीच रखने पर प्राप्त अनुक्रम एक गुणोत्तर श्रेणी बन जाय।
माना संख्याएँ G1 और G2 हैं। अनुक्रम 3, G1, G2, 81 G.P. में है।
यहाँ a = 3, चौथा पद a4 = 81
ar3 = 81 ⇒ 3r3 = 81 ⇒ r3 = 27 ⇒ r = 3
G1 = ar = 3 × 3 = 9
G2 = ar2 = 3 × 9 = 27
उत्तर: संख्याएँ 9 और 27 हैं।
27. n का मान ज्ञात कीजिए ताकि an+1 + bn+1an + bn, a तथा b के बीच गुणोत्तर माध्य हो।
गुणोत्तर माध्य (G.M.) = √ab = a1/2 b1/2
अतः an+1 + bn+1an + bn = a1/2 b1/2
तिर्यक गुणा (Cross multiply): an+1 + bn+1 = an+1/2b1/2 + a1/2bn+1/2
an+1 - an+1/2b1/2 = a1/2bn+1/2 - bn+1
an+1/2 (a1/2 - b1/2) = bn+1/2 (a1/2 - b1/2)
an+1/2 = bn+1/2 ⇒ (ab)n+1/2 = 1 = (ab)0
n + 1/2 = 0 ⇒ n = -1/2
घातांकों को व्यवस्थित करके दोनों ओर (a1/2 - b1/2) को कॉमन (Common) निकाल कर काट दिया गया है।
28. दो संख्याओं का योगफल उनके गुणोत्तर माध्य का 6 गुना है तो दिखाइए कि संख्याएँ (3+2√2) : (3-2√2) के अनुपात में हैं।
माना संख्याएँ a तथा b हैं। दिया है: a + b = 6√ab
अतः a + b2√ab = 31
योगांतरातुपात (Componendo & Dividendo) नियम से: a + b + 2√aba + b - 2√ab = 3 + 13 - 1 = 42 = 21
(√a + √b)2(√a - √b)2 = 2 ⇒ √a + √b√a - √b = √21
पुनः योगांतरातुपात नियम से: 2√a2√b = √2 + 1√2 - 1
वर्ग करने पर: ab = (√2 + 1)2(√2 - 1)2 = 2 + 1 + 2√22 + 1 - 2√2 = 3 + 2√23 - 2√2. सिद्ध हुआ।
योगांतरातुपात (Componendo & Dividendo) का दो बार प्रयोग करके प्रश्न को बहुत सरलता से हल किया गया है।
29. यदि A तथा G दो धनात्मक संख्याओं के बीच क्रमशः समांतर माध्य तथा गुणोत्तर माध्य हों, तो सिद्ध कीजिए कि संख्याएँ A ± √(A+G)(A-G) हैं।
माना संख्याएँ a और b हैं।
A = a + b2 ⇒ a + b = 2A
G = √ab ⇒ ab = G2
द्विघात समीकरण जिसके मूल a और b हों: x2 - (a+b)x + ab = 0
x2 - 2Ax + G2 = 0
श्रीधराचार्य सूत्र से: x = 2A ± √(4A2 - 4G2)2 = 2A ± 2√(A2 - G2)2
x = A ± √(A2 - G2) = A ± √((A+G)(A-G)). सिद्ध हुआ।
मूलों के योग और गुणनफल की मदद से द्विघात समीकरण (Quadratic Equation) बनाकर हल किया गया।
30. किसी कल्चर में बैक्टीरिया की संख्या प्रत्येक घंटे पश्चात् दुगुनी हो जाती है। यदि प्रारंभ में 30 बैक्टीरिया थे, तो दूसरे, चौथे तथा nवें घंटों बाद बैक्टीरिया की संख्या क्या होगी?
यहाँ यह एक गुणोत्तर श्रेणी है जिसका प्रथम पद (प्रारंभिक) a = 30 और सार्व अनुपात r = 2 है।
t घंटे बाद संख्या = a rt
दूसरे घंटे बाद (t=2) = 30 × 22 = 30 × 4 = 120
चौथे घंटे बाद (t=4) = 30 × 24 = 30 × 16 = 480
nवें घंटे बाद (t=n) = 30 × 2n
nवें घंटे "बाद" का अर्थ है कि (n+1)वाँ पद निकालना है, इसलिए सूत्र arn उपयोग होगा।
31. 500 रुपये धनराशि 10% वार्षिक चक्रवृद्धि ब्याज पर 10 वर्षों बाद क्या हो जाएगी, ज्ञात कीजिए?
मूलधन (P) = 500 रुपये, दर (R) = 10% वार्षिक, समय (n) = 10 वर्ष
मिश्रधन (A) = P(1 + R100)n
A = 500 (1 + 10100)10 = 500 (1 + 0.1)10
A = 500(1.1)10 रुपये
यह भी एक प्रकार से गुणोत्तर श्रेणी का ही अनुप्रयोग है जहाँ सार्व अनुपात 1.1 है।
32. यदि किसी द्विघात समीकरण के मूलों के समांतर माध्य एवं गुणोत्तर माध्य क्रमशः 8 तथा 5 हैं, तो द्विघात समीकरण ज्ञात कीजिए।

समांतर माध्य (A.M.) = a + b2 = 8 ⇒ a + b = 16
गुणोत्तर माध्य (G.M.) = √ab = 5 ⇒ ab = 25
द्विघात समीकरण का रूप: x2 - (मूलों का योग)x + (मूलों का गुणनफल) = 0
x2 - (16)x + 25 = 0
उत्तर: अभीष्ट द्विघात समीकरण x2 - 16x + 25 = 0 है।
