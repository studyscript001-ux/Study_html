1. (5i)(–35i) को a + ib के रूप में व्यक्त कीजिए।
(5i)(–35i) = –3 i²
i² = -1 ⇒ –3 × (–1) = 3
अतः 3 = 3 + 0·i
हिंदी स्पष्टीकरण: i² = -1 का प्रयोग करें। गुणा करने पर 3 प्राप्त होता है, काल्पनिक भाग शून्य।
2. i⁹ + i¹⁹ का मान ज्ञात कीजिए।
i⁹ = i⁸·i = (i⁴)²·i = (1)²·i = i
i¹⁹ = i¹⁶·i³ = (i⁴)⁴·i³ = 1·(-i) = -i
i⁹ + i¹⁹ = i + (-i) = 0
स्पष्टीकरण: i की घातें 4 से विभाजित करें: i⁴=1, i³=-i ⇒ योग शून्य।
3. i⁻³⁹ को a+ib में बदलिए।
i⁻³⁹ = 1 / i³⁹
i³⁹ = i³⁶·i³ = (i⁴)⁹·i³ = 1·(-i) = -i
अतः i⁻³⁹ = 1/(-i) = -1/i = -1/i × i/i = -i/(-1) = i
i = 0 + 1·i
हिंदी: i³⁹ = -i, व्युत्क्रम लेने पर i प्राप्त होता है।
4. 3(7 + i7) + i(7 + i7) को a+ib रूप में लिखिए।
3(7+7i) = 21 + 21i

वितरण नियम + i² = -1 रखने पर 14 + 28i प्राप्त होता है।
5. (1 – i) – (–1 + i6) को सरल कीजिए।
(1 – i) – (–1 + 6i) = 1 – i + 1 – 6i
= (1+1) + (–i – 6i) = 2 – 7i

6. (15 + i25) – (4 + i52) को a+ib रूप में लिखिए।

7. [(13 + i73) + (4 + i13)] – (–43 + i) का मान ज्ञात कीजिए।

8. (1 – i)⁴ का a+ib रूप ज्ञात कीजिए।
(1 – i)² = 1 – 2i + i² = 1 – 2i – 1 = –2i
(1 – i)⁴ = [(1 – i)²]² = (–2i)² = 4i² = 4(-1) = –4
अतः –4 = –4 + 0·i
पहले वर्ग करके फिर पुनः वर्ग, i² = -1 से मात्र वास्तविक संख्या प्राप्त होती है।
9. (13 + 3i)³ को a+ib रूप में व्यक्त कीजिए।

10. (–2 – 13i)³ को a+ib रूप में लिखिए।

11. 4 – 3i का गुणात्मक प्रतिलोम ज्ञात कीजिए।
माना z = 4 – 3i, तो z⁻¹ = z / |z|²
z = 4 + 3i, |z|² = 4² + (–3)² = 16+9 = 25
अतः z⁻¹ = 4+3i25 = 425 + i325
संयुग्मी / मापांक का वर्ग से प्रतिलोम सूत्र।
12. √5 + 3i का गुणात्मक प्रतिलोम ज्ञात कीजिए।
√5 + 3i, संयुग्मी = √5 – 3i
|z|² = (√5)² + 3² = 5 + 9 = 14
z⁻¹ = √5 – 3i14 = √514 – i314
प्रतिलोम = संयुग्म / (वास्तविक भाग वर्ग + काल्पनिक वर्ग)
13. – i का गुणात्मक प्रतिलोम ज्ञात कीजिए।
z = –i = 0 – i ⇒ z = 0 + i = i
|z|² = 0² + (–1)² = 1
z⁻¹ = i / 1 = i = 0 + 1·i
–i का व्युत्क्रम i होता है, क्योंकि (–i)·(i) = –i² = 1
14. (3+i√5)(3−i√5)(√3+√2 i) − (√3−√2 i) को a+ib रूप में लिखिए।
अंश: (3+i√5)(3−i√5) = 3² − (i√5)² = 9 − (i²·5) = 9 − (−5) = 14
हर: (√3 + i√2) − (√3 − i√2) = √3 + i√2 − √3 + i√2 = 2i√2

1. [i¹⁸ + (1/i)²⁵]³ का मान ज्ञात कीजिए।
i¹⁸ = i¹⁶·i² = (i⁴)⁴·i² = 1·(–1) = –1
(1/i)²⁵ = i⁻²⁵ = 1 / i²⁵, i²⁵ = i²⁴·i = (i⁴)⁶·i = i ⇒ 1/i = –i? वास्तव में i⁻²⁵ = i⁻²⁴·i⁻¹ = 1·(–i) = –i
अतः i¹⁸ + (1/i)²⁵ = –1 + (–i) = –1 – i
अब [ –1 – i ]³ = –(1+i)³ = –[1 + 3i + 3i² + i³] = –[1+3i –3 – i] = –[–2 + 2i] = 2 – 2i
घातों को सरल कर i⁴=1 का उपयोग करें।
2. सिद्ध कीजिए: Re(z₁z₂) = Re z₁·Re z₂ – Im z₁·Im z₂
माना z₁ = a+ib, z₂ = c+id
z₁z₂ = (ac – bd) + i(ad + bc)
Re(z₁z₂) = ac – bd
Re z₁·Re z₂ = a·c, Im z₁·Im z₂ = b·d
∴ ac – bd = a·c – b·d, अतः सिद्ध हुआ।
गुणा के बाद वास्तविक भाग ac−bd प्राप्त होता है।

10. यदि (x+iy)³ = u+iv, तो सिद्ध कीजिए u/x + v/y = 4(x²–y²)

11. यदि α, β भिन्न सम्मिश्र संख्याएँ हैं, |β|=1 तो β−α1−αβ का मापांक ज्ञात कीजिए।
मानक परिणाम: |β| = 1 ⇒ β·β = 1
तब |1 − αβ| = |β − α| (क्योंकि β से गुणा)
वास्तव में |β−α| = |β| · |1 − αβ| ⇒ |β−α| = |1 − αβ|
अतः अभीष्ट मान |β−α||1−αβ| = 1
या स्पष्टः β−α1−αβ का मापांक = |β|·|1−αβ| / |1−αβ| = |β| = 1
हल: |β|=1 का उपयोग करते हुए, अंश और हर के मापांक बराबर हो जाते हैं, इसलिए मान 1 आता है।
