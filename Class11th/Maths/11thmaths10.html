📘 प्रश्नावली 10.1 (वृत्त)
1. केंद्र (0, 2) और त्रिज्या 2 वाले वृत्त का समीकरण ज्ञात कीजिए।
वृत्त का मानक समीकरण: (x - h)² + (y - k)² = r²
यहाँ (h, k) = (0, 2) और r = 2 है।
(x - 0)² + (y - 2)² = 2²
x² + y² - 4y + 4 = 4 ⇒ x² + y² - 4y = 0
2. केंद्र (-2, 3) और त्रिज्या 4 वाले वृत्त का समीकरण ज्ञात कीजिए।
(h, k) = (-2, 3) और r = 4
(x - (-2))² + (y - 3)² = 4² ⇒ (x + 2)² + (y - 3)² = 16
x² + 4x + 4 + y² - 6y + 9 = 16
x² + y² + 4x - 6y - 3 = 0
3. केंद्र (12, 14) और त्रिज्या 112 वाले वृत्त का समीकरण ज्ञात कीजिए।
(h, k) = (12, 14), r = 112
(x - 12)² + (y - 14)² = (112)²
x² - x + 14 + y² - y2 + 116 = 1144
x² + y² - x - y2 + 516 - 1144 = 0
x² + y² - x - y2 + 44144 = 0 ⇒ x² + y² - x - y2 + 1136 = 0
36x² + 36y² - 36x - 18y + 11 = 0
4. केंद्र (1, 1) और त्रिज्या √2 वाले वृत्त का समीकरण ज्ञात कीजिए।
(h, k) = (1, 1), r = √2
(x - 1)² + (y - 1)² = (√2)²
x² - 2x + 1 + y² - 2y + 1 = 2
x² + y² - 2x - 2y = 0
5. केंद्र (-a, -b) और त्रिज्या √(a² - b²) वाले वृत्त का समीकरण ज्ञात कीजिए।
(h, k) = (-a, -b), r = √(a² - b²)
(x - (-a))² + (y - (-b))² = (√(a² - b²))²
(x + a)² + (y + b)² = a² - b²
x² + 2ax + a² + y² + 2by + b² = a² - b²
x² + y² + 2ax + 2by + 2b² = 0
6. (x + 5)² + (y - 3)² = 36 का केंद्र और त्रिज्या ज्ञात कीजिए।
समीकरण की तुलना (x - h)² + (y - k)² = r² से करने पर:
h = -5, k = 3, r² = 36 ⇒ r = 6
केंद्र: (-5, 3), त्रिज्या: 6
7. x² + y² - 4x - 8y - 45 = 0 का केंद्र और त्रिज्या ज्ञात कीजिए।
व्यापक रूप x² + y² + 2gx + 2fy + c = 0 से तुलना करने पर:
2g = -4 ⇒ g = -2
2f = -8 ⇒ f = -4
c = -45
केंद्र = (-g, -f) = (2, 4)
त्रिज्या r = √(g² + f² - c) = √((-2)² + (-4)² - (-45)) = √(4 + 16 + 45) = √65
8. x² + y² - 8x + 10y - 12 = 0 का केंद्र और त्रिज्या ज्ञात कीजिए।
2g = -8 ⇒ g = -4
2f = 10 ⇒ f = 5
c = -12
केंद्र = (-g, -f) = (4, -5)
त्रिज्या r = √(g² + f² - c) = √(16 + 25 - (-12)) = √53
9. 2x² + 2y² - x = 0 का केंद्र और त्रिज्या ज्ञात कीजिए।
पूरे समीकरण को 2 से भाग देने पर:
x² + y² - x2 = 0
2g = -12 ⇒ g = -14, 2f = 0 ⇒ f = 0, c = 0
केंद्र = (-g, -f) = (14, 0)
त्रिज्या r = √(g² + f² - c) = √((1/4)² + 0 - 0) = 14
10. बिंदुओं (4, 1) और (6, 5) से जाने वाले वृत्त का समीकरण ज्ञात कीजिए जिसका केंद्र रेखा 4x + y = 16 पर स्थित है।
माना केंद्र (h, k) और त्रिज्या r है।
(4 - h)² + (1 - k)² = r² ... (i)
(6 - h)² + (5 - k)² = r² ... (ii)
(i) और (ii) को बराबर करने पर: 16 - 8h + h² + 1 - 2k + k² = 36 - 12h + h² + 25 - 10k + k²
⇒ 4h + 8k = 44 ⇒ h + 2k = 11 ... (iii)
केंद्र रेखा 4x + y = 16 पर है, अतः 4h + k = 16 ⇒ k = 16 - 4h ... (iv)
(iv) को (iii) में रखने पर: h + 2(16 - 4h) = 11 ⇒ h + 32 - 8h = 11 ⇒ 7h = 21 ⇒ h = 3
k = 16 - 4(3) = 4. अतः केंद्र (3, 4) है।
त्रिज्या r² = (4 - 3)² + (1 - 4)² = 1 + 9 = 10
समीकरण: (x - 3)² + (y - 4)² = 10 ⇒ x² + y² - 6x - 8y + 15 = 0
11. बिंदुओं (2, 3) और (-1, 1) से जाने वाले वृत्त का समीकरण ज्ञात कीजिए जिसका केंद्र रेखा x - 3y - 11 = 0 पर स्थित है।
माना केंद्र (h, k) है।
(2 - h)² + (3 - k)² = (-1 - h)² + (1 - k)²
4 - 4h + h² + 9 - 6k + k² = 1 + 2h + h² + 1 - 2k + k²
-6h - 4k = -11 ⇒ 6h + 4k = 11 ... (i)
केंद्र रेखा पर है: h - 3k = 11 ⇒ h = 3k + 11 ... (ii)
(ii) को (i) में रखने पर: 6(3k + 11) + 4k = 11 ⇒ 18k + 66 + 4k = 11 ⇒ 22k = -55 ⇒ k = -52
h = 3(-52) + 11 = 72. केंद्र (72, -52)
r² = (2 - 72)² + (3 + 52)² = 94 + 1214 = 1304
समीकरण: (x - 72)² + (y + 52)² = 1304 ⇒ x² + y² - 7x + 5y - 14 = 0
12. त्रिज्या 5 के उस वृत्त का समीकरण ज्ञात कीजिए जिसका केंद्र x-अक्ष पर हो और जो बिंदु (2, 3) से जाता हो।
x-अक्ष पर केंद्र (h, 0) होगा और r = 5.
वृत्त (2, 3) से जाता है: (2 - h)² + (3 - 0)² = 5²
(2 - h)² + 9 = 25 ⇒ (2 - h)² = 16 ⇒ 2 - h = ±4
स्थिति 1: 2 - h = 4 ⇒ h = -2. समीकरण: (x + 2)² + y² = 25 ⇒ x² + y² + 4x - 21 = 0
स्थिति 2: 2 - h = -4 ⇒ h = 6. समीकरण: (x - 6)² + y² = 25 ⇒ x² + y² - 12x + 11 = 0
13. (0,0) से होकर जाने वाले वृत्त का समीकरण ज्ञात कीजिए जो निर्देशांक अक्षों पर a और b अंतः खंड काटता है।
माना वृत्त का समीकरण x² + y² + 2gx + 2fy + c = 0 है।
यह (0,0) से जाता है: c = 0
x-अक्ष पर a अंतः खंड काटता है, अतः यह (a,0) से जाता है: a² + 2ga = 0 ⇒ 2g = -a
y-अक्ष पर b अंतः खंड काटता है, अतः यह (0,b) से जाता है: b² + 2fb = 0 ⇒ 2f = -b
मान रखने पर समीकरण: x² + y² - ax - by = 0
14. उस वृत्त का समीकरण ज्ञात कीजिए जिसका केंद्र (2, 2) हो तथा बिंदु (4, 5) से जाता हो।
त्रिज्या r² = केंद्र और बिंदु के बीच की दूरी का वर्ग
r² = (4 - 2)² + (5 - 2)² = 2² + 3² = 4 + 9 = 13
समीकरण: (x - 2)² + (y - 2)² = 13
x² - 4x + 4 + y² - 4y + 4 = 13 ⇒ x² + y² - 4x - 4y - 5 = 0
15. क्या बिंदु (-2.5, 3.5) वृत्त x² + y² = 25 के अंदर, बाहर या वृत्त पर स्थित है?
वृत्त का केंद्र (0,0) है और त्रिज्या r = 5 (चूँकि r² = 25)।
बिंदु (-2.5, 3.5) की केंद्र से दूरी d = √((-2.5 - 0)² + (3.5 - 0)²) = √(6.25 + 12.25) = √18.5
चूँकि d = √18.5 < √25 = 5 है, अर्थात् दूरी त्रिज्या से कम है।
अतः बिंदु वृत्त के अंदर स्थित है।

प्रश्न 7 से 12 तक परवलय का समीकरण ज्ञात कीजिए जो दिए गए प्रतिबंधों को संतुष्ट करता है:

12. शीर्ष (0, 0), (5, 2) से जाता है और y-अक्ष के सापेक्ष सममित (symmetric) है
